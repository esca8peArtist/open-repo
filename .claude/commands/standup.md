# /standup

Quick daily standup. Three bullets maximum. No preamble.

## Steps

1. Read last 20 lines of `WORKLOG.md`
2. Read `BLOCKED.md` active blocks
3. Read `PROJECTS.md` current focus for active projects

## Output Format

```
**Done**: [what was completed since last standup]
**Doing**: [what's actively in progress right now]  
**Blocked**: [anything waiting on user — or "nothing" if clear]
```

Then ask: "Anything to redirect or unblock?"
