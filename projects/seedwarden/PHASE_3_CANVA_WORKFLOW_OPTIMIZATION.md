# Phase 3 Canva Workflow Optimization

**Design lock**: July 3, 2026 EOD. No cover changes after this date.  
**Brand kit load deadline**: June 21, 2026 (15-minute action, zero-float).  
**Lead finding**: Phase 3 is a palette swap and image swap on Phase 2 templates — not a rebuild. Total design hours for all 5 bundles: 14 hours, fully parallel to writing with 3–14 days of float per task.

---

## Core Principle: Adapt, Do Not Rebuild

Phase 2 templates are the structural foundation. You are changing:
1. Color palette (header background per bundle)
2. Hero images (botanical species per bundle)
3. Framing language in subheadings (forager → herbalist/practitioner)

You are NOT changing:
- Template layout (grid, column structure, margins)
- Typography (Playfair Display Bold headers, Lato Regular body, Lato Italic for Latin binomials)
- Page size (8.5"×11" for guides, standard listing thumbnail dimensions for covers)
- The Seedwarden logo or wordmark placement

**Time estimate per bundle cover: 1.2 hours.** If a cover takes more than 2 hours, stop and apply the fallback rule: simplify to a color block header with title text — this is still a professional-looking guide cover and costs 20 minutes.

## Bundle Cover Design Specs

Each cover is a Canva page sized 2400×2400px (square, for Etsy listing thumbnail):

| Element | Spec |
|---|---|
| Header background | Bundle-specific hex, full-bleed top third |
| Hero image | Centered in header or full-bleed with text overlay on semi-transparent layer |
| Title text | Playfair Display Bold, white or Clinical Cream, 48–60pt |
| Subtitle | Lato Regular, white or Clinical Cream, 24–28pt ("A Cultivation & Medicinal Use Guide") |
| Accent bar | Apothecary Gold #D4AF37, 6–8px horizontal rule below header |
| Lower panel background | Clinical Cream #F9F5F0 |
| Latin binomial | Lato Italic, Dark Charcoal, 16pt, positioned below title |
| Seedwarden logo | Bottom right or bottom center, same placement as Phase 2 |
| Thumbnail test | Export at 2400×2400px; preview at 170×135px crop — title must be legible |

**Do not** use drop shadows on text over the hero image — this makes text look dated and reduces legibility.  
**Do** use a semi-transparent dark overlay (#2C2C2C at 40–50% opacity) behind title text if the hero image is light-toned.

## Batch Production Workflow — 5 Bundles + 20 Single-Herb Guides in 6–8 Days

### Design Batching Rule

Never open Canva to design only one thing. Batch by template type:
- All covers in one session (5 covers × 1.2 hrs = 6 hours over 2 sessions)
- All zone cards in one session (5 cards × 0.8 hrs = 4 hours in one session)
- Practitioner cover in one session (1.5 hrs)
- Consistency review and export in one session (0.5 hrs)

This batching reduces context-switching overhead and ensures color consistency across all covers (you can compare them side by side in the same Canva workspace).

### Cover Production Order

1. Women's Health cover (June 23 — 4-day float; hero pre-staged)
2. Respiratory cover (June 24 — 4-day float; elderberry hero is visually strong, prioritize)
3. Immunity cover (June 29 — 4-day float; hero: Goldenseal root or Ashwagandha root)
4. Sleep cover (June 30 — 3-day float; Passionflower hero pre-staged)
5. **Digestive cover (July 3 — ZERO FLOAT — must be final EOD regardless of upload timing)**

### Single-Herb Guide Design (Post-Sprint, If Scoping)

If producing 20+ single-herb guides (post-sprint, Q3 2026), the batch workflow is:
- Build one master single-herb template with placeholders for: hero image, herb name, Latin binomial, bundle color (variable)
- Duplicate template × number of herbs needed
- Swap image and name per duplicate — 15–20 minutes per guide with a clean template
- Do not redesign layout per herb — the template is the product

For Phase 3 sprint, single-herb guides are not in scope (they are post-sprint Q3). Focus all Canva time on the 5 bundle covers and 5 zone cards.

## Google Docs PDF Fallback

If Canva becomes unavailable, a template is corrupted, or a cover design exceeds 2 hours without resolution:

Google Docs can produce a launch-quality guide PDF in 15–20 minutes:
- Set page size to Letter (8.5"×11")
- Header: colored text box matching bundle hex (use Google Docs background color)
- Hero image: insert from Wikimedia CC download
- Body: paste guide text, apply Heading styles
- Export as PDF (File → Download → PDF Document)
- Result: professional but simpler layout — fully acceptable for launch

The guide content is the primary purchase driver for practitioner buyers, not the visual design. A clean Google Docs PDF with correct content outperforms a Canva template with a single FTC-compliance error.

## Design–Writing Interaction Protocol

**Never stop writing to handle a design issue.** Protocol for design questions that arise during sprint:

1. Log the question in a "Design Questions" note (one running list)
2. Continue writing
3. Resolve all Design Questions during the next scheduled design session (which has float)
4. If a cover design question is blocking a Cover from being finished: use a placeholder color block cover (5 minutes) to unblock the listing QA pass; swap for final cover before upload

Writing float is 2 named days. Design float is 3–14 days. Every minute spent on design instead of writing adds proportional risk to the critical path.
