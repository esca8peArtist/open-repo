# Zone Quick-Start Card — Canva Setup and Execution Guide

**Who this is for**: You — the Seedwarden operator — working in Canva to build the 8 Zone Quick-Start Card PDFs. This guide assumes you know the Canva basics (adding text boxes, using the toolbar, exporting files) but need step-by-step guidance specific to THIS card.

**Total estimated time**: 30–45 minutes for account setup and Brand Kit (Steps 1–3, done once), then 90 minutes for the Zone 5 master template (Step 4), then 35–45 minutes per zone for Zones 6 through 10 (Step 7). Full 8-card set: approximately 7–9 hours across multiple sessions.

**Before you start this guide**: Read CANVA_TEMPLATE_PROTOTYPE.md once to see the full visual layout and branding rules. Then return here and follow Steps 1–9 in order.

---

## Step 1: Canva Account Prerequisites

**Estimated time: 5 minutes (if you have a Canva account) or 10 minutes (new account)**

### Canva free vs. Pro

You can build the Zone Quick-Start Cards on Canva Free. The one Pro feature that would help — background removal — is not needed for this card (solid color backgrounds, no photo compositing). Use Free unless you are also doing lifestyle photo editing in the same session.

If you want Pro for the Brand Kit upload feature (uploading your own fonts): it is $15/month billed monthly. Cancel after one month. The free-tier Brand Kit supports hex color storage and Google Font selection, which is all this guide requires.

### Account setup checklist

- [ ] Canva account active (canva.com, free tier minimum)
- [ ] You can create a new design from the dashboard
- [ ] You have access to the Brand Hub section (left sidebar on Canva dashboard, or via canva.com/brand-hub)

## Step 2: Card Dimensions and Grid Setup in Canva

**Estimated time: 5 minutes**

### Create the master design file

1. On the Canva dashboard, click "Create a design" (top right)
2. In the size picker, click "Custom size" (bottom of the list)
3. Enter: Width **8.5**, Height **11**, Units **inches**
4. Click "Create new design"
5. Immediately rename the file: click the untitled filename at the top of the screen and type `Seedwarden Zone 5 Quick-Start Card — MASTER`

Do not skip the rename. Files named "Untitled" get lost in Canva's project folders.

### Set up guides for column alignment

Canva does not have automatic column grids. You will set six vertical guides manually. These guides snap elements into position and prevent eyeballing alignment.

1. Go to View (top menu) > Rulers and Guides > Show Rulers
2. Drag a vertical guide from the left ruler to each of these positions:
   - **0.4 in** — left margin
   - **3.1 in** — right edge of Column 1
   - **3.35 in** — left edge of Column 2
   - **6.05 in** — right edge of Column 2
   - **6.3 in** — left edge of Column 3
   - **8.1 in** — right margin

To set a guide to an exact position: drag it roughly into place, then double-click it to enter the exact number.

These guides define:
- Column 1: 0.4 in to 3.1 in (2.7 in wide)
- Column 2: 3.35 in to 6.05 in (2.7 in wide)
- Column 3: 6.3 in to 8.1 in (1.8 in wide, within the 0.4 in right margin)

## Step 3: Brand Kit Application

**Estimated time: 15 minutes (done once, applies to all 8 cards)**

The Brand Kit stores your colors and fonts so you can apply them in one click instead of entering hex codes every time. Set it up before building the first element.

### 3A. Add brand colors

1. Go to canva.com/brand-hub (or click "Brand Hub" in the left dashboard sidebar)
2. Click "Create a Brand Kit" if none exists, or open your existing Seedwarden brand kit
3. In the Colors section, click "+ Add color" for each of the following:

| Label to enter | Hex code |
|---|---|
| Deep Forest Green | #2D5016 |
| Warm Cream | #F5EDD6 |
| Burnt Sienna | #A0522D |
| Dark Charcoal | #2C2C2C |
| Parchment | #EDE0C4 |
| Warm Grey | #7A7060 |
| Cool Slate Blue | #3D6B8A |
| Warm Amber | #C9943A |

Enter each hex code exactly as written. Label each with the name shown — these labels are visible in the Brand Kit color picker when you are working inside a design.

### 3B. Set brand fonts

In the Brand Kit Fonts section:

1. Set **Heading font**: Playfair Display (search and select from Google Fonts)
2. Set **Body font**: Montserrat (search and select)
3. Leave the third font slot empty, or add Montserrat Light as a variant if Canva allows it

These two fonts cover every element on the card. Both are free on Google Fonts and available in Canva without Pro.

## Step 4: Building the Zone 5 Master Template

**Estimated time: 90 minutes (session 1A: layout — 60 min; session 1B: content — 30 min)**

Build Zone 5 first. It uses the primary Forest Green zone band, which matches the brand color. Every other card is a duplicate of this master.

### 4A: Set the background color

1. In your open Canva design, click anywhere on the blank canvas (not on any element)
2. The background color swatch appears in the top toolbar
3. Click the color swatch > enter hex: `F5EDD6`
4. The canvas turns Warm Cream. This is the background for all 8 cards.

### 4B: Build the header block

The header occupies the top 1.5 inches of the page. Add these elements in order:

**Element 1 — SEEDWARDEN wordmark (top left)**
- Add text box (Text tool in left sidebar)
- Type: `SEEDWARDEN`
- Font: Playfair Display, 16pt
- Color: `#2D5016` (Forest Green — from Brand Kit)
- Letter spacing: +50 (in the text formatting toolbar, look for "Spacing" or "Letter spacing")
- Position: approximately 0.3 in from left edge, 0.25 in from top

**Element 2 — Zone number (center/dominant element)**
- Add text box
- Type: `5`
- Font: Playfair Display, size 72pt, Bold
- Color: `#A0522D` (Burnt Sienna — from Brand Kit)
- Position: centered horizontally in the header area, below the wordmark row
- This is the largest element on the page. If anything else competes visually, reduce that element, not this one.

**Element 3 — Region name (below zone number)**
- Add text box
- Type: `Central Corridor, Southern New England, Mid-Elevation West`
- Font: Playfair Display Italic, 18pt
- Color: `#2D5016` (Forest Green)
- Position: directly below the zone number, left-aligned to the zone number

**Element 4 — "Zone Quick-Start Card" label (top right)**
- Add text box
- Type: `Zone Quick-Start Card`
- Font: Montserrat Light, 12pt
- Color: `#7A7060` (Warm Grey)
- Position: top-right corner, same vertical position as the SEEDWARDEN wordmark
- Right-align this text box to the right margin guide (8.1 in)

**Element 5 — Zone color band**
- In the left sidebar, click "Elements" > "Shapes" > select Rectangle
- Draw a rectangle spanning the full page width: from 0 to 8.5 in wide, 0.125 in tall
- Fill color: `#2D5016` (Forest Green — this is the Zone 5 / Zone 6 band color)
- No stroke/border — fill only
- Position: flush with the left and right page edges, immediately below the header text block (approximately 1.5 in from the top)
- In the position panel, set X: 0, Width: 8.5 in to ensure it is truly edge-to-edge

### 4C: Build the three-column body

This is the largest section, from the zone band to approximately 7.75 in from the top.

For each column, add elements in this sequence: icon, then column header text, then body text block.

**Column 1 — Frost Dates and Season**

Icon:
- Elements > search "calendar outline" — pick a line-style icon (no color fill inside)
- Size: 0.25 inches (24px equivalent)
- Color: `#2D5016`
- Position: top of Column 1 area, centered between 0.4 in and 3.1 in guides, approximately 1.75 in from top of page

Header text:
- Add text box
- Type: `FROST DATES AND SEASON`
- Font: Montserrat, 11pt, all caps (type in capitals)
- Letter spacing: +50
- Color: `#2D5016`
- Position: directly below the icon

Body text (paste-ready Zone 5 content):
- Add text box covering Column 1 width (0.4 in to 3.1 in guides)
- Font: Montserrat Light, 10pt
- Line height: 1.4 (found in text formatting > spacing)
- Color: `#2C2C2C` (Dark Charcoal)
- Paste this content exactly:

```
Last frost: April 15 – May 10
First frost: October 1 – October 20
Growing season: 150–180 days
Denver CO, Des Moines IA,
Boston MA suburbs

THIS MONTH — MAY 2026

1. Transplant broccoli, cauliflower,
cabbage, kale, and onion starts
outdoors now — frost-tolerant and
need to establish before heat arrives.

2. Direct sow beets, carrots, and
succession lettuce — these are your
spring harvests.

3. Start cucumbers, squash, pumpkins,
and melons indoors late April. Do not
start cucurbits too early — they
resent root disturbance.
```

Style note: Make "THIS MONTH — MAY 2026" slightly heavier than the surrounding text. Select just that line and change its weight to Montserrat Regular or Medium (not Light). This creates the visual break between static frost data and the time-sensitive task block.

**Column 2 — Quick-Start Crops**

Icon:
- Elements > search "seed envelope outline" or "seed packet line"
- Same size and color as Column 1 icon
- Position: top of Column 2 area (3.35 in to 6.05 in guides), same vertical position as Column 1 icon

Header text:
- Type: `QUICK-START CROPS`
- Same styling as Column 1 header

Body text (Zone 5):
```
1. Dragon Tongue bush bean (H)
Direct sow after last frost. 57 days,
no staking, high yield. Best easy
bean in Zone 5.

2. Mortgage Lifter tomato (H)
80 days, large beefsteak. Forgiving
for beginners. Full Zone 5 season.

3. Lemon cucumber (H)
65 days, compact vine, prolific,
less bitter than slicing types.
Great for containers.
```

**Column 3 — Storage and Preservation**

Icon:
- Elements > search "mason jar outline" or "canning jar line"
- Same size and color
- Position: top of Column 3 area (6.3 in to 8.1 in guides), same vertical position

Header text:
- Type: `STORAGE AND PRESERVATION`
- Same styling

Body text (Zone 5):
```
Zone 5 has two preservation
windows: early summer (peas,
greens — freeze) and late summer
(tomatoes, peppers, beans — can,
ferment, or dehydrate).

Tomato sauce is the highest-value
project. One canning day in August
yields 18–24 quarts — 6 months of
pasta sauce.

Hot sauce fermentation: low-effort,
uses peppers before frost. Ferment
2–3 weeks, refrigerate — no
canning needed.
```

### 4D: Build the Variety Spotlight section

This is the full-width band at the bottom of the content area, from approximately 7.75 in to 10.3 in from the top.

**Background band**:
- Elements > Shapes > Rectangle
- Width: 8.5 in (full page), Height: approximately 2.5 in
- Fill color: `#EDE0C4` (Parchment)
- No stroke
- Position: X: 0, Y: 7.75 in (flush with page edges)

**Seedling icon**:
- Elements > search "seedling sprout outline"
- Size: 0.25 in, Color: `#2D5016`
- Position: left side of the spotlight band, top of the band area, at the left margin (0.4 in)

**Section header**:
- Add text box
- Type: `HEIRLOOM VARIETY SPOTLIGHT`
- Font: Montserrat, 11pt, all caps, letter spacing +50
- Color: `#2D5016`
- Position: to the right of the seedling icon, aligned to its vertical center

**Variety content** (Zone 5):
- Add text box spanning the usable width (0.4 in to 8.1 in)
- Font: Montserrat Light, 10pt, line height 1.4
- Color: `#2C2C2C`
- Variety names in Montserrat Italic, 10pt (use italics for the variety name portion only)

Content to paste:
```
Mortgage Lifter tomato (H): 80 days, bred by M.C. Byles of West Virginia for home gardeners.
Exceptionally large sweet pink beefsteak. Seed-saveable, true to type, a Zone 5 staple.

Dragon Tongue bean (H): Yellow with purple streaks, 57 days, no staking. The variety that converts
non-bean-growers.

Chioggia beet (H): 55 days, candy-stripe interior, dual-purpose root and greens, grows in 12 inches
of soil. Ideal for Zone 5 raised beds and containers.
```

### 4E: Build the footer

The footer occupies the bottom 0.7 inches (from 10.3 in to 11 in).

Use two separate text boxes:

Left text box:
- Font: Montserrat Light, 8pt
- Color: `#7A7060` (Warm Grey)
- Left-aligned to the left margin
- Content: `Get the full Zone 5 Calendar — [ETSY-ZONE-CALENDAR-LINK]`

Right text box:
- Same font and color
- Right-aligned to the right margin
- Content: `seedwarden.com/zone`

**Placeholder discipline**: Keep `[ETSY-ZONE-CALENDAR-LINK]` as bracket-marked placeholder text until the Etsy listing is live. The brackets make it impossible to accidentally export without noticing. Replace with the live URL before final export.

## Step 5: Image and Background Application

**Estimated time: Included in Step 4 (no additional images required)**

The Zone Quick-Start Card uses a solid color background (Warm Cream `#F5EDD6`) and a solid Parchment band for the Variety Spotlight. There are no photographs or background images on this card. The design is typography and icon based.

If you want to add a subtle botanical texture in a future version:
- Use it only in the Variety Spotlight band
- Opacity: 8–12% maximum (it should be barely visible behind text)
- Ensure it does not reduce text legibility at 10pt

For the Phase 2 launch, keep the design clean — no background images.

## Step 6: QR Code Placement

**Estimated time: 10 minutes per card (if adding a QR code)**

The Zone Quick-Start Card spec does not require a QR code at launch — the footer contains the Kit landing page URL as text (`seedwarden.com/zone`). A QR code is an enhancement for Phase 3 or for printed distribution.

If you want to add a QR code:

**Generate the QR code**:
1. Go to qr-code-generator.com or qrcode-monkey.com (both free)
2. Enter the URL: your Kit landing page URL (not a placeholder — the QR code must link to a live page)
3. Set foreground color to `#2D5016` (Forest Green) and background to transparent
4. Download as PNG or SVG

**Place in Canva**:
- Position: bottom-right of the Variety Spotlight band, or within the footer
- Size: 0.75 x 0.75 inches maximum — large enough to scan but not dominant
- Do not add a QR code until the Kit landing page URL is confirmed live and stable

**Do not add a QR code if the landing page URL is not final**. A QR code with a broken or redirected URL is worse than no QR code.

## Step 7: 8-Zone Content Fill Process

**Estimated time: 35–45 minutes per zone (after the Zone 5 master is complete)**

Build in this order. The sequence groups cards by zone band color, which prevents the most common error (duplicating with the wrong band color).

| Build order | Zone | Band color hex | Duplicate from |
|---|---|---|---|
| 1 | Zone 5 (master) | `#2D5016` | Blank canvas (done in Step 4) |
| 2 | Zone 6 | `#2D5016` | Zone 5 master |
| 3 | Zone 3 | `#3D6B8A` | Zone 5 master — change band to Cool Slate Blue |
| 4 | Zone 4 | `#3D6B8A` | Zone 3 |
| 5 | Zone 7 | `#C9943A` | Zone 5 master — change band to Warm Amber |
| 6 | Zone 8 | `#C9943A` | Zone 7 |
| 7 | Zone 9 | `#A0522D` | Zone 5 master — change band to Terracotta |
| 8 | Zone 10 | `#A0522D` | Zone 9 |

### Per-zone duplication steps

For each zone after Zone 5:

1. Open the source card (Zone 5 master, or the previous zone in the same color group)
2. Click the three-dot menu (top right) > "Duplicate"
3. Rename immediately: `Seedwarden Zone [X] Quick-Start Card`
4. Make these changes **in this exact order**:
   - Change the zone number (72pt text box in the header)
   - Change the zone band color if crossing to a new color group (click the rectangle > change fill hex)
   - Update the region name line
   - Update Footer left text ("Zone 5" → "Zone X")
   - Replace all Column 1 body text with the new zone's frost dates and This Month tasks
   - Replace all Column 2 body text with the new zone's Quick-Start Crops
   - Replace all Column 3 body text with the new zone's Storage and Preservation tips
   - Replace all Variety Spotlight content with the new zone's three varieties

All paste-ready zone content is in CANVA_ZONE_CARD_BATCH_WORKFLOW.md Part 3. Keep that document open in a second window during your Canva session.

### Zone band color changes — when they apply

You only need to change the zone band color when moving from one color group to another. When duplicating within the same group (Zone 5 → Zone 6, Zone 3 → Zone 4, etc.), the band is already correct.

| If duplicating from | And building | Change band? | New hex |
|---|---|---|---|
| Zone 5 master | Zone 6 | No | — |
| Zone 5 master | Zone 3 | Yes | `#3D6B8A` |
| Zone 3 | Zone 4 | No | — |
| Zone 5 master | Zone 7 | Yes | `#C9943A` |
| Zone 7 | Zone 8 | No | — |
| Zone 5 master | Zone 9 | Yes | `#A0522D` |
| Zone 9 | Zone 10 | No | — |

## Step 8: Export to PDF

**Estimated time: 2–3 minutes per card**

Export each zone card individually after completing and checking it. Do not batch-export all 8 at once — export per-zone so you can catch any issues before the next card is built.

### Canva export settings

1. Click "Share" (top right of Canva)
2. Click "Download"
3. File type: select **PDF Print** — not PDF Standard. PDF Print outputs at 300 DPI. PDF Standard is screen resolution (72 DPI) and is not acceptable for a document subscribers will print.
4. Color profile: RGB (Canva does not offer CMYK — RGB is correct)
5. Flatten PDF: check this box if available — reduces file size by removing layer structure
6. Click "Download"

### File naming

Save with this exact filename format (all lowercase, hyphens, no spaces):

```
zone-3-quick-start-card.pdf
zone-4-quick-start-card.pdf
zone-5-quick-start-card.pdf
zone-6-quick-start-card.pdf
zone-7-quick-start-card.pdf
zone-8-quick-start-card.pdf
zone-9-quick-start-card.pdf
zone-10-quick-start-card.pdf
```

### Save location

Save all 8 files to: `projects/seedwarden/assets/zone-cards/`

Create this directory if it does not exist.

### Expected file size

800 KB to 1.2 MB per card. If a card exports at over 2 MB:
- Click the background of the canvas (not an element) and confirm it shows as a solid color fill (not an uploaded image)
- Check the Variety Spotlight background — confirm it is a Shapes rectangle with a solid fill, not an uploaded texture or image
- If an icon is a PNG rather than an SVG/vector element from the Canva library, replace it with a Canva library element

## Step 9: Verification Checklist

**Estimated time: 5 minutes per card**

Run this checklist inside Canva (at 100% zoom) before clicking Download for each card.

### Per-card checks

| Check | What to confirm | How to check |
|---|---|---|
| Zone number is correct | The 72pt number in the header matches this zone | Read it aloud |
| Zone band color is correct | Band color matches the table in Step 7 | Compare against the hex in the color fill |
| Region name is correct | Text below zone number matches this zone's region | Read it against CANVA_TEMPLATE_PROTOTYPE.md Section 6 |
| This Month shows correct month | Header reads "THIS MONTH — MAY 2026" (or current month) | Scan Column 1 |
| Three task bullets are zone-appropriate | Tasks make sense for this zone's climate | Read each bullet — does it mention the right crops and timing? |
| Quick-Start Crops are correct for this zone | Column 2 shows crops from this zone's content table | Read variety names |
| Variety Spotlight shows this zone's varieties | Spotlight section varieties match the zone | Read variety names |
| Footer left shows correct zone number | "Zone [X]" in the footer matches this card | Read the footer |
| Footer placeholder text is bracketed | `[ETSY-ZONE-CALENDAR-LINK]` is visible in brackets (until live URL is ready) | Check footer left text |
| No text is clipped or cut off | All text boxes are fully visible inside the canvas | Scroll to inspect all text boxes |
| Font rendering is correct | Playfair Display and Montserrat appear (not a fallback font) | Check header and column headers visually |
| Background color is Warm Cream | Canvas background is a warm off-white, not white | Click background — confirm hex F5EDD6 |
| Variety Spotlight band is Parchment | The lower band is slightly darker cream than the background | Compare the two visually |

### Post-download checks (outside Canva)

After downloading the PDF:

- [ ] Open the PDF in your default PDF viewer — confirm it opens without errors
- [ ] Zoom to 100% in the PDF viewer — confirm body text at 10pt is readable
- [ ] Check the zone number at a glance — is it the first thing your eye goes to?
- [ ] Open in an incognito/private browser window from the Google Drive / Kit link — confirm it downloads correctly
- [ ] File size is under 1.5 MB — check the file properties
- [ ] Filename matches the zone-X-quick-start-card.pdf convention — check in your file manager

## Appendix A: Monthly This Month Refresh

The "This Month" block is the only element that updates monthly. All other content (frost dates, crops, storage tips, variety spotlight) is stable year-round.

**Time required**: 15–25 minutes for all 8 zones.

**Monthly protocol**:

1. Open the Zone 5 Canva file
2. In Column 1, find the "THIS MONTH — [MONTH YEAR]" text block
3. Update the month name: "MAY 2026" → "JUNE 2026" (or current month)
4. Replace the three task bullets with the current month's tasks for Zone 5 (source: the Seedwarden Zone-by-Zone Seed Starting Calendar product)
5. Export: Download > PDF Print > `zone-5-quick-start-card.pdf`
6. In Google Drive (or Kit): upload the new file, replacing the old one at the same URL
7. Repeat for Zones 6, 3, 4, 7, 8, 9, 10 in that order

Set a calendar reminder: first business day of each month. Label it "Zone card This Month refresh — 20 min."
