## Section 3: Phase 3 Template Structure

### Bundle Cover Specification (Per-Bundle)

Each of the 5 bundles requires one cover template. The structural layout is identical across
all five; only header color and hero image change per bundle.

**Cover layout (US Letter, 8.5 x 11 in)**:
- Top 40% of page: header background block in bundle-specific color
- Bundle title centered in header: Playfair Display 42-48pt Bold, Clinical Cream #F9F5F0
- Apothecary Gold #D4AF37 accent bar: 8-10px horizontal line between header and body
- Hero herb photograph: full-width in middle 40% of page; CC-licensed (see hero image
  assignments below)
- Bottom 20%: Clinical Cream background; subtitle text "A Cultivation and Use Guide"
  in Cormorant Garamond 16pt italic; Seedwarden logo bottom right

**Bundle-specific color and hero image assignments**:

| Bundle | Header Color | Hex | Hero Image Species | Image Source |
|---|---|---|---|---|

### Interior Page Specification

All interior species pages use the same Phase 2 structure with Phase 3 color swap:

- Page header bar: bundle-specific color (same hex as cover), 1.5 inches tall
- Species name (common + italic Latin): Playfair Display 22-26pt Bold in Clinical Cream

- Section dividers: Apothecary Gold #D4AF37 thin horizontal rule (2px)
- Photo placement: right-aligned 3-inch inset per species page; CC attribution in 8pt Cormorant Garamond italic below photo
- FTC disclaimer block: Clinical Cream background with 1pt Sage Green border; body text 10pt

## Section 4: Color and Typography Consistency Requirements

These rules apply to all Phase 3 Canva work without exception:

1. **Never mix Phase 2 and Phase 3 palette colors in a single design file.** Covers,
   interior pages, and zone cards for any Phase 3 bundle use only Phase 3 colors.

2. **Brand Kit must contain both Phase 2 and Phase 3 colors, clearly labeled.** Label Phase 2
   colors as "Phase 2 — Forager" and Phase 3 colors as "Phase 3 — Herbalist."

3. **Species names are always italicized** in every design element. Actaea racemosa, not
   Actaea racemosa. This is a typographic standard, not optional.

4. **Apothecary Gold #D4AF37 accent bar appears on every Phase 3 page.** It is the single
   visual unifier across all five bundles regardless of header color.

5. **Body font must be one of two options: Lato or Source Sans 3.** Pick one before starting
   the first cover and use it consistently across all five bundles.

6. **Do not use more than two font families in any single design element.** Playfair Display +
   Lato (or Source Sans 3) is the standard pair. Cormorant Garamond is the accent only.

7. **CC photo attribution must be visible in every design that uses a Wikimedia or iNaturalist
   image.** Font: Cormorant Garamond 8pt italic. Placement: below or beside the photo.
