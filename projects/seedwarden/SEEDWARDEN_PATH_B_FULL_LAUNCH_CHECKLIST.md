## SECTION 3: Troubleshooting Guide

### Instagram

**Problem: Profile not appearing after sign-up**
- Wait 2 minutes — Instagram takes 60–120 seconds to propagate a new account
- Search for your handle in another browser tab (not the logged-in tab)
- If still missing after 5 minutes: log out, log back in, then search again
- If still missing after 10 minutes: the account exists but may be in review — proceed with
  other platform setups, return to Instagram in 20 minutes

**Problem: Email verification link not arriving**
- Check spam folder in wanka95@gmail.com
- If not in spam after 3 minutes: in Instagram signup, request a new verification email
- Backup: use phone number SMS verification instead (Settings > Security > Phone Number)
- Time budget for this fix: 5 minutes maximum before switching to SMS

**Problem: Cannot switch to Creator account (option is greyed out)**
- Instagram requires a verified email before allowing account type switch
- Verify email first, then retry the account type switch
- If still greyed out: use Business account type instead of Creator (minor organic reach
  difference, not launch-blocking)

---

### TikTok

**Problem: Age restriction blocks account (account rejected as under 18)**
- TikTok requires date of birth during signup — enter actual birth date
- If you accidentally entered a date that flags as under-18: delete the app, reinstall,
  sign up again with correct birth date using a different email (or wait 30 days for the
  flagged account to age past the restriction — this is not viable for today's launch)
- Use the backup email sub-address: wanka95+tiktok@gmail.com if primary is locked

**Problem: Video upload fails (for initial intro video)**
- TikTok rejects videos over 4 GB or with unsupported codecs
- Accepted formats: MP4, MOV (H.264 codec)
- If video fails: convert using Handbrake (free, 5 minutes)
  - Handbrake settings: MP4 container / H.264 codec / 1080p / 30fps / RF 22
  - Export, then re-upload
- If Handbrake is not installed: skip the video upload for now — launch without it,
  upload within 24 hours

**Problem: Bio link won't save (greyed out or error)**
- TikTok Business accounts require a verified phone number before enabling bio links
- Add phone number: Profile > Settings > Manage account > Phone > Add number
- Verify via SMS, then retry the bio link field

---

### Pinterest

**Problem: Pin scheduling errors (pins fail to schedule)**
- Pinterest's native scheduler accepts up to 10 pins per day and up to 30 days in advance
- If scheduling fails: reduce the batch to 5 pins and retry
- If individual pins fail: check image dimensions — Pinterest requires at least 600px wide;
  standard template is 1000x1500px which always passes

**Problem: Board creation blocked ("You can't create more boards")**
- Pinterest limits new business accounts to 5 boards on day of creation
- Workaround: create 5 boards now, create the remaining 3 boards tomorrow (June 2)
  - Priority order: Boards 1, 2, 3, 4, 8 today (highest traffic intent keywords)
  - Boards 5, 6, 7 tomorrow

**Problem: Image format issues (upload rejected)**
- Pinterest accepts JPEG, PNG, WebP
- Canva exports in JPEG by default — this always works
- If a specific image is rejected: re-export from Canva as PNG and retry

---

### Kit

- Skip payment processor integration entirely at this stage
- Kit's free tier does not require payment processing for email list building
- Payment processing is relevant only if selling directly through Kit (not the strategy)
- Skip it and retry in 10 minutes if Kit's onboarding wizard re-prompts; or click "Skip"

- If Kit prompts for an external email service: ignore or dismiss the prompt
- All delivery goes through Kit's servers by default

**Problem: Shipping calculator configuration prompt**

- If Kit prompts for shipping settings: select "Digital products only" or equivalent
- If no such option: skip the prompt

**Problem: Kit automation stuck in Draft after clicking Publish**
- Navigate away from the automation page, then return
- Click Publish again — occasionally the button fires but the UI does not update
- Confirm by going to Automations list view: status column should show "Active"
- If still Draft after 2 attempts: clear browser cache, reload Kit, click Publish once more
- If persists: log out of Kit, log back in, navigate to automation, publish again

**Problem: Test email (Email 1) not arriving after 5 minutes**
- Confirm automation status is Active (not Draft) in Kit > Automations list
- Check wanka95@gmail.com spam folder
- Confirm the test sign-up used the correct landing page URL (not a Kit preview URL)
- If spam: this is a sender reputation issue at day 0 — click "Not spam" in Gmail to train
  the filter; this affects only day-0 deliverability, resolves within 24 hours
- If not in spam and still missing at 10 minutes: Kit support at kit.co/support

### Cross-Platform

**Problem: OAuth conflicts (logged into multiple accounts, wrong account is active)**
- Use incognito/private browser windows for all platform testing
- One platform per incognito window eliminates OAuth session contamination
- If you are logged into a personal Instagram on the main browser, open a separate
  incognito window for the @seedwarden account

**Problem: Duplicate product listings on Etsy (listings appear twice)**
- Etsy does not auto-duplicate — if two listings appear, one was manually duplicated
- Delete the duplicate: Shop Manager > Listings > find duplicate > delete
- If unsure which is the original: check "Listed date" — the older one is the original

**Problem: Inventory sync between Kit and Etsy fails**
- Kit does not natively sync inventory with Etsy — no sync is expected or configured
- Etsy manages its own inventory; Kit manages email subscribers
- These are two separate systems with no live inventory link
- If you need to update Etsy inventory: Etsy Shop Manager > Listings > Edit > Quantity
- This is a manual action with no automation dependency
