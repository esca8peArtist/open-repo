### 3. **phase-3-decision-framework.md** (Decision Guide)
**21 KB | Decision tree | 4 execution options**

This document translates Phase 1 performance data into a specific Phase 3 execution option:

**Decision Architecture:**
- **Option A (Conservative)**: Phase 1 underperforms → regional listings only
- **Option B (Standard)**: Phase 1 at expected range → full Phase 3 roadmap
- **Option C (Aggressive)**: Phase 1 overperforms → compressed 12-week execution
- **Option D (Focused)**: One cohort dominates → concentrate on cohort-specific products

**Decision Process:**
- Branch A: Sales volume classification (A1–A4)
- Branch B: Cohort concentration (B1–B3)
- Branch C: Repeat purchase rate validation
- Branch D: Etsy message themes
- Fast decision path (5 steps)

**30-Minute Decision Checklist:**
1. Pull Phase 1 data (10 min)
2. Classify into branches (15 min)
3. Select Option A/B/C/D (5 min)

**Use this** on June 15–July 1 to decide which Phase 3 products to launch and in what order. Example: "Phase 1 achieved 45 sales, foragers are 52% of buyers → Option D (Focused), prioritize Wild Edibles products first."

## Support & Escalation

**If Phase 3 execution hits blockers:**

1. **Product specification unclear?**
   - Consult `phase-3-product-specifications.json` (the source of truth for each product)
   - Cross-reference with `phase-3-product-expansion-roadmap.md` Section 1 (Category Analysis) for strategic intent

2. **Cohort or messaging question?**
   - Consult `phase-3-cohort-messaging.md` for how to address the target cohort
   - Cross-reference with `customer-cohort-analysis-framework.md` for deeper cohort psychographics

3. **Timeline sliding or revenue missing target?**
   - Consult `phase-3-product-expansion-roadmap.md` Section 8 (Risk and Contingency)
   - Identify which risk matches your situation; implement the suggested response

4. **Should we shift Options mid-Phase 3?**
   - Consult `phase-3-decision-framework.md` "Reversal Triggers" section
   - Make the call based on data; Option changes are legitimate if Phase 1 signals change
