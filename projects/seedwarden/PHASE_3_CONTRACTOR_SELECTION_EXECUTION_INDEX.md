**Q: What if a contractor doesn't respond by June 28?**

**Q: Can I modify the scoring formula?**  
A: Yes. Default is (Fit × 0.35) + (Budget × 0.20) + (Availability × 0.30) + (Success Prob × 0.15). If you prioritize differently (e.g., availability over fit), adjust weights. Document reasoning in RESPONSE_TRACKING_TEMPLATE.md notes.

**Q: What if I run behind and need to execute faster?**

- Prioritize B1–B2 (top writers)

- Goal: Have at least 1 photographer + 1 writer confirmed by June 28 EOD

**Q: How do I handle timezone issues (e.g., UK-based contractors)?**
