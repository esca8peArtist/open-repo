## EMAIL SEND TEMPLATE

**Use for**: All email sends (Women's Health, Respiratory, Sleep emails)

### Before You Start
- [ ] Kit dashboard open at https://kit.co/account/broadcasts
- [ ] Etsy listing URL confirmed (copy from Etsy Seller Dashboard)
- [ ] Recipient segment selected (e.g., "Phase_3_Interested")
- [ ] Send time set in UTC (not ET)

### Step-by-Step

1. **Create new broadcast** (NOT automation — standalone send for Week 1-2)
   - Click "+ New Broadcast" button
   - Choose "Broadcast" type (not "Automation" or "Scheduled")

2. **Fill in email subject line** (exactly as shown below for your bundle)
   ```
   [PASTE SUBJECT LINE FROM CALENDAR OR BUNDLE TEMPLATE]
   ```

3. **Fill in preview text** (the text that shows in inbox preview)
   ```
   [PASTE PREVIEW FROM CALENDAR OR BUNDLE TEMPLATE]
   ```

4. **Fill in email body** (copy from PHASE_3_BUNDLE_LAUNCH_EMAIL_SEQUENCES.md)
   ```
   Hi there,

   [PASTE BODY SECTION FROM PHASE_3_BUNDLE_LAUNCH_EMAIL_SEQUENCES.md]

   [PASTE CTA WITH ETSY LINK]

   Questions? Reply here.

   Seedwarden
   ```

5. **Replace Etsy link placeholder**
   - Find: `[ETSY_LINK]`
   - Replace with: `https://www.etsy.com/listing/[ACTUAL_LISTING_ID]`
   - Example: `https://www.etsy.com/listing/1234567890`

6. **Select recipient segment**
   - Dropdown: "Segment"
   - Choose: "Phase_3_Interested" (or segment name per date)
   - Verify: [Number] recipients shown

7. **Schedule send time**
   - Set time: 09:00 ET = **13:00 UTC** (for morning sends)
   - OR: 15:00 ET = **19:00 UTC** (for afternoon sends)
   - **CRITICAL**: Use UTC time (not ET) to avoid timezone confusion

8. **Preview and send**
   - Click "Preview" → check formatting
   - Click "Send Now" (if immediate) OR "Schedule for [time]" (if scheduled)

9. **Verification screenshot**
   - After send success, screenshot the "Sent" confirmation page
   - Screenshot should show:
     - Broadcast name
     - Recipient count
     - Send time (UTC)
     - Status: "Sent" or "Scheduled"

10. **Log send in execution log**
    ```
    DATE: [Jun 29 / Jul 6 / etc.]
    BUNDLE: [Women's Health / Respiratory / Sleep]
    EMAIL #: [1 / 2 / 3]
    SUBJECT: [Exact subject line]
    RECIPIENTS: [Number from Kit]
    SEND TIME (UTC): [13:00 / 19:00 / etc.]
    SCREENSHOT: [Filename or link to screenshot]
    EXPECTED OPEN RATE: 22-28%
    NOTES: [Any issues or special notes]
    ```

## SOCIAL POST TEMPLATE — INSTAGRAM

**Use for**: All Instagram posts (Women's Health, Respiratory, Sleep bundles + educational content)

### Before You Start
- [ ] Instagram app or web open at https://www.instagram.com/seedwarden/
- [ ] Image downloaded or Canva design ready
- [ ] Image is 1080×1350px (optimal for Instagram feed)
- [ ] Image formatted as PNG or JPG

### Step-by-Step

1. **Open Instagram and start new post**
   - Click "+" (Create) button at top of Instagram
   - Choose "Post" (not Story, Reel, or other format)

2. **Upload image**
   - Click "Select from computer"
   - Navigate to image file (from Canva or contractor)
   - Confirm image size: 1080×1350px (Instagram will resize if needed)
   - If image too large: see Failure Recovery, Failure Mode 2

3. **Add caption** (copy-paste entire caption below)
   ```
   [PASTE CAPTION EXACTLY FROM CHECKLIST OR CALENDAR]

   [PASTE HASHTAGS BELOW]
   ```

4. **Add hashtags** (8-12 hashtags, paste entire line)
   ```
   #herbalism #[bundle-specific-hashtag] #medicinalherbs #seedwarden #[additional-hashtags]
   ```

5. **Add location** (optional, improves reach)
   - Click "Add location"
   - Type: "Seedwarden" or leave blank if no physical location

6. **Enable comments and likes**
   - Default: ON (allow comments and likes)
   - Keep ON (disable only if spam detected)

7. **Schedule or post immediately**
   - Option A (Post now): Click "Share" → Post appears immediately
   - Option B (Schedule): Click "Schedule" → Set time (e.g., 10:00 ET = 14:00 UTC next day)

8. **Verification screenshot**
   - After post appears, screenshot the live post showing:
     - Post caption (first line visible)
     - Image
     - Engagement counter (likes, comments)
     - Post URL (copy from Instagram share button)

9. **Log post in execution log**
   ```
   DATE: [Jun 29 / Jul 6 / etc.]
   PLATFORM: Instagram
   POST #: [1 / 2 / 3 / etc.]
   CAPTION: [First 50 characters]
   IMAGE: [Filename]
   POST TIME (UTC): [14:00 / 19:00 / etc.]
   POST URL: https://www.instagram.com/p/[POST_ID]/
   SCREENSHOT: [Filename]
   EXPECTED ENGAGEMENT: [4-8% / 6-12% depending on post type]
   ```

## FAILURE RECOVERY PROCEDURES (Quick Reference)

For detailed procedures, see PHASE_3_WEEK_1_2_FAILURE_RECOVERY_PROCEDURES.md.

### Failure Mode 1: Email Send Fails

**Symptom**: Email platform returns error (API timeout, invalid recipient, credential issue)

**Recovery**:
1. Check Kit dashboard error log (https://kit.co/account/broadcasts → History)
2. Verify API credentials valid (Settings → Integrations)
3. Retry send with 30-min delay
4. If still failing: escalate to user with error log screenshot

**Time cost**: 10-15 min

---

### Failure Mode 2: Social Post Upload Fails

**Symptom**: Image too large, format unsupported, platform down

**Recovery**:
1. Check platform status (Instagram → Meta Status, LinkedIn → LinkedIn Status)
2. If image issue: resize to 1080×1350px (Instagram) or 1200×627px (LinkedIn)
3. Retry within 30 min (before post window closes)
4. If platform down: reschedule to next day, same time

**Time cost**: 5-10 min

---

### Failure Mode 3: Contractor Unresponsive (3+ days)

**Symptom**: No response to check-in after 3+ days, no status update

**Recovery**:
1. Send follow-up message (see Contractor Follow-Up Template above)
2. If no response within 24 hours of follow-up: escalate to user
3. Prepare contingency content (see PHASE_3_CONTINGENCY_CONTENT_LIBRARY.md for backup images/copy)
4. Document in PHASE_3_CONTRACTOR_SUCCESS_DASHBOARD.md

**Time cost**: 15-20 min + escalation

---

### Failure Mode 4: Email Engagement Low (<15% open rate)

**Symptom**: Email opens <15% cumulative (vs expected 22-28%)

**Recovery**:
1. Investigate root cause:
   - Check spam folder deliverability (resend to test account, check spam folder)
   - Compare subject line to historical subjects (engagement may be down across board)
   - Check send time impact (resend same email at alternate time next day as A/B test)
2. Document findings in PHASE_3_EXECUTION_LOG.md
3. Adjust Week 2 subject lines / content based on learnings
4. No escalation needed unless <10%

**Time cost**: 20-30 min

---

### Failure Mode 5: Unsubscribe Rate Spikes (>0.5% daily)

**Symptom**: More than 0.5% of recipients unsubscribe per day (content or frequency issue)

**Recovery**:
1. Check unsubscribe comments (if Kit provides feedback)
2. Review most recent emails (off-topic? frequency too high? design issues?)
3. Adjust Week 2: reduce email frequency OR improve content relevance
4. If spike >1%/day: escalate to user

**Time cost**: 15-20 min
