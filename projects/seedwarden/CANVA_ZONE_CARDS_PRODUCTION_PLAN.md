## 1. Card Layout and Dimensions

**Format**: US Letter, portrait orientation — 8.5 inches wide x 11 inches tall

**Canvas setup in Canva**:
- Create a design > Custom size > Width: 8.5 in, Height: 11 in, Units: inches
- Export format: PDF Print (300 DPI — Canva applies this automatically when you select PDF Print)

**Layout structure (top to bottom)**:

| Section | Position (approx) | Height | Content |
|---|---|---|---|
| Header | 0 – 1.5 in from top | 1.5 in | Wordmark, "Zone Quick-Start Card" label, large zone number, region name |
| Zone color band | 1.5 – 1.625 in | 0.125 in (12px) | Full-width solid color bar — the zone's band color |
| Three-column body | 1.75 – 7.75 in | 6 in | Column 1: Frost dates + season; Column 2: Quick-start crops; Column 3: Storage tips |
| Variety spotlight band | 7.75 – 10.3 in | 2.55 in | Parchment (#EDE0C4) background, 3 variety entries |
| Footer | 10.3 – 11 in | 0.7 in | Etsy link (left) + Kit landing page (right) |

**Column widths** (based on 7.7 inches usable after 0.4 in margins each side):
- Column 1: 2.7 in (35%)
- Column 2: 2.7 in (35%)
- Column 3: 2.3 in (30%)
- Gap between columns: 0.25 in

**Background color**: Warm Cream `#F5EDD6` — set as the canvas background color; does not change between zones.

## 2. Template Structure — All Zone Cards

Every zone card uses the same template structure. Only these elements change between zones:
- Zone number (the large 72pt number in the header)
- Zone band color (the 12px horizontal rule)
- Region name (text under the zone number)
- All three body columns (frost dates, crops, storage tips)
- Variety spotlight entries
- Footer zone reference ("Get the full Zone X Calendar")

These elements are IDENTICAL across all 8 cards:
- Background color: Warm Cream `#F5EDD6`
- Canvas size: 8.5 x 11 in portrait
- Font system: Playfair Display (headings) + Montserrat (body) — both free in Canva
- Icon style: line-style (not filled) — calendar, seed envelope, mason jar icons
- Variety spotlight background: Parchment `#EDE0C4`
- Footer text color: Warm Grey `#7A7060` at 8pt Montserrat Light

**Build approach**: Build Zone 5 as the master template from a blank canvas (150–180 min).
Duplicate it 7 times, changing only the zone-specific elements each time (30–45 min per duplicate).

## 7. Google Drive Hosting and Link Generation (May 25, 30 min)

After all 8 PDFs are exported and saved to `projects/seedwarden/assets/zone-cards/`:

1. Go to drive.google.com — log in with wanka95@gmail.com
2. Create a folder: "Seedwarden Zone Cards"
3. Upload all 8 PDFs to this folder
4. For each file: right-click > Share > "Anyone with the link" > Viewer access > Copy link
5. The standard share link is: `https://drive.google.com/file/d/[FILE_ID]/view?usp=sharing`
6. CONVERT this to a direct download link. Replace `/view?usp=sharing` with format below:
   Direct download format: `https://drive.google.com/uc?export=download&id=[FILE_ID]`
   - Example: if share link is `.../file/d/1aBcDeFgHiJkLmNoPqRsTuVwXyZ/view?usp=sharing`
   - Direct download link: `https://drive.google.com/uc?export=download&id=1aBcDeFgHiJkLmNoPqRsTuVwXyZ`
7. Test each converted link in incognito: the PDF should download immediately, not open a viewer
8. If the link opens a Drive viewer instead of downloading: you used the wrong URL format — re-check step 6
