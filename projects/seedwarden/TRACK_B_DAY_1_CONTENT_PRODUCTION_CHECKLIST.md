## Instagram — Day 1 Reel

### Specs

| Field | Value |
|-------|-------|
| Format | Reel (vertical video) |
| Duration | 30–45 seconds |
| Dimensions | 1080x1920px (9:16 aspect ratio) |
| File format | MP4 |
| Audio | On (voice or music with voice) |
| Caption max | 2,200 characters |

### Hook / Content Script

**Opening hook (first 3 seconds — must appear as text or spoken)**:
"I just redesigned how my Etsy listings look. Here's why the original mockups were wrong."

**Content structure**:
1. (0–5 sec) Hook, spoken to camera or as text overlay on B-roll
2. (5–20 sec) Show the before: screen recording or photo of the original mockup-only listing
   (white background mockup from `mockups/`). Narrate: "This is what the listings looked like —
   clean, but it felt like a stock photo."
3. (20–35 sec) Show the after: the new lifestyle photo on the listing. Narrate: "This is the
   same guide in a real kitchen / on a real shelf. I wanted it to look like something a real
   person would use."
4. (35–45 sec) CTA: "Link in bio — every listing now has 5 images."

**Caption (copy-paste ready)**:

```
I redesigned how my Etsy listings look — here's the before and after.

The original mockups were technically correct. Clean background, readable text, professional-ish. But they felt like stock photos, not something you'd actually use.

Spent the last few weeks shooting the guides in real settings — actual kitchens, actual shelves, actual hands. This is what they look like now.

Every listing in the shop has 5 images. Link in bio.

#etsydigitaldownload #etsyseller #growyourown #homesteading #urbanfarming #seedsaving #foodsovereignty #smallbusiness #etsyshop #seedwarden
```

**Hashtags (copy-paste):**
```
#etsydigitaldownload #etsyseller #growyourown #homesteading #urbanfarming #seedsaving #foodsovereignty #smallbusiness #etsyshop #seedwarden
```
