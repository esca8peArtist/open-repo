## Part 3: Quality Checklist

### Plant ID Accuracy (CRITICAL)

- [ ] **Crop names match Seedwarden plant database**
  - Quick-Start Crops (Column 2): Dragon Tongue bean, Mortgage Lifter tomato, Lemon cucumber, etc.
  - Variety Spotlight (3 profiles): Names must be exact botanical names or common names as listed in plant DB
  - Source: `projects/seedwarden/plant-database.csv` or equiv. (verify with user if file location differs)
- [ ] **Growing season info is zone-accurate**
  - Frost dates: Last frost and first frost windows (e.g., "April 15–May 10" for Zone 5)
  - Days to maturity: Verify against USDA Hardiness Zone maps + seed catalogs (e.g., Fedco Seeds)
  - Source: `CANVA_ZONE_CARD_BATCH_WORKFLOW.md` Part 3 (all zones)
- [ ] **No copyrighted crop photos**
  - Variety Spotlight contains botanical names only (no images), so no licensing risk
  - If images are added later, source from: public domain (USDA), Creative Commons (with attribution), or user's own photos
- [ ] **Storage & Preservation tips are accurate**
  - Column 3 recommendations: Verify against NCHFP (National Center for Home Food Preservation) or equivalent trusted source
  - No recommendations for botulism-risk processes (e.g., water-bath canning of low-acid vegetables without tested recipes)

### Image Licensing & Attribution (if images are used)

- [ ] **No unlicensed images embedded in Canva designs**
  - Canva's built-in image library: most are Canva-licensed for use
  - If adding external images (user uploads): verify they are public domain or CC-licensed
  - Attribution: If CC-licensed images are used, attribute in footer or metadata
- [ ] **Seedwarden wordmark/logo**: Verified as project asset (no external copyright)
  - Source: `projects/seedwarden/assets/seedwarden-logo.png` or equiv.

### Metadata & Accessibility

- [ ] **PDF filenames are consistent**: `zone-[NUM]-quick-start-card.pdf` (e.g., `zone-5-quick-start-card.pdf`)
- [ ] **PDF properties filled in** (Canva download dialog):
  - Title: "Seedwarden Zone [X] Quick-Start Card"
  - Author: "Seedwarden Project"
  - Keywords: "gardening, zone [X], planting, frost dates, crops"
- [ ] **Text is selectable in PDF** (not rasterized/image-based)
  - Verify: Open PDF in Adobe Reader or equivalent, try to select/copy text
  - If text is not selectable, re-export from Canva using "PDF Print" (vector), not "Image" (raster)
- [ ] **Contrast ratios meet WCAG AA** (4.5:1 for body text minimum)
  - Font size ≥10pt with 4.5:1 contrast against background passes WCAG AA
  - Seedwarden Brand Kit colors meet this standard (verified in Session 1333)

### Format & File Size

- [ ] **PDF file size is reasonable**: <1.5 MB per card (8 cards = <12 MB total)
  - If file size is >2 MB, check for embedded high-res images or fonts; optimize export settings
  - Typical Zone card size: 400–600 KB (text-only template)
- [ ] **Export format: PDF Print** (vector, not PDF Image/raster)
  - Canva menu: Share > Download > PDF Print (NOT "PDF Image")
  - Verify: Zoom to 200% in PDF reader; text should remain sharp (not pixelated)

### Visual Quality

- [ ] **Layout is balanced** (no text overflow)
  - Column text should fit within column boundaries without overflow
  - Variety Spotlight band should be full-width with no text bleeding
  - Footer should be within safe margin (0.5" from bottom)
- [ ] **Zone number is dominant** (72pt, clearly visible)
- [ ] **Region name is legible** (18pt italic, distinct from body text)
- [ ] **Column headers are distinct** from body text (14pt bold vs 10pt regular)
- [ ] **Color palette consistency**: Zone band color matches Brand Kit
  - Zone 5–6 (Temperate): `#2D5016` (green)
  - Zone 3–4 (Cool): `#3D6B8A` (blue-grey)
  - Zone 7–8 (Warm): `#C9943A` (brown)
  - Zone 9–10 (Hot): `#A0522D` (dark brown)

### Content Accuracy (Batch Workflow Alignment)

- [ ] **All 8 zone cards match `CANVA_ZONE_CARD_BATCH_WORKFLOW.md` exactly**
  - No copy-paste errors or omissions
  - Verify each zone's:
    - Frost date windows
    - "This Month" (May 2026) tasks
    - Quick-Start Crops (3 varieties)
    - Variety Spotlight (3 one-line profiles)
    - Storage & Preservation tips
- [ ] **Footer URLs are either live links or standardized placeholders**
  - If Etsy listing is live: `https://www.etsy.com/shop/...` (actual URL)
  - If Kit landing page is live: `https://[domain]/...` (actual URL)
  - If not yet live: `[ETSY-ZONE-CALENDAR-LINK]` and `[KIT-LANDING-PAGE-URL]` (all caps + brackets)

## Appendix: Troubleshooting

### "Canva says my design is too large to download as PDF"
- **Cause**: Design has high-resolution embedded images or very complex elements
- **Fix**: Simplify — remove unnecessary images, reduce font sizes if oversized, re-export

### "PDF text is not selectable (appears rasterized)"
- **Cause**: Exported as "PDF Image" instead of "PDF Print"
- **Fix**: Re-export: Share > Download > PDF Print (check the option name in Canva menu)

### "Zone band color doesn't match Brand Kit"
- **Cause**: Manual color entry instead of Brand Kit selection
- **Fix**: Delete the zone band element; re-add from Brand Kit colors (Canva Brand Hub > colors)

### "Google Drive link returns 404 or 'Access Denied'"
- **Cause**: Sharing permission not set to "Anyone with the link"
- **Fix**: Right-click file in Drive > Share > Change to "Anyone with the link" > Copy link again

### "PDF file size is >2 MB"
- **Cause**: Embedded high-res images or multiple fonts
- **Fix**: Check for images (should be none in template); reduce font variety; re-export at lower quality if needed
