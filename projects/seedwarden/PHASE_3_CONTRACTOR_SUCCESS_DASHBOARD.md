## 3. Quality Gate Checklist Per Contractor Type

Run these gates at: (1) first-draft review, (2) final-draft review, (3) handoff sign-off.

### Photographer Quality Standards

**Gate 1 — After Session 1 delivery (within 48 hours)**
- [ ] 5 images delivered (top-down flat-lay, 45° flat-lay, lifestyle ×2, close-up detail)
- [ ] Minimum resolution: 2000×2000px confirmed on each image
- [ ] White balance consistent across the 5-image set (compare guide cover text readability)
- [ ] sRGB color space (check in image info / Lightroom metadata)
- [ ] No harsh shadows obscuring herbs or guide cover text
- [ ] File naming convention followed: `[bundle-slug]-[type]-[sequence].jpg`
- [ ] Commercial license clause referenced in contract (no additional license required)

**Gate 2 — Final handoff (all 3 sessions)**
- [ ] 15 total images delivered (5 per bundle: Respiratory, Immunity, Digestive)
- [ ] All images pass Gate 1 checklist
- [ ] Reshoots (if any) completed and re-reviewed
- [ ] Attribution log updated for any sourced CC prop images
- [ ] Images uploaded to named Canva folders or shared drive

**Shoot style pass/fail indicators**:
- Pass: Warm, diffuse light; natural surface; herb specimens clearly identifiable; guide mockup text legible
- Fail: Harsh flash, dark dramatic background, guide text obscured, herbs unidentifiable or wilted

### Writer Copy Standards

**Gate 1 — After first bundle draft (within 72 hours of receipt)**
- [ ] Word count within 10% of target (Respiratory 3,600 ±360w, Immunity 3,800 ±380w, Digestive 3,600 ±360w)
- [ ] All species covered with Latin binomial on first mention
- [ ] FTC review: no unqualified therapeutic claims ("prevents," "cures," "treats" without evidence-tier context)
- [ ] CITES sidebar present and verbatim in Immunity bundle (Goldenseal section — 200 words, mandatory)
- [ ] Contraindication panels cite primary source database (NatMed Pro, MSK Integrative Medicine, or equivalent)
- [ ] Black Cohosh and Echinacea angustifolia conservation sidebars present (UpS At-Risk language)
- [ ] Thyroid warning present in Ashwagandha section (Immunity) and Lemon Balm section (Sleep/Digestive)
- [ ] MAOI interaction note present in Passionflower section (if Sleep bundle in scope)
- [ ] Valerian sedative drug interaction warning present (if Sleep bundle in scope)

**Gate 2 — Final draft (after revision round)**
- [ ] All Gate 1 items confirmed clean
- [ ] No content placeholders remaining (search: "[TBD]", "[INSERT]", "[XXX]")
- [ ] Self-edit pass complete (writer confirms in submission message)
- [ ] Evidence citations format consistent throughout
- [ ] FTC compliance sign-off (writer confirms each claim is evidence-tier framed)

**Writer copy fail indicators**:
- "Elderberry prevents colds" → fail; "Studies suggest elderberry may reduce cold duration" → pass
- Missing Latin binomial → fail
- CITES sidebar abbreviated or missing → fail (Immunity bundle)
- Drug interaction warning absent on Valerian, Passionflower, Ashwagandha → fail

### Habitat Specialist Accuracy Standards

**Gate 1 — After first scoped section delivery**
- [ ] Each habitat claim cites a primary source (USDA PLANTS, NatureServe, UpS, regional flora)
- [ ] Range maps reference specific states or ecoregions (not "Eastern US" without specificity)
- [ ] At-risk species sourcing guidance follows UpS current conservation status (not outdated)
- [ ] Forest farming notes cite established cultivation research (FGV, native plant nursery data)
- [ ] No conflicting recommendations between habitat specialist notes and writer content

**Gate 2 — Final handoff**
- [ ] All Gate 1 items clean
- [ ] Regional coverage cross-referenced with bundle content (Pacific NW — Rebecca Lexa; NE — Arthur Haines; Midwest/SE — Conservation Board referral)
- [ ] No unverified endemic range claims
- [ ] User accuracy review completed before upload clearance granted
