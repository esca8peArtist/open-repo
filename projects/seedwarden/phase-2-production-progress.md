## Guide Draft Quality Checklist

For each guide draft before it moves to Canva layout:

- [ ] Word count within 2,000–2,500 range
- [ ] All sections from template structure present (Identification, Conservation Story, Legal Landscape, Cultivation, Propagation, Site Selection, Year-by-Year Timeline, Harvest Ethics, Indigenous Context, Sourcing, Conservation Impact, Quick Reference)
- [ ] Species name accuracy verified (scientific + common)
- [ ] CITES/conservation status confirmed against USFWS and UpS sources
- [ ] No CC-BY-NC photo placeholders in final layout
- [ ] Cherokee / Indigenous cultural content flagged for accuracy review before publication
- [ ] "Production notes" section removed before final export
- [ ] Supplier list verified (links checked, still active)
- [ ] Quick Reference card contains: scientific name, family, zones, status, shade requirement, soil pH, propagation method, years to harvest, price range, best suppliers, key companion plants, harvest season, harvest rate

**American Ginseng draft v1**: Passes all content checks. Two review flags noted:
1. Cherokee cultural content — verify *a-ta-li-gv* pronunciation/spelling with primary ethnobotany source
2. Wild ginseng harvest data (68% decline from baseline) — confirm against most recent USFWS annual report before publication

**Goldenseal draft v1**: Passes all content checks. One review flag:

**Black Cohosh draft v1**: Passes all content checks. One review flag:
