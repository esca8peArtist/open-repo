## Decision Tree: New Account vs. Upgrading an Existing Personal Account

Before creating any account, check whether a personal Seedwarden account already exists on any of these platforms.

```
Do you have any existing personal accounts for Seedwarden or for your own name?
  YES
    → Is the existing account active with followers or content you want to keep?
        YES → Upgrade the existing account to Business (see per-platform upgrade instructions below)
        NO  → Create a new Business account from scratch (simpler; avoids confusion with old content)
  NO
    → Create a new Business account from scratch
```

**Recommendation**: Create a new Business account on each platform rather than upgrading a personal account. Reason: a new account starts with a clean algorithmic slate. Upgrading a personal account with casual content history can suppress reach on business content until the algorithm recalibrates (typically 2–4 weeks). Given the May 30 launch date, a fresh account is lower risk.

**Exception**: If you already have an Instagram account with 500+ followers in the gardening niche, upgrade it to a Business account rather than starting over. The existing audience is worth more than the clean slate.

### Step 3.5 — Domain Verification

Domain verification tells Pinterest that you own the website in your profile, which enables Rich Pins (enhanced pins that pull metadata from your website automatically).

**Decision tree: Kit subdomain vs. custom domain**:

```
Do you have a custom domain (e.g., seedwarden.com)?
  YES — go to Domain Verification via Custom Domain (Option A below)
  NO  — do you have a Kit landing page URL (e.g., [handle].ck.page/[form-slug])?
    YES — go to Domain Verification via Kit URL (Option B below)
    NO  — skip domain verification for now; revisit when Kit account is set up
```

**Option A — Custom Domain Verification** (if you own seedwarden.com):
1. Pinterest > Settings > Claimed accounts > Website > Claim.
2. Pinterest provides a meta tag to add to your website's HTML. Add this tag to your site's `<head>` section (where your domain is hosted — Squarespace, WordPress, etc.).
3. After adding the tag: return to Pinterest and click "Confirm." Pinterest verifies within 24 hours.

**Option B — Kit URL Verification** (if using the Kit landing page URL):
1. Pinterest > Settings > Claimed accounts > Website > Claim.
2. Enter your Kit landing page URL.
3. Pinterest provides a meta tag. You cannot add this tag to Kit's landing page HTML directly (Kit's free tier does not allow custom HTML head injection). Use the DNS verification method instead:
   - Pinterest provides a TXT record to add to your domain's DNS.
   - If your domain is not yet registered, skip this step. Revisit when the domain is live.
4. Alternatively: skip domain verification for Phase 2. Rich Pins are helpful but not required for Phase 2 operations. Pinterest pins without Rich Pins still appear and drive traffic. Verification can be completed in Phase 3.

### Step 3.6 — Rich Pins Setup

Rich Pins automatically pull structured data (title, description, price) from linked pages. For Seedwarden, the most relevant Rich Pins type is "Article" (pulls title and description from Kit landing page or Etsy listings).

**To enable Rich Pins**:
1. Complete domain verification (Step 3.5) first — Rich Pins require a claimed website.
2. Go to pinterest.com/richpins/validate — enter the URL of any page on your claimed domain and click "Validate."
3. After validation: Pinterest reviews and enables Rich Pins within 24 hours.

**If domain verification is not complete**: skip Rich Pins for now. Standard product pins work for Phase 2.

## Common Issues and Fixes

**Instagram shows "Error creating account"**: Try a different browser. Instagram signup can fail in Firefox — try Chrome or Safari. Alternatively, use the Instagram mobile app: download, open, tap "Create new account."

**TikTok email verification code not arriving**: Check Gmail Spam. If not in Spam: TikTok rate-limits verification codes — wait 5 minutes, then request a new code. Do not request more than 3 codes in 10 minutes.

**Pinterest "Website already claimed by another account"**: This means a previous Pinterest account (personal or old business account) already claimed the domain. Log into the old account, unclaim the domain under Settings > Claimed accounts, then re-claim under the new Seedwarden account.

**Username "seedwarden" taken on one platform**: Use the first available fallback handle and apply it consistently across all three platforms. Update the handle table above and note in WORKLOG.md.
