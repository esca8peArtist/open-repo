## What Phase 3 Is

Phase 3 expands the Seedwarden catalog from 21 products to 47+ by adding:
- 14 native plants and survival garden regional listings ($5.99–$12 each)
- 4 preservation derivative products (canning, fermentation, dehydrating, pressure canning)
- Wild Edibles Quick Reference and Photo Pack
- Medicinal Herb Growing and Harvest Guide (conditional — tied to Phase 2 medicinal herbs reception)
- 3 new bundles (Preservation Season Starter, Master Preserver, Regional Forager)
- 2 planning and homesteading products

**Phase 3 execution options** (decided June 15–July 1 using Phase 2 data):
- **Option A (Conservative)**: Regional listings only — if Phase 2 underperforms
- **Option B (Standard)**: Full roadmap — expected scenario
- **Option C (Aggressive)**: Compressed 12-week timeline — if Phase 2 significantly overperforms
- **Option D (Focused)**: One cohort dominated Phase 2 → concentrate Phase 3 on that cohort's products

   - Total Etsy orders May 30–June 15 (pull from Shop Manager)

   - Which products ordered most (identify top 3)
   - Kit subscriber count and zone distribution
   - Email 3 click rate (seed-saver tag — best proxy for engaged audience)

2. **Classify into branches (15 min)**:
   - Branch A: Volume — how many sales by June 15?
   - Branch B: Cohort — which buyer type dominated (foragers / preppers / homesteaders / gift)?
   - Branch C: Repeat — any Phase 1 buyers purchased Phase 2 products?

   - Underperform (< 20 orders): Option A — regional listings only

   - Overperform (60+ orders): Option C — compressed 12-week timeline
   - One cohort is 60%+ of buyers: Option D — focused on that cohort
