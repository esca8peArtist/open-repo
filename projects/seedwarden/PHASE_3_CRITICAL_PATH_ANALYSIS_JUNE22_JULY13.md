## Appendix: Float Day Prioritization Decision Tree

When Float Day 1 (July 12, 8 hours) is needed:

**First claim**: Any incomplete writing task from Days D15–D20. Writing float has priority over everything else.

**Second claim**: FTC compliance gaps found in review pass (D18). Uncorrectable claims block the affected bundle upload — fix before anything else.

**Third claim**: PDF export failures — file over 5MB, placeholder text found, attribution missing. Must resolve before upload.

**What does NOT get Float Day 1 time**: Design revisions (use Google Docs PDF fallback), photography additions (Wikimedia CC covers it), supplier follow-ups (5 minutes via email), social media scheduling (post-sprint task).
