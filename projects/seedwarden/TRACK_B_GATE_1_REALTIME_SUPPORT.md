Do platforms in this order (each typically 10–15 min):

### Platform 1: Instagram (10–15 min)

**Why first**: Most similar to Etsy customer base (gardeners, homesteaders, visual discovery). Highest uptime of the 3 platforms.

**Quick steps**:
1. Open instagram.com → **"Create new account"** (not log into existing account)
2. Phone number or email: **Use your primary email** (same email for TikTok + Pinterest)
3. Username: **`seedwarden`** (or backup `seedwarden_farm` if taken)
4. Password: [Create new — must include uppercase, lowercase, number, symbol; save to password manager]
5. Full name: **Seedwarden** (or "Seedwarden Farm" if you prefer)
6. Bio: **"Native plants, wild foraging, habitat restoration. Zone-specific guides."** (exactly this or shorten to fit)
7. Profile picture: **Upload `seedwarden_logo_1.png`** (should be square, at least 110×110 px)
8. **Don't** make this private yet (stay public for discovery)
9. **Don't** fill "website" yet (we'll add Etsy/Kit link later)
10. Click "✅ Done"

**Troubleshooting if stuck**:
- **"Username taken"**: Use `seedwarden_farm` or `seedwardenfarming` (write down which you chose for TikTok/Pinterest consistency)
- **"Invalid email"**: Check email format (no spaces, proper @ domain). If using personal email with + alias (e.g. `you+seedwarden@gmail.com`), try base email first, then add + alias if base taken
- **"Can't upload profile picture"**: File must be .jpg, .png, or .gif. Try resizing if >5MB. If stuck, skip for now (do after TikTok/Pinterest; come back and batch-upload logos)
- **Account created but can't log back in**: Check password was saved correctly. Try "Forgot password?" recovery if stuck >5 min

**If Instagram succeeds in <10 min**, start TikTok immediately. **If stuck >10 min on any step**, skip to next platform and come back later (TikTok/Pinterest might have better UX for you).

### Platform 2: TikTok (10–15 min)

**Why second**: Fastest growth for gardening/homesteading content. Reels-like platform (good for short videos). Usually good UX for account creation.

**Quick steps**:
1. Open tiktok.com → **"Sign up"** button
2. **"Sign up with email"** (not phone or social login yet)
3. Email: **Same email as Instagram** (critical for consistency)
4. Birthdate: Enter valid date (TikTok legally requires 13+)
5. Username: **`seedwarden`** (or match whatever you chose for Instagram if taken)

7. Click "✅ Create account"
8. **Verify email**: TikTok will email you verification link. **Click the link in your email.** (This happens in real-time; check inbox/spam folder)
9. After email verified, profile opens. Fill:
   - **Avatar**: Upload `seedwarden_logo_1.png` (should appear as circular profile picture)
   - **Name/Display name**: **Seedwarden** (or "Seedwarden Farm")
   - **Bio**: **"Native plants, wild foraging, habitat restoration. Zone-specific guides."** (exactly this)
10. **Skip** "Follow topics" / "Follow creators" (we'll build audience later)
11. Click "✅ Finish"

**Troubleshooting if stuck**:
- **"Email already in use"**: If you used that email for a prior TikTok account, click "Sign in instead" and reset password. If you can't recover that account, use `you+tiktok@gmail.com` variant email instead (and note it for next platform)
- **"Email verification not arriving"**: Check spam folder. If not there, try `you+tiktok@gmail.com` alternate email
- **"Username taken"**: Use `seedwarden_farm` or `seedwardenfarming` (must match Instagram choice for consistency)
- **Avatar upload fails**: TikTok is usually lenient. If PNG fails, try JPG version. If >5MB, resize to <1MB
- **"Account creation too fast"**: TikTok rate-limits. Wait 5 minutes, try again
