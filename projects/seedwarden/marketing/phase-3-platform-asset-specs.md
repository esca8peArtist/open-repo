# Phase 3 Platform Asset Specifications

Quick-reference guide for producing and exporting content assets for each platform. Use this when exporting from Canva, editing video, or building pin templates. All specs are current as of April 2026.

---

## TikTok

### Video

| Spec | Value |
|---|---|
| Aspect ratio | 9:16 (vertical) |
| Recommended resolution | 1080 x 1920 px |
| Minimum resolution | 720 x 1280 px |
| File formats | MP4 or MOV |
| Max file size | 287.6 MB |
| Max video length | 10 minutes (organic); target 30–60 seconds for homesteading content |
| Frame rate | 24–60 fps (30 fps standard for phone footage) |
| Audio | Stereo, 44.1 kHz or higher. Spoken voiceover should be normalized to -14 LUFS. |

**Safe zones** (avoid placing text or critical visuals here):
- Top 15% of frame: TikTok UI overlays (username, sound name)
- Bottom 25% of frame: caption text, sound bar, share buttons
- Right edge (10%): like/comment/share action bar
- **Safe zone for on-screen text**: center 60% of frame width, rows 15–75% of height

**Caption**:
- Limit: 2,200 characters
- First 150 characters visible without "more" tap — lead with the hook
- Hashtags in caption, not first comment (TikTok treats them as caption metadata, not separate)
- Target 3–5 hashtags: 1 broad (millions of views), 2–3 mid-tier niche, 1 micro-niche
- Working Seedwarden hashtag stack (educational): `#homesteading #seedsaving #growyourown #urbangardening #seedwarden`

**Export settings from Canva / CapCut**:
- Canva: Export as MP4, "High quality" (1080p)
- CapCut: 1080p, H.264 codec, 30fps
- Avoid exporting with a watermark from any tool other than Seedwarden branding

## Instagram

### Reels (primary growth format)

| Spec | Value |
|---|---|
| Aspect ratio | 9:16 (full-screen vertical preferred); 4:5 acceptable |
| Recommended resolution | 1080 x 1920 px (9:16) or 1080 x 1350 px (4:5) |
| File format | MP4 (H.264 codec) |
| Max file size | 1 GB |
| Recommended length | 15–45 seconds for homesteading content; max 90 seconds for IG Reels |
| Frame rate | 23–60 fps |

**Safe zones for Reels**:
- Top 14%: IG UI (back arrow, audio info)
- Bottom 35%: caption preview, username, audio bar, engagement icons
- Right edge 8%: follow/like/comment/share buttons
- **Safe zone**: center 80% width, rows 14–65% height

**Cover image** (shown in grid): Export a still frame from the video, or create a separate 1:1 (1080x1080px) cover. Make this the strongest visual frame — it represents the Reel in your profile grid.

### Carousels

| Spec | Value |
|---|---|
| Aspect ratio | 1:1 (square) or 4:5 (portrait preferred for feed presence) |
| Resolution | 1080 x 1080 px (1:1) or 1080 x 1350 px (4:5) |
| File format | JPG or PNG |
| Max slides | 10 |
| Max file size per image | 8 MB |

**Carousel design notes**:
- Slide 1 is the hook — visible in feed without swiping. Must standalone.
- Use a visual "peek" at the right edge of Slide 1 (partial element from Slide 2) to signal there's more content.
- Consistent font and color palette across slides. Seedwarden palette: earth tones (warm browns, muted greens, cream), readable sans-serif.
- Add slide numbers (e.g., "2/6") in a corner for wayfinding if slides don't flow obviously.
- CTA on final slide: product name, price, and "link in bio."

### Stories

| Spec | Value |
|---|---|
| Aspect ratio | 9:16 |
| Resolution | 1080 x 1920 px |
| File formats | JPG, PNG (images); MP4, MOV (video) |
| Max video length | 60 seconds per Story card |
| Max file size | 4 GB (video), 30 MB (image) |

**Safe zones for Stories**:
- Top 250px (13%): profile photo, username, progress bar
- Bottom 250px (13%): link button, "Send message," swipe-up area
- **Safe zone**: rows 13–87%, full width (but keep key text centered)

**Interactive elements** (use at least one per Story):
- Poll sticker: "Which would you plant first — tomatoes or peppers?"
- Question sticker: "What are you growing this season?"
- Link sticker: direct link to specific Etsy listing (no swipe-up threshold required)

### Static Feed Posts

| Spec | Value |
|---|---|
| Aspect ratio | 1:1 (square) or 4:5 (portrait) |
| Resolution | 1080 x 1080 px or 1080 x 1350 px |
| File format | JPG or PNG |

Static posts are secondary to Reels in Phase 3. Use for: product flat lays, quote graphics, educational infographic headers.

**Hashtag strategy for Instagram**:
- Use 5–10 hashtags (Instagram, unlike TikTok, does not benefit from stacking 20–30 tags)
- Place in first comment to keep caption clean, or at end of caption after a few line breaks
- Mix: 2 broad (`#homesteading`, `#urbangardening`), 4–5 mid-tier (`#seedsaving`, `#growyourown`, `#heirloomseeds`, `#foodpreservation`), 2–3 micro-niche (`#seedwarden`, `#containergarden`, `#apartmentgarden`)

## Pinterest

### Standard Pin (primary format)

| Spec | Value |
|---|---|
| Aspect ratio | 2:3 (vertical — this is the algorithm-favored format) |
| Recommended resolution | 1000 x 1500 px |
| Minimum resolution | 600 x 900 px |
| File formats | JPG or PNG |
| Max file size | 20 MB |
| Max title length | 100 characters (first 30–40 shown in feed) |
| Max description length | 500 characters (write to 150–200; first 75 shown) |

**Safe zones**:
- Bottom 10%: Pinterest UI can overlay save/visit buttons
- **Key text placement**: upper third and center of image; avoid bottom 15%

**Text overlay best practice for Seedwarden pins**:
- Lead with the search-intent phrase as headline: "How to Save Tomato Seeds at Home"
- Sub-line: product benefit or content hook: "Fermentation method — works for any open-pollinated variety"
- Brand name small at bottom: "Seedwarden | Etsy"
- Font: bold, readable at thumbnail size. Minimum 24pt equivalent at 1000px width.
- Text contrast ratio: minimum 4.5:1 against background. Dark text on light mockup, or white text on dark overlay.

**Three pin variants per product (build at first use, reuse over months)**:
1. Product mockup image (tablet or phone frame from Phase 2 mockups) + search-intent headline
2. Flat lay or lifestyle image + benefit-focused headline ("Grow hot peppers in a 5-gallon bucket")
3. Educational infographic / chart slide repurposed from an Instagram carousel

### Video Pin

| Spec | Value |
|---|---|
| Aspect ratio | 1:1 or 2:3 (2:3 preferred) |
| Resolution | 1080 x 1080 px (1:1) or 1080 x 1920 px (2:3) |
| File format | MP4 or MOV |
| Length | 4 seconds to 15 minutes; 5–10 seconds for product awareness pins |
| Max file size | 2 GB |
| Must have cover image | Yes — export a strong still frame |

**Seedwarden video pin use case**: 5–8 second clips showing pages of a guide being flipped, a seed fermentation jar, or a time-lapse of seedlings emerging. No audio needed — Pinterest video pins are often watched without sound.

**Pin description writing formula**:
`[What it is] + [who it's for] + [what problem it solves] + [brand name] + [relevant keywords]`

Example: "Save heirloom tomato seeds using the fermentation method — the only reliable way to remove germination inhibitors and prevent seed-borne disease. Works for any open-pollinated variety. Step-by-step guide from Seedwarden. No special equipment needed. #seedsaving #heirloomseeds #growyourown"

**Linking**: Every pin links directly to the specific Etsy product listing. Not the shop homepage. The fewer clicks from pin to purchase, the higher the conversion rate.

**Scheduling**: Use Tailwind (free tier covers basics) or Pinterest's native scheduler. Pin at consistent days and times — Pinterest's algorithm rewards cadence. Do not create 30 pins in one day.

## Etsy Shop Banner and Listing Images

### Shop Banner

| Spec | Value |
|---|---|
| Recommended size | 3360 x 840 px |
| Minimum size | 1200 x 300 px |
| Aspect ratio | 4:1 |
| File format | JPG or PNG |
| Max file size | 10 MB |

**Etsy banner safe zone**: Etsy crops the banner on mobile to approximately the center 1200 x 300 px of the full 3360 x 840 banner. Keep the shop name, tagline, and primary visual within the center 35% of the banner width. Do not place critical text near the edges — it will be hidden on mobile.

**Banner content**: Shop name (Seedwarden), brief tagline ("Heirloom seeds and guides for city growers"), and one representative visual (a seed packet, a guide open on a tablet, or a flat lay of products). Keep it clean — the banner is first-impression real estate, not an ad.

### Listing Images (product mockups)

| Spec | Value |
|---|---|
| Recommended size | 2000 x 2000 px minimum (Etsy zoom requires 2000px shortest side) |
| Seedwarden mockup size | 2400 x 2400 px (already generated from Phase 2) |
| Aspect ratio | 1:1 (square) |
| File format | JPG or PNG |
| Max file size | 20 MB per image |
| Max images per listing | 10 |

**Seedwarden listing image slot plan** (apply to all 21 products):
1. Tablet mockup — cover page on tablet frame (generated, ready)
2. Phone mockup — cover on portrait phone frame (generated, ready)
3. Interior mockup — 2x2 grid of pages 2–5 (generated, ready)
4. Flat lay or lifestyle photo (Phase 2 gap — to be added post-launch)
5–10. Reserved for additional lifestyle shots, detail views, or infographic summaries

### Listing Video (optional but recommended for digital products)

| Spec | Value |
|---|---|
| Recommended ratio | 1:1 or 4:3 |
| Min resolution | 1080 x 1080 px (1:1) |
| Length | 5–15 seconds |
| File format | MP4 or MOV |
| Max file size | 100 MB |

**Seedwarden listing video idea**: A 5–10 second clip of the PDF pages scrolling or flipping on a tablet screen. Simple. Demonstrates product is real and content-rich. Can be screen-recorded from a PDF viewer and trimmed in CapCut or iMovie.

## Cross-Platform Repurposing Map

| Source content | Repurpose to |
|---|---|
| 45-second TikTok / IG Reel | Trim to 20s for IG Stories; export best still frame as Pinterest cover; extract 3 key frames for carousel slides |
| IG Carousel (6 slides) | Each slide becomes a standalone Pinterest pin; lead slide becomes a TikTok text-on-screen hook |
| Pinterest pin (2:3 static) | Resize to 1:1 for Instagram post; add brief video element for Pinterest video pin |
| Flat lay photo | Use as IG static post, Pinterest pin background, and Etsy listing slot 4 |
| Product mockup (from Phase 2) | Core asset for Pinterest pins, Etsy listing slots 1–3, IG product posts |

**One-piece-to-four-platform rule**: Every content piece created should appear on at least two platforms in the same week. Don't create TikTok-only or Pinterest-only assets — adapt.
