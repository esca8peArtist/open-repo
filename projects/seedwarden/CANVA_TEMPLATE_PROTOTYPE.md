# Zone Quick-Start Card — Canva Template Prototype

**What this document is**: A visual design brief for the Seedwarden Zone Quick-Start Card. It shows the card layout as an annotated mockup, defines every branding rule in one place, and includes a fully filled-in Zone 5 example so you can see exactly what a finished card looks like before opening Canva. Use this alongside CANVA_SETUP_AND_EXECUTION_GUIDE.md when building.

**What this is not**: A step-by-step Canva walkthrough. That is in CANVA_SETUP_AND_EXECUTION_GUIDE.md.

## Section 2: Branding Rules

### Color Palette

All colors must be entered by hex code in Canva. Never use Canva's named color presets — they do not match these values.

| Element | Color name | Hex | Applied to |
|---|---|---|---|
| Primary brand | Deep Forest Green | `#2D5016` | Logo/wordmark, column headers, icon fills, zone 5–6 band |
| Page background | Warm Cream | `#F5EDD6` | Full canvas background (set via Canva background color, not a rectangle) |
| Accent | Burnt Sienna | `#A0522D` | Zone number, zone 9–10 band |
| Body text | Dark Charcoal | `#2C2C2C` | All body copy in columns and spotlight section |
| Spotlight band | Parchment | `#EDE0C4` | Variety Spotlight background rectangle only |
| Footer / label | Warm Grey | `#7A7060` | Footer text, "Zone Quick-Start Card" label in header |
| Zone 3–4 band | Cool Slate Blue | `#3D6B8A` | Zone band rectangle for cold zones |
| Zone 7–8 band | Warm Amber | `#C9943A` | Zone band rectangle for warm zones |

**Brand Kit setup**: Before building any card, enter all eight hex values into the Canva Brand Kit. This prevents manual hex entry per element. See CANVA_SETUP_AND_EXECUTION_GUIDE.md Step 3 for exact Brand Kit setup steps.

### Typography

Both fonts are free in Canva. Search by name in the font selector.

| Element | Font | Size | Weight/Style | Tracking |
|---|---|---|---|---|
| SEEDWARDEN wordmark | Playfair Display | 16pt | Regular (typed in ALL CAPS) | +50 letter spacing |
| Zone number | Playfair Display | 72pt minimum | Bold | Default |
| Region name | Playfair Display | 18pt | Italic | Default |
| "Zone Quick-Start Card" label | Montserrat | 12pt | Light | Default |
| Column headers | Montserrat | 11pt | Regular (typed in ALL CAPS) | +50 letter spacing |
| "THIS MONTH" sub-header | Montserrat | 10pt | Medium (or Regular) | Default |
| Column body text | Montserrat | 10pt | Light | Default, line height 1.4 |
| Variety names in spotlight | Montserrat | 10pt | Italic | Default |
| Spotlight body descriptions | Montserrat | 10pt | Light | Default |
| Footer text | Montserrat | 8pt | Light | Default |

**Critical rule**: Do not go below 10pt for any body text. At 8.5x11 inches, 10pt Montserrat Light is readable when printed at home. 9pt or smaller risks legibility on consumer inkjet printers.

### Spacing and Margins

| Measurement | Value | Notes |
|---|---|---|
| Page margins (all sides) | 0.4 inches | Left, right, top, bottom |
| Usable page width | 7.7 inches | 8.5 minus two 0.4-inch margins |
| Column 1 width | 2.7 inches | 35% of usable width |
| Column 2 width | 2.7 inches | 35% of usable width |
| Column 3 width | 2.3 inches | 30% of usable width |
| Inter-column gap | 0.25 inches | Included in column widths above |
| Zone band height | 0.125 inches | 12px at 96 DPI; full page width, flush edge-to-edge |
| Header block height | ~1.5 inches | Zone number 72pt drives this |
| Column body height | ~6 inches | From below zone band to variety spotlight top |
| Variety spotlight height | ~2.5 inches | Full-width parchment band |
| Footer height | ~0.7 inches | Bottom of page |

### Icon Rules

| Rule | Specification |
|---|---|
| Style | Line-style (outline only, no fill inside the icon shape) |
| Size | 24px (approximately 0.25 inches in Canva) |
| Color | Forest Green `#2D5016` for all column header icons |
| Placement | Centered above each column header text |
| Consistency | All three icons must come from the same visual family — do not mix styles |

Recommended Canva search terms to find matching line icons:
- Column 1 (dates/season): "calendar outline" or "botanical calendar line"
- Column 2 (crops): "seed envelope outline" or "plant seed line"
- Column 3 (storage): "mason jar outline" or "canning jar line"
- Variety spotlight: "seedling sprout outline" or "plant sprout line"

If no exact match appears, search "botanical outline" and browse results for a consistent set.

### Logo / Wordmark Placement

The Seedwarden wordmark appears in the top-left of the header. At Phase 2, this is typographic-only (the SEEDWARDEN text in Playfair Display). If a logo mark (botanical illustration) is finalized before card production begins, it sits to the left of the wordmark at 24–32px height.

Do not use the logo-concepts.md AI prompts to generate a placeholder — leave the wordmark text-only until the actual logo file exists.

## Section 3: Content Zones — What Goes Where

The card has four distinct content areas. Each has a fixed purpose. Do not move content between zones.

| Zone | Purpose | Time-sensitive? | Update cadence |
|---|---|---|---|
| Header | Zone identity — number, region, brand | No (zone band color is static) | Never (unless rebranding) |
| Column 1 — Frost Dates and Season | Static climate data + monthly task block | Yes (This Month tasks only) | Monthly — update the 3 task bullets only |
| Column 2 — Quick-Start Crops | Top 3 crops for this zone, one-line each | No | Never (or when crop research updates) |
| Column 3 — Storage and Preservation | 3 zone-appropriate preservation tips | No | Never (or when content improves) |
| Variety Spotlight | 3 heirloom varieties with one-line descriptions | No | Never (or when variety research updates) |
| Footer | Etsy link + landing page link | Yes — URLs not live at first build | Once, when both URLs go live |

**The only element you update monthly is the "This Month" task block in Column 1** (the three bullet points and the month name in the header "THIS MONTH — [MONTH YEAR]"). Everything else is set-once.

## Section 5: File Specifications

| Specification | Value | Notes |
|---|---|---|
| Document dimensions | 8.5 x 11 inches | US Letter, portrait |
| Resolution | 300 DPI | Set automatically by Canva when exporting as PDF Print |
| Color mode | RGB | Canva does not support CMYK; RGB is correct for digital delivery and home printing |
| Export format | PDF Print | Use "PDF Print" not "PDF Standard" in Canva download settings |
| Target file size | Under 1.5 MB per card | If over 2 MB, check that background is a solid color fill, not an image |
| Bleed | None | Home printing; no bleed required |
| Number of pages | 1 page per zone file | 8 separate PDF files total |
| Total files at launch | 8 | Zones 3, 4, 5, 6, 7, 8, 9, 10 |

### File Naming Convention

```
zone-3-quick-start-card.pdf
zone-4-quick-start-card.pdf
zone-5-quick-start-card.pdf
zone-6-quick-start-card.pdf
zone-7-quick-start-card.pdf
zone-8-quick-start-card.pdf
zone-9-quick-start-card.pdf
zone-10-quick-start-card.pdf
```

All lowercase. Hyphens between words. No spaces. No version numbers in the filename — the file is updated in place in Google Drive or Kit each month.

**Save location**: `projects/seedwarden/assets/zone-cards/` (create this directory before your first export session if it does not exist).

## Section 7: Design Decisions Already Made

These decisions are locked. Do not revisit them during a Canva session.

| Decision | Chosen | Rationale |
|---|---|---|
| Orientation | Portrait | Prints on any home printer; matches standard reference document format |
| Template approach | Blank canvas (not a pre-made template) | No Canva template matches the 3-column + spotlight band structure |
| Zone number size | 72pt minimum | Must be the dominant visual element — zone identity at a glance |
| Font system | Playfair Display + Montserrat | Both free in Canva; strong serif/sans contrast; spec-specified |
| Icon style | Line-style only | Filled icons conflict with text-forward layout; line icons recede correctly |
| Body text minimum | 10pt | Smallest readable size on home-printed US Letter |
| Color mode | RGB | Canva limitation; correct for digital and home print delivery |
| Delivery | 8 individual zone PDFs | One clean file per subscriber; zone identity at file level |
| Build order | Zone 5 master first | Temperate green band is the primary brand color; cleanest starting point |
| Monthly update scope | This Month block only | All other content is stable; monthly update is a 20-minute task |
