# Social Account Creation Checklist

**Purpose**: Step-by-step platform-specific instructions for creating all three Seedwarden
social accounts. Bio copy, profile image path, and handle fallbacks are all specified here.
This is a 30-60 minute session, executable in one sitting.

**When to do this**: Today or May 6 at the latest. Account creation unblocks all content
scheduling and bio link setup.

**Profile image to use**: `projects/seedwarden/logos/seedwarden_logo_1.png`
Download this file to your phone or computer before starting.

**Email for all accounts**: wanka95@gmail.com

## Instagram

### Step 1: Create the Account
1. Go to instagram.com on your phone or desktop
2. Tap "Sign up" (or "Create new account")
3. Enter email: wanka95@gmail.com
4. Set username: `seedwarden`
   - If unavailable: try `seedwarden.co`, then `seedwarden.seeds`, then `seedwarden_guides`
   - Record actual handle claimed in the confirmation table at the bottom of this document
5. Set a password you will remember
6. Complete age verification if prompted

### Step 2: Switch to a Business Account
1. Go to your profile > Settings > Account
2. Tap "Switch to professional account"
3. Select account type: "Business"
4. Select category: "Product/service" (or "Education" if preferred)
5. You do not need to add a phone number — skip if prompted

### Step 3: Configure the Profile
1. Tap "Edit profile"
2. **Profile photo**: upload `seedwarden_logo_1.png`
   - On mobile: tap the profile photo circle > "New profile photo" > select from photo library
   - The logo is square-safe and reads clearly at 110px Instagram thumbnail size
3. **Name field**: `Seedwarden`
4. **Bio**: Copy and paste exactly (150 character limit):
   ```
   Field guides for growers, foragers + food preservers.
   Heirloom seeds. Wild edibles. Real food skills.
   Zone-specific free card below
   ```
5. **Link**: Add the Kit landing page URL once Step 3 of Action 3 in the master guide is done
   - Placeholder: leave blank until Kit landing page is live; add it before first post
6. **Contact options**: Turn on email button, enter wanka95@gmail.com
7. Tap "Done" or "Submit"

### Step 4: Confirm Account Is Ready
- [ ] Account exists at instagram.com/seedwarden (or fallback handle)
- [ ] Account type: Business
- [ ] Profile photo matches seedwarden_logo_1.png
- [ ] Bio text entered correctly (150 chars)
- [ ] Link-in-bio set to Kit landing page URL
- [ ] Email contact option enabled

## TikTok

### Step 1: Create the Account
1. Download the TikTok app (mobile only for account creation)
2. Tap "Sign up" > "Use phone or email"
3. Select "Use email"
4. Enter: wanka95@gmail.com
5. Set username: `seedwarden`
   - If unavailable: use the same fallback handle as Instagram
   - TikTok allows username changes once every 30 days — claim fallback immediately; change
     to primary if it becomes available
6. Complete birthday verification if prompted

### Step 2: Switch to a Business Account
1. Go to Profile > Menu (three lines) > Settings and privacy
2. Tap "Account" > "Switch to Business Account"
3. Select category: "Agriculture" or "Education"
4. Confirm switch

### Step 3: Configure the Profile
1. Tap "Edit profile"
2. **Profile photo**: tap the profile photo > "Upload photo" > select seedwarden_logo_1.png
3. **Name**: `Seedwarden`
4. **Bio**: Copy and paste exactly (80 character limit — tight):
   ```
   Field guides for growers + foragers
   Free zone card in bio
   ```
5. **Link**: Add Kit landing page URL (one link allowed on business accounts)
   - Add once Kit landing page is live
6. Tap "Save"

### Step 4: Confirm Account Is Ready
- [ ] Account exists at tiktok.com/@seedwarden (or fallback)
- [ ] Account type: Business
- [ ] Profile photo uploaded
- [ ] Bio entered (80 chars max)
- [ ] Link set to Kit landing page URL

"cross-post to TikTok" feature. TikTok's algorithm suppresses content detected as
cross-posted. Film separately or upload the same video file as a separate native upload.

## Pinterest

### Step 1: Create the Account
1. Go to pinterest.com
2. Click "Sign up"
3. Enter: wanka95@gmail.com
4. Set a password
5. After signup: Pinterest may default to a personal account — proceed to Step 2 immediately

### Step 2: Switch to a Business Account
1. Click your profile icon (top right) > "Settings"
2. Under "Account settings" > "Account changes" > "Convert to business"
   (Or: go to business.pinterest.com and connect your personal account)
3. Business type: "Online retailer" or "Media" — either works
4. Fill in the business name: `Seedwarden`

### Step 3: Configure the Profile
1. Click your profile icon > "Edit profile"
2. **Profile photo**: upload seedwarden_logo_1.png (reads at 165px on Pinterest)
3. **Display name**: `Seedwarden`
4. **About your profile** (160 character limit):
   ```
   Seedwarden — heirloom seeds, wild edibles, food preservation + zone-specific growing guides.
   Practical field guides for real food growers.
   ```
5. **Website**: Enter the Kit landing page URL (higher priority than seedwarden.com for list building)
   - If seedwarden.com is registered, enter it here instead — Kit link is the bio link, not website field
6. Tap "Save"

### Step 4: Claim Your Website (Optional, Recommended)
1. In Settings > "Claimed accounts" — claim seedwarden.com if the domain is registered
2. This unlocks Rich Pins (automatically pulls product title and price from Etsy listings)
3. If domain is not yet registered, skip — this is a Phase 3 enhancement

### Step 5: Enable Rich Pins (If Website Claimed)
1. Go to developers.pinterest.com/tools/url-debugger
2. Enter your Etsy shop URL or Kit landing page URL
3. Request validation
4. Rich Pins can take 24-48 hours to activate

### Step 6: Confirm Account Is Ready
- [ ] Account exists at pinterest.com/seedwarden (or fallback)
- [ ] Account type: Business (shows Analytics tab in top nav)
- [ ] Profile photo uploaded
- [ ] Bio entered (160 chars max)
- [ ] Website field filled (Kit landing page URL or seedwarden.com)
- [ ] Handle matches Instagram and TikTok handle
