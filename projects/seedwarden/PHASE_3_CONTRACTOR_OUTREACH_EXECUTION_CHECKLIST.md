## Troubleshooting

### Email Bounces / Delivery Failures

**Symptom**: Email marked as "undelivered" or "bounced" within 1 hour of sending

**Action**:
1. Check the recipient email address in PHASE_3_CONTRACTOR_POOL.md
2. If address is incorrect, find the correct address by:
   - Searching the contractor's website for a contact form or published email
   - Checking LinkedIn for direct message option (if available)
   - Searching for "[Contractor Name] + email + hire" on Google
3. Re-send email with corrected address
4. Update CONTRACTOR_OUTREACH_SEND_LOG.md: Add new row with corrected address and note "RE-SENT: [Original error message]"

### Contact Says No / Not Interested

**Symptom**: Contractor responds (before June 28) saying they are unavailable or not interested

**Action**:
1. Update RESPONSE_TRACKING_TEMPLATE.md: Mark [Response Status] as "NO RESPONSE" and [Fit Score] as "N/A"
2. Move to backup candidate list (in PHASE_3_CONTRACTOR_SELECTION_TIMELINE.md, Section 5)
3. Send backup email to next-tier candidate same day
4. Note in send log: "[Original candidate name] declined. Outreach to [backup name] sent [date]."

### Need More Info Before Deciding / Questions About Scope

**Symptom**: Contractor responds with questions about deliverables, timeline, or rates before committing

**Action**:
1. Update RESPONSE_TRACKING_TEMPLATE.md: Mark [Response Status] as "CLARIFICATION REQUESTED"
2. Send clarification email within 24 hours (use template in PHASE_3_COMMUNICATION_TEMPLATES.md)
3. Follow up: If no response within 48 hours (e.g., by June 25), send one more clarification email marked "Final clarification before decision deadline: June 28 EOD"
4. Document all clarification exchanges in CONTRACTOR_OUTREACH_SEND_LOG.md notes field

### Email Client Issues / Can't Send

**Symptom**: Email client crashes, internet down, or spam filter blocks emails

**Action**:
1. Try alternate email client (Gmail web if Outlook desktop fails, or vice versa)
2. For Gmail: If suspicious activity warning appears, approve access and re-send
3. For Outlook: Disable antivirus temporarily if blocking emails, or send from Gmail as fallback
4. If still unable to send: Create a text file with all 11 email drafts and share with a trusted assistant to send on your behalf (note in CONTRACTOR_OUTREACH_SEND_LOG.md "Sent by [Assistant Name] due to [Reason]")
