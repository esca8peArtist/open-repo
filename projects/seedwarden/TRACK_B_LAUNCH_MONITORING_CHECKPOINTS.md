# Track B Launch Monitoring Checkpoints

**Launch date**: May 30, 2026

**Day 7 checkpoint**: June 6, 2026

## What to Measure

These metrics are readable without any analytics infrastructure beyond what you can count manually.

| Metric | Where to Check | How to Count |
|--------|---------------|--------------|
| Gist views | gist.github.com — check the view counter on the Gist page (visible to owner) | Record the number; subtract prior checkpoint count to get delta |
| PDF clicks / downloads | GitHub raw URL click count (not natively tracked) — use Gist view count as proxy OR set up a free Bitly link to the Gist and track there | Bitly link click count if set up; otherwise Gist views |
| Reddit post engagement | Your Reddit profile > submitted posts > check upvotes + comments on each zone card post | Upvotes + comment count per post |
| Instagram engagement | Instagram Insights (available on creator/business accounts) | Reach, likes, comments, link-in-bio clicks |
| TikTok views | TikTok app > your videos | View count + comment count |
| Direct messages / inquiries | Instagram DMs, Reddit replies, email | Count of inbound messages referencing Zone Cards |
| Social shares / reposts | Instagram Stories mentions, Reddit cross-posts, TikTok duets/stitches | Count of times others shared your content |

## Checkpoint 1 — Day 3 (June 2, 2026)

**Time to complete**: 5 minutes
**Decision question**: Is initial distribution working? Should outreach be escalated?

### Metrics to Record

| Metric | Target | Record Here |
|--------|--------|-------------|
| Gist views (cumulative since May 30) | 50+ | |
| Reddit post upvotes (total across all zone card posts) | 20+ | |
| Instagram post reach | 200+ | |
| TikTok video views | 100+ | |
| Direct messages or download requests | 2+ | |
| Herbalist network contacts who shared | 1+ | |

### Decision Triggers

**If Gist views < 50 by Day 3**:
- Escalate outreach: send Tier 3 contacts (UpS, Strictly Medicinal Seeds, Seattle Herbalism Society) if not yet contacted
- Re-post in one additional Reddit community that approved posts on May 28
- Add one more Instagram Story with a direct link sticker pointing to the Gist

**If Gist views 50–100 by Day 3**:
- Distribution is working at baseline. No action change needed.
- Focus on responding to any comments or DMs to maintain engagement momentum.

**If Gist views > 100 by Day 3**:
- Strong launch signal. Proceed to Tier 2 planning: begin planning a paid zone-card bundle (Etsy listing) as a follow-on product.
- Email the contacts who expressed interest in affiliate arrangements and confirm terms.

**If 0 Reddit traction (0 upvotes or posts removed)**:
- Check if posts were approved by mods. If removed, message the mod team with the pre-approved community resource framing from the outreach template.
- Do not repost the same content — revise the title to emphasize the free/educational angle.

**Day 3 verdict**: Mark as GREEN (on track), YELLOW (below baseline, escalate outreach), or RED (no traction, diagnose distribution blocker).

## Checkpoint 2 — Day 7 (June 6, 2026)

**Time to complete**: 8 minutes
**Decision question**: Is organic sharing happening? What is the conversion rate from views to engagement?

### Metrics to Record

| Metric | Target | Record Here |
|--------|--------|-------------|
| Gist views (cumulative) | 150+ | |
| Reddit post upvotes (total) | 50+ | |
| Instagram reach (total across all launch posts) | 500+ | |
| TikTok views (total) | 300+ | |
| Organic shares / reposts by others | 3+ | |
| New followers gained (Instagram) | 20+ | |
| Email or DM inquiries | 5+ | |
| Herbalist network contacts who responded | 3+ | |

### Decision Triggers

**If Gist views < 100 by Day 7**:
- Distribution has stalled. Primary action: post in the one Reddit community not yet reached (the third subreddit from your list). Secondary action: reach back to any Tier 1 contacts who have not responded and follow up with a one-sentence message ("Did you get a chance to look at the Zone Cards? Happy to share any of them with your community").

**If Gist views 100–200 by Day 7**:
- Solid baseline. No escalation needed.
- Begin drafting the Etsy product listing for the paid Zone Card bundle (this can be a $5–9 "full bundle" that bundles all 8 as a single download with a cover page and bonus content).

**If Gist views > 200 by Day 7**:
- Strong organic traction. Proceed immediately to Tier 2 planning:
  - Build the Etsy listing for the paid bundle
  - Begin email list building (add a Kit/ConvertKit form to the landing page)
  - Consider a second content wave (zone-specific deep-dive posts, one per zone, over the following two weeks)

**If organic shares = 0 by Day 7**:
- The content is being viewed but not shared. This usually means the format is not shareable enough from the current distribution point (i.e., a PDF link is less shareable than an image post).
- Action: Create an Instagram carousel that previews 2–3 cards visually and is shareable as a post. Carousels get saved and shared at higher rates than single images.

**Day 7 verdict**: Mark as GREEN, YELLOW, or RED. If GREEN, begin Etsy listing draft. If YELLOW, escalate second Reddit wave. If RED, schedule a diagnosis session.

## Checkpoint 3 — Day 14 (June 13, 2026)

**Time to complete**: 10 minutes
**Decision question**: What is the two-week baseline? What does this tell us about Phase 3 (practitioner tier) timing?

### Metrics to Record

| Metric | Target (Two-Week Baseline) | Record Here |
|--------|---------------------------|-------------|
| Gist views (cumulative) | 300+ | |
| Reddit total upvotes | 100+ | |
| Instagram total reach | 1,000+ | |
| TikTok total views | 500+ | |
| Organic shares by others | 8+ | |
| New followers (Instagram) | 50+ | |
| Affiliate interest confirmed | 1+ | |
| Etsy listing live (paid bundle) | Yes / No | |

### Decision Triggers

**If cumulative Gist views < 150 by Day 14**:
- Zone Cards are not achieving distribution breakthrough. Before investing in Phase 3 (practitioner tier), diagnose the distribution gap.
- Hypotheses to test: (a) the Gist is not the right distribution point — try Etsy free listing or a Gumroad free product instead; (b) the outreach did not reach the right communities — identify which Reddit posts performed and which did not; (c) the product title needs refinement for search ("Zone Quick-Start Cards" may not be the most searched phrase).

**If cumulative Gist views 150–400 by Day 14**:
- Baseline distribution is working. Track B is viable as a free lead-generation product.
- Next step: Build the paid Etsy listing. Set price at $5–9 for all 8 cards as a bundle. Include a cover page, a zone-selection guide ("which zone am I in?"), and a bonus seasonal planting calendar as added value for the paid version.

**If cumulative Gist views > 400 by Day 14**:
- Zone Cards have strong organic traction. This validates the product format and audience.
- Next step: Phase 3 practitioner tier is de-risked. Begin June 22 sprint with confidence that the Zone Card format has a proven audience.
- Consider a social media growth sprint in late June: paid promotion on Instagram or Pinterest, $20–50 test budget, targeting gardening and homesteading interests.

**Herbalist outreach assessment**:
- Count how many of the 18 contacts responded (any response counts).
- If 3+ responded: outreach strategy is working. Proceed with same approach for Phase 3.
- If 1–2 responded: adjust approach for Phase 3 — the email/DM format may need a stronger subject line or a more specific hook per contact.
- If 0 responded: the outreach channels may not be the right path for this product. Reddit and Pinterest organic search may be the primary drivers instead.

**Day 14 verdict**: Mark as GREEN (strong traction, proceed to Phase 3 and paid listing), YELLOW (baseline traction, proceed with adjustments), or RED (low traction, diagnose before Phase 3 investment).
