# Phase 3 Medicinal Herbs — Photo Attribution Log

**Prepared**: 2026-05-22 (pre-staged by agent; user fills confirmed image selections June 1–21)

---

## How to Use This Log

1. For each species, search the Wikimedia Commons URL listed.
2. Select the image(s) you download for the guide. Prefer CC-BY-SA or CC0 licenses.
3. Fill in the confirmed columns: filename, source URL, photographer/uploader, license.
4. For iNaturalist sources: confirm the specific observation's license (it is per-observation, not per-species).
5. Download images to `assets/phase-3-medicinal-herbs/[species-slug]/` using the filename convention shown.
6. The "Attribution string" column is the exact text that goes in the guide PDF attribution page.
7. Log is complete when all species have at least a habit image confirmed with no blank fields in the confirmed columns.

**Minimum per species**: 1 confirmed image (habit). Ideal: 2–4 images (habit, flower, root/preparation, lifestyle).
