## Timing: How Many Per Day

**Day 1 (Monday, Apr 28) — Shop setup + first 2 listings**
- Complete shop profile, policies, banner, and icon (see pre-upload checklist below)
- Upload listing #1 (Companion Planting Chart) at 10–11 AM local time
- Upload listing #2 (Food Sovereignty Starter Guide) at 2–3 PM
- Keep shop in "vacation mode" or "private" so listings aren't indexed until all 6 are live

**Day 2 (Tuesday, Apr 29) — Middle batch**
- Upload listing #3 (Anti-Catalog) at 10 AM
- Upload listing #4 (Seed Saving Field Manual) at 2 PM
- Verify first two listings display correctly on mobile and desktop

**Day 3 (Wednesday, Apr 30) — Final batch + go live**
- Upload listing #5 (Apartment Plant Catalog) at 9 AM
- Upload listing #6 (Survival Garden Regional Plans) at 11 AM
- Switch shop from private/vacation to Active at noon (midday = peak browsing window)
- Announce via email list and social media immediately after going public

**Why not upload all at once?**
Etsy's recency signal applies per listing, not per shop. Six listings created simultaneously
all get the same timestamp and compete with each other for the same recency window. Spacing
by 2–4 hours within a day, and over 2–3 days, staggers the recency boosts and gives each
listing a slightly longer window of elevated visibility.

## Step-by-Step Upload Checklist (Per Listing)

Complete each step in order for every listing. Do not skip steps — all fields contribute
to Etsy's relevancy matching.

### Step 0: Before Creating Any Listing (One-Time Shop Setup)

- [ ] Shop name confirmed as `seedwarden` (or whatever final name is chosen)
- [ ] Shop icon uploaded: use `logos/seedwarden_logo_1.png` (1024x1024 PNG, correct format for Etsy icon)
- [ ] Shop banner uploaded: use `logos/etsy_banner.png` (2064x512 PNG — meets Etsy's 3360x840 recommended but 2064x512 is functional; consider regenerating at 3360x840 for full-bleed on wide screens)
- [ ] Shop announcement written: copy from `etsy-store-copy.md` shop announcement (126 chars — fits 160 char limit)
- [ ] About section written: copy from `etsy-store-copy.md` shop description
- [ ] Policies written: copy from `etsy-store-copy.md` shop policies (processing/delivery, refunds, response time)
- [ ] Listing category pre-selected: "Craft Supplies & Tools > Garden Supplies > Gardening Guides" OR "Books, Movies & Music > Books > Nonfiction" — pick one and use it consistently
- [ ] Payment method linked (Etsy Payments required for most markets)
- [ ] Digital delivery confirmed: "Instant download after purchase"
- [ ] Tax settings reviewed (Etsy auto-collects in most US states — verify your state if you have nexus)

---

### Step 1: Create Listing

In Etsy Seller Dashboard → Listings → Add a listing:

- [ ] **Listing type**: Digital download
- [ ] **Title**: Paste exactly from `etsy-store-copy.md` for this product (all titles verified under 140 chars)
- [ ] **Category**: Craft Supplies & Tools > Garden Supplies > Gardening Guides
- [ ] **Price**: Enter the product price (see table above)
- [ ] **Quantity**: Leave as 1 (Etsy handles digital inventory automatically)

---

### Step 2: Upload Photos

- [ ] Click "Add a photo" → select the product mockup from `projects/seedwarden/mockups/`
- [ ] Upload the matching `-mockup.png` file (e.g., `food-sovereignty-starter-guide-mockup.png`)
- [ ] Confirm thumbnail shows the product title clearly in the preview
- [ ] Drag the mockup to position 1 (primary image slot) if it was not uploaded first
- [ ] Slots 2–10 are empty for Phase 1 — acceptable, fill in Phase 2

---

### Step 3: Write the Description

- [ ] Open `etsy-store-copy.md` and locate the product's listing description
- [ ] Copy the full description text
- [ ] Paste into Etsy's description field
- [ ] Verify the opening hook appears first (before "What's inside:")
- [ ] Verify the "instant digital PDF download" delivery line is present near the end
- [ ] Verify cross-sell references are present where written (e.g., Seed Saving Manual mentions Food Sovereignty Guide)
- [ ] Check description does not contain: external URLs, promises of specific income, all-caps shouting, keyword stuffing

**Description length check**: Etsy allows up to 20,000 characters in descriptions. All current
descriptions are 400–700 words (~2,500–4,500 characters) — well within limits and within the
400–600 word sweet spot for conversion.

---

### Step 4: Add Tags

- [ ] Open `etsy-store-copy.md` and locate the product's tags line
- [ ] Enter all 13 tags
- [ ] **IMPORTANT**: Several tags in the current documentation exceed Etsy's 20-character limit per tag.
  Before entering tags, use the corrected tag sets from the Tags Corrections section at the bottom
  of this document — do not copy raw tags from `etsy-store-copy.md` without checking length first.
- [ ] Confirm no tag exceeds 20 characters (Etsy will reject or truncate silently)
- [ ] Confirm all 13 tag slots are filled

---

### Step 5: Attach the Digital File

- [ ] Scroll to "Digital files" section in the listing form
- [ ] Click "Upload file" → navigate to `projects/seedwarden/scripts/output/`
- [ ] Select the matching PDF (e.g., `food-sovereignty-starter-guide.pdf`)
- [ ] Confirm upload completes (progress bar reaches 100%)
- [ ] Verify file name shown in the listing matches the product
- [ ] Confirm file size is shown (should be 650–900 KB for all Phase 1 products)
- [ ] Delivery setting: "Instant download"

---

### Step 6: Fill Remaining Fields

- [ ] **Renewal**: Automatic (Etsy charges $0.20 every 4 months — leave on automatic)
- [ ] **Shop section**: Create "Digital Guides" section if not already present; assign all products to it
- [ ] **SEO slug**: Etsy generates this from the title automatically. After publishing, verify the URL slug
  is readable (e.g., `/listing/123456789/food-sovereignty-guide-for-beginners`). If it looks garbled,
  edit the title slightly and Etsy will regenerate.
- [ ] **Production partners**: Leave blank (no partners)

---

### Step 7: Publish

- [ ] Click "Publish" (or "Save as draft" if shop is still private and you are not yet ready to go live)
- [ ] If shop is in private/draft mode: use "Save as draft" for all listings until Day 3 launch
- [ ] If going live immediately: click "Publish" — listing is now active and indexing begins
