## Quick-Reference: Source-Treatment-Storage Combinations by Scenario

### Rural Landowner — High Resilience Configuration

**Primary source**: Deep drilled well (confined aquifer) + hand pump backup
**Secondary source**: 2,500-gal below-ground HDPE rainwater cistern (filled primarily from March-April snowmelt and spring rains)
**Tertiary (emergency)**: Farm pond or stream — full 5-stage treatment required
**Treatment — well water**: No treatment required if confined aquifer; annual testing only
**Treatment — cistern water (drinking)**: Ceramic gravity filter + calcium hypochlorite maintenance dose (0.5 ppm residual)
**Treatment — surface water (emergency)**: 5-stage sequence: settle → coarse filter → ceramic filter → hypochlorite → boil; or UV if solar-powered system available
**Storage**: 2,500-gal buried HDPE (primary buffer) + 330-gal IBC tote seasonal overflow (drain in October)
**Greywater**: L2L for laundry (immediate) → subsurface constructed wetland (6-18 months later)
**Blackwater**: Gravity-fed conventional septic; composting toilet for new construction or ADU
**Governance**: Written well-sharing agreement if neighbors share aquifer; annual testing log; shock chlorination protocol posted at wellhead
**Cost range**: $12,000–$35,000 total (dominated by well drilling if no well exists; $2,000–$5,000 if well exists)

### Suburban Homeowner — Moderate Resilience Configuration

**Primary source**: Municipal water (continue during normal operations)
**Secondary source**: 750-gal below-ground HDPE cistern, fed from roof downspout collection (March-November active collection)
**Tertiary (emergency)**: WaterBob bathtub bladder (100 gal, filled from tap before disruption)
**Treatment — stored tap water**: Already treated (chlorinated); retest with test strips every 6 months
**Treatment — cistern water (drinking)**: Ceramic gravity filter + hypochlorite dose; store in sanitized food-grade containers
**Treatment — emergency (any source)**: Ceramic filter + boiling; or UV if solar panel available
**Storage**: 750-gal buried cistern + 100-gal WaterBob emergency fill
**Greywater**: L2L laundry redirect in summer; no winter greywater reuse required at suburban scale
**Blackwater**: Municipal sewer (typical); if septic, verify system type and power dependency
**Total stored capacity**: ~850 gallons → approximately 60 days drinking water at 1 gal/person/day for 4 people; 14 days at 5 gal/person/day comfortable use
**Cost range**: $1,500–$4,000 total (cistern + collection plumbing + purification consumables)

## Decision Flowchart: Source Selection

```
START: Do you have an existing well?
├── YES → What depth and aquifer type?
│   ├── 100+ feet, confined aquifer → EXCELLENT. Install hand pump backup ($300-900). Annual testing. DONE.
│   ├── 50-100 feet, unconfined/drift → MODERATE. Test annually for nitrates and bacteria.
│   │   ├── Nitrates >8 mg/L → Supplement with rainwater cistern for drinking water.
│   │   └── Nitrates <8 mg/L → Hand pump backup. Annual testing. DONE.
│   └── <50 feet, surface water influenced → HIGH RISK. Test before each use. Plan alternate source.
│
└── NO → Does your property have a spring with 2+ gpm year-round flow?
    ├── YES → Develop spring box. Verify August flow rate. Gravity distribution. Annual coliform test.
    └── NO → Is drilling permitted and affordable on your property?
        ├── YES → Drill to confined aquifer. Install hand pump. Primary source established.
        └── NO (suburban or cost-constrained) → Rainwater cistern is your primary resilience investment.
            ├── Space for below-ground cistern? → 750-2,500 gal HDPE, 48-60 in depth.
            └── Space-constrained → 275-500 gal above-ground (seasonal) + indoor barrels for winter.
```
