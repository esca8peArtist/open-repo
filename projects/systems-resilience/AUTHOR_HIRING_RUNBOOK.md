# Author Hiring Runbook — Phase 6

**Owner**: Orchestrator  
**Timeline**: Start outreach May 28–29; confirmation by May 30 16:00 UTC (primary) or June 3 08:00 UTC (contingency)

---

## Pre-Outreach Setup (30 minutes — do first)

- [ ] Confirm which domain has been selected (from user CHECKIN.md entry)
- [ ] Pull the relevant domain candidate file: `phase-6-candidate-community-economic-resilience.md`, `phase-6-candidate-skills-development.md`, or `phase-6-candidate-governance-scaling.md`
- [ ] Pull the author onboarding kit: `phase-6-author-onboarding-kit.md`
- [ ] Confirm project rate range approved by user (see Section 2 of `PHASE_6_AUTHOR_HIRING_AND_ONBOARDING_FRAMEWORK.md`)
- [ ] Create outreach tracking log (copy the table structure from `author-outreach-tracking.md`)

---

## Outreach Execution (Day 1)

**Step 1**: Build candidate shortlist (3–5 candidates, from source channels in Framework Section 2)

- [ ] Search primary channel for domain (academic network, practitioner org, or freelance platform)
- [ ] Identify 3–5 candidates with writing samples accessible for review
- [ ] Log candidates in outreach tracking log: Name | Source | Contact | Writing sample URL | Status

**Step 2**: Review writing samples before outreach

- [ ] Read at least one writing sample per candidate
- [ ] Confirm sample shows practitioner voice (not pure academic)
- [ ] Confirm sample shows domain-relevant knowledge signal
- [ ] Mark candidates as PROCEED or SKIP based on sample review

**Step 3**: Send outreach emails (use `AUTHOR_ONBOARDING_TEMPLATE.md` outreach email section)

- [ ] Send to all PROCEED candidates simultaneously
- [ ] Log send time in outreach tracking log
- [ ] Set response deadline (3–5 business days from send)
- [ ] Calendar reminder for follow-up if no response in 48 hours

---

## Candidate Qualification Screen (Day 2–4)

For each candidate who responds:

**Gate 1 — Writing voice** (15 minutes):
- [ ] Open candidate's submitted writing sample (or sample from outreach review)
- [ ] Compare to model document: `phase-5-wave-2-community-implementation-playbook.md`
- [ ] Does voice read practitioner (clear, direct, actionable) or academic (passive, hedge-heavy, jargon-dense)?
- [ ] PASS / FAIL — note in tracking log

**Gate 2 — Domain knowledge signal** (15 minutes):
- [ ] Review candidate background against domain-specific profile in Framework Section 1
- [ ] If unclear from written materials: schedule 15-minute call
- [ ] Does candidate demonstrate relevant domain knowledge? (cooperative systems for A, adult learning for C, governance theory for D)
- [ ] PASS / FAIL — note in tracking log

**Gate 3 — Timeline confirmation** (5 minutes — explicit written confirmation):
- [ ] Send one-sentence follow-up: "Can you confirm you are available 20–25 hours/week from June 1 to August 9?"
- [ ] PASS if yes, specific, with no conflicting commitment disclosures
- [ ] FAIL if candidate hedges, discloses a major July commitment, or says "roughly" or "probably"
- [ ] Note in tracking log

Candidates who pass all three gates move to confirmation. All others are logged as declined or screened out.

---

## Confirmation (Day 4–5)

When first qualifying candidate is identified:

- [ ] Send project brief (from `AUTHOR_ONBOARDING_TEMPLATE.md` confirmation section)
- [ ] Send agreement letter for signature (see template note below)
- [ ] Once signed: process first milestone payment (20%)
- [ ] Log confirmation status in `author-outreach-tracking.md`: CONFIRMED with date/time

**If no qualifying candidate by May 30 14:00 UTC**: Begin contingency sequence (see `author-outreach-tracking.md` Section "Contingency Activation Checklist").

**Agreement letter note**: Use a short-form freelance agreement (1–2 pages) covering: project scope, payment milestones (20/30/30/20), delivery dates, intellectual property assignment to the project (all work produced is project property), confidentiality (project materials are not for external publication or sharing), and termination (either party can terminate with 7 days notice; completed work is compensated proportionally).
