# California Building Codes, Permits, and Inspections for the Residential GC

> **Use this document as the working reference before every project.** It is structured for fast lookup: jump to a section, scan the table, return to the field. Every dollar lost on a stop-work order, failed inspection, or permit redo is dollars you can keep by knowing what's in here.

The single most important habit: **call the local building department before you bid.** California sets the state code floor; every city and county can layer amendments and local interpretations on top. The CRC says one thing; Los Angeles, San Francisco, and Yolo County each enforce it slightly differently.

---

## 1. California Code Framework

### 1.1 How California Codes Are Adopted

California operates on a **triennial code adoption cycle**. The Building Standards Commission (BSC) adopts the model codes (mostly ICC family), incorporates California-specific amendments through state agencies (HCD, DSA, OSHPD, CEC), publishes the result as **Title 24 of the California Code of Regulations**, and ships it to local jurisdictions for adoption.

The current cycle is the **2025 California Building Standards Code**, published July 1, 2025, **effective January 1, 2026.** Any permit application submitted on or after that date is reviewed under the 2025 code. Applications submitted before that date are typically grandfathered under 2022 code, **but only if the permit issues within a defined window** (usually 180 days from application).

There is also a less-publicized **intervening code adoption cycle** in between triennials. The BSC can amend code mid-cycle (e.g., 2024 intervening update). Always confirm the version your jurisdiction has currently adopted.

**Local amendments**: Cities and counties can amend the state code, but only to make it more stringent (with limited exceptions for local climate, geology, or topography). They must file the amendments with HCD and BSC. **Always pull the local amendments before bidding** — they are usually published on the building department's website or in the municipal code.

### 1.2 CBC vs. IBC, CRC vs. IRC

California does **not** directly enforce the IBC or IRC. California adopts the ICC model code and re-promulgates it as the CBC and CRC with state amendments. When a designer or product datasheet cites "IBC 2021" or "IRC 2021," you must confirm whether California amended that specific section.

| Code | Full Name | Title 24 Part | Scope |
|------|-----------|---------------|-------|
| **CBC** | California Building Code | Part 2 | All buildings not covered by CRC |
| **CRC** | California Residential Code | Part 2.5 | Detached one- and two-family dwellings + townhouses ≤ 3 stories with separate egress |
| **CPC** | California Plumbing Code | Part 5 | All plumbing systems (based on UPC, not IPC) |
| **CMC** | California Mechanical Code | Part 4 | All mechanical/HVAC (based on UMC, not IMC) |
| **CEC** | California Electrical Code | Part 3 | All electrical (NEC + CA amendments) |
| **CalGreen** | California Green Building Standards | Part 11 | Sustainability mandates (all occupancies) |
| **Energy Code** | California Energy Code | Part 6 | Energy efficiency (Title 24, the famous one) |
| **CFC** | California Fire Code | Part 9 | Fire protection (based on IFC) |
| **CEBC** | California Existing Building Code | Part 10 | Alterations, additions, repairs to existing |
| **CHBC** | California Historical Building Code | Part 8 | Qualified historic structures |

### 1.3 Which Code Applies — CRC vs. CBC

**Use the CRC if the project meets ALL these conditions:**
- Detached one- or two-family dwelling, **OR** townhouse with separate egress and not more than three stories above grade plane
- Not exceeding three stories
- Each townhouse separated by a 1-hour fire-rated wall (or two 1-hour walls per CRC R302.2)

**Use the CBC for everything else**, including:
- Apartment buildings (R-2 occupancy with shared egress)
- Mixed-use buildings
- Any residential over 3 stories
- ADUs are governed by the CRC if attached to or detached from a CRC-regulated dwelling; if attached to an apartment building, they fall under CBC

This distinction matters because the CRC has prescriptive tables (joist spans, header sizes, wall bracing) that don't require engineering. The CBC defaults to engineered design.

### 1.4 The Other Codes That Will Bite You

| Code | Where the GC most often gets surprised |
|------|----------------------------------------|
| **CPC** | Trap arm lengths, vent termination heights, water heater seismic strapping (two straps, upper third + lower third), gas piping sizing, condensate disposal |
| **CMC** | Combustion air requirements for water heaters in confined spaces; duct sealing per Title 24; clothes dryer venting (no more than 35 ft of equivalent length) |
| **CEC** | AFCI required in nearly all habitable spaces; GFCI in expanded locations (now includes laundry, basements, dishwasher receptacle); tamper-resistant receptacles required almost everywhere; working clearance at panels (3 ft × 6.5 ft × width of equipment) |
| **CalGreen** | 65% C&D waste diversion (raised from 50% in 2022 code); low-VOC documentation; water-conserving fixtures; pre-wiring for EV |
| **Title 24 Part 6** | HERS-verified measures (duct leakage, envelope leakage, refrigerant charge, QII insulation); PV mandate; high-efficacy lighting |

### 1.5 What Local Layers Add

Local jurisdictions stack these **on top of** the state code:
- **Local amendments to the state code** (must be more stringent)
- **Zoning code** (setbacks, height, use, lot coverage, FAR — see §7)
- **Subdivision/grading ordinance** (slope, drainage, erosion control)
- **Local fire code amendments** (sprinkler thresholds, defensible space in WUI zones)
- **Specific plan or historic overlay**
- **HOA CC&Rs** (private — not enforced by city, but they can sue and force you to undo work)
- **Coastal Commission** (any work seaward of the coastal zone boundary)
- **BCDC** (San Francisco Bay shoreline)
- **HCD** (mobile home parks and factory-built housing)

### 1.6 State Housing Law — What You Can Build Without a Variance

California has progressively preempted local zoning to force more housing. As a residential GC, this is **a major opportunity** because it expands what you can build "by right" (ministerial approval, no public hearings).

| Law | What it does | Practical effect |
|-----|--------------|------------------|
| **ADU State Law** (Gov. Code §66310 et seq.) | Mandates ministerial approval of ADUs and JADUs statewide | Most lots in California can add an ADU + JADU without discretionary review |
| **SB 9** (2021, Gov. Code §65852.21 / §66411.7) | Ministerial approval of duplex on single-family lots; lot splits into two; up to 4 units total | Convert SFR lot → duplex + lot split → 4 units, with limited local objection grounds |
| **AB 2221** (2022) | Tightened ADU law; capped local design review; codified height rules | Detached ADU height: 16 ft baseline, 18 ft if within 1/2 mi of transit, 25 ft if attached |
| **SB 897** (2022) | Allowed up to 25-ft height for attached ADUs; reduced demolition findings | Easier to attach ADU to existing home, even older non-conforming structure |
| **SB 1211** (2024) | Up to 8 detached ADUs on multifamily lots | Major opportunity on existing apartment parcels |
| **AB 1033** (2023) | Allows ADUs to be sold separately as condos (local opt-in) | Some jurisdictions now permit ADU condo sales |
| **RHNA** (Regional Housing Needs Allocation) | Each city must zone for a state-mandated number of housing units | Cities behind on RHNA lose discretionary review power (Builder's Remedy) |

**Practical advice**: For an ADU project, start by checking your jurisdiction's ADU ordinance against the latest state law. If the local rule is more restrictive than state law on a topic state law preempts (e.g., owner-occupancy, parking near transit, minimum lot size), the state rule wins — and the jurisdiction is legally required to approve.

## 2. Permits — What Requires One

### 2.1 The General Rule

Per CRC R105.1 / CBC §105.1: **Any owner or authorized agent who intends to construct, enlarge, alter, repair, move, demolish, or change the occupancy of a building or structure, or to erect, install, enlarge, alter, repair, remove, convert, or replace any electrical, gas, mechanical, or plumbing system, the installation of which is regulated by this code, or to cause any such work to be done, shall first make application to the building official and obtain the required permit.**

In practice, most jurisdictions exempt minor work via local "exempted work" lists. The default is **when in doubt, call the building department**.

### 2.2 Projects That Always Require a Permit

| Project | Code reference / reason |
|---------|------------------------|
| New construction (including ADUs and JADUs) | CRC R105.1 |
| Additions of any size | CRC R105.1 |
| Structural alterations (remove/move/modify walls, beams, posts) | CRC R105.1 |
| Foundation work | CRC R105.1 |
| Re-roof over 100 sq ft (one square) in most jurisdictions | CRC R105.2 (varies locally) |
| Window/door replacement when changing size, header, or shear wall | CRC R105.1 |
| New or relocated electrical circuits | CEC + local rule |
| Electrical service or panel upgrade | CEC + local rule |
| New HVAC system, A/C, furnace, ductwork | CMC + Title 24 |
| Water heater replacement | CPC (most jurisdictions; see §2.3) |
| New plumbing fixtures or relocated drains/vents | CPC |
| Gas piping work | CPC |
| Retaining walls over 3 ft (or 4 ft in some areas) from bottom of footing | CBC §1807 |
| Decks more than 30 inches above grade | CRC R507 |
| Swimming pools, spas | CBC + local pool barrier ordinance |
| Sheds over 120 sq ft (some areas 200 sq ft) | CRC R105.2 |
| Solar PV systems | Title 24 + local |
| Backup batteries (Powerwall etc.) | CEC + Fire Code |
| Driveway approach in public ROW | Encroachment permit (Public Works) |
| Grading over the local threshold (often 50 cy or any cut/fill > 3 ft) | Grading permit |
| Demolition of any structure | Demolition permit + sometimes air district notification (asbestos NESHAP) |
| Tree removal of regulated species (heritage/protected) | Local tree ordinance permit |

### 2.3 Projects That Typically Do NOT Require a Permit

Per CRC R105.2 (state baseline; **always confirm locally — many cities pull from this list**):
- Painting, papering, tiling, carpeting, cabinets, counter tops, and similar finish work
- Like-for-like fixture replacement (faucets, toilets, sinks, lighting fixtures) where no piping/wiring is relocated
- Floor coverings (carpet, sheet vinyl, hardwood)
- Window awnings supported by exterior wall, not projecting more than 54"
- Prefabricated swimming pools < 24" deep, < 5,000 gal
- Retaining walls ≤ 4 ft from bottom of footing to top of wall, no surcharge
- Fences not over 7 ft
- Sidewalks and driveways not in a public ROW
- Re-roof under 100 sq ft (some jurisdictions: no permit needed for spot repairs)

**Important nuance:** Many jurisdictions still require a permit for water heater replacement even though it's like-for-like, because of seismic strapping, T&P discharge routing, gas connection, and combustion air. **Most California jurisdictions require a water heater permit. Confirm before you swap one out.**

### 2.4 The "Unpermitted Work" Problem

Common scenarios:
- Owner bought a house with a converted garage, a finished attic, or an "office in the backyard" that has no permit history
- Previous owner DIY'd an addition, never pulled a permit
- Owner did permitted work but final inspection was never called — permit expired, technically the work is unpermitted

**Risks:**
- **To the owner**: Required disclosure under Civ. Code §1102 (Transfer Disclosure Statement); lender refusal to finance unless cured; insurance refusal/non-coverage of damage; safety hazards (commonly: bad electrical, no vapor barrier, undersized framing); city can require demolition; assessor can re-bill back taxes.
- **To the GC who built it**: CSLB discipline (B&P §7110 — failure to comply with building laws), civil liability to current and future owners.
- **To the GC who later works on it**: When you pull a permit on a property with unpermitted work, the building department can require that you bring the unpermitted work into compliance as a condition of approval — sometimes even if your scope doesn't touch it. **Disclose the existing conditions in writing to the owner before signing the contract**, and define in the contract who pays if the city forces remediation.

**Paths to legalize:**

| Path | When to use | What it costs |
|------|-------------|---------------|
| **Retroactive / as-built permit** | Work is fundamentally code-compliant or can be brought into compliance with surgical fixes | 2x–10x normal permit fee + investigation fee + cost of code corrections + as-built plans (architect/engineer) |
| **Demolition + rebuild under new permit** | Work is so non-compliant or in such a bad location (setback violation, no foundation) that legalization is impossible or more expensive than rebuilding | Cost of demo + new construction + permits |
| **Remove and don't replace** | Owner doesn't need the space; cheapest option for setback violators | Demo cost only |
| **Code enforcement-driven legalization** | City has already cited the property; you're working on a clock | Higher fees, less negotiating room |

**Process for retroactive permit:**
1. As-built survey of existing conditions (architect or engineer)
2. As-built plans showing what's there
3. Submit plan check with letter explaining the legalization scope
4. Pay investigation fee (2x–10x normal permit fee in most jurisdictions)
5. Inspector inspects existing conditions (may require destructive testing — opening walls)
6. Engineering analysis if structural questions
7. Correct deficiencies (often: add hold-downs, add fire blocking, fix electrical, install smoke/CO, address Title 24 envelope)
8. Pass inspections and obtain final

**Special ADU legalization path**: California has an **ADU amnesty program** in many jurisdictions (per AB 587, AB 671) — unpermitted ADUs built before a date certain can be legalized under more lenient standards if they meet health/safety. Check your jurisdiction's ordinance.

### 2.5 Building Permit vs. Other Permits — Don't Conflate

| Permit | Issued by | Triggered by |
|--------|-----------|--------------|
| **Building permit** | Building & Safety | Any construction subject to CRC/CBC |
| **Encroachment permit** | Public Works | Any work in the public right-of-way (driveway approach, curb cut, sidewalk, water/sewer laterals to street, scaffolding over sidewalk) |
| **Grading permit** | Building or Public Works | Earthwork over local threshold |
| **Demolition permit** | Building & Safety + (sometimes) Air District for NESHAP asbestos | Removing a structure |
| **Tree removal permit** | Parks/Planning | Removing a protected/heritage tree |
| **Fire department permit** | Fire | Sprinklers, alarm systems, hazardous materials, sometimes hood/duct |
| **Health permit** | County Environmental Health | Septic systems, wells, food prep |
| **Coastal Development Permit (CDP)** | Coastal Commission or local with CDP authority | Any development in the coastal zone |
| **Planning/zoning approval** | Planning | Variances, CUPs, design review (separate from building permit) |

**These permits do not substitute for each other.** Pulling a building permit does not give you encroachment rights. Many jobs need 3–5 permits from different departments.

### 2.6 Separate vs. Combined Sub-trade Permits

In California residential, the building permit typically includes electrical, mechanical, and plumbing as sub-permits on the same application — but this varies. In some jurisdictions:
- The GC pulls the building permit; the licensed electrician, plumber, HVAC contractor pull their own sub-permits in their own names
- The GC pulls everything under one combination permit and is responsible for all inspections

**Important**: If a sub pulls their own permit, the inspector calls **that sub**, not you. Coordinate scheduling.

## 3. The Permit Application Process

### 3.1 Who Can Pull a Permit

| Pullable by | Conditions |
|-------------|-----------|
| **Licensed contractor** | Active CSLB license matching the work (B for general; C-class for specialty); active workers' comp (or valid exemption) |
| **Owner-builder** | Per B&P §7044; see below |
| **Authorized agent** | With written authorization from owner or contractor |

### 3.2 California Owner-Builder Rules (B&P §7044)

A property owner can pull a permit and act as their own GC without a contractor's license, **but only in narrow circumstances**:

| Scenario | Requirements | Sale restriction |
|----------|-------------|------------------|
| **Owner performs all work personally** | Owner does the work or uses W-2 employees paid wages | Not intended for sale |
| **Owner contracts with licensed subs only** | Every sub is properly licensed for their trade | No more than 4 SFR structures offered for sale per calendar year |
| **Owner's principal residence** | Owner has lived in residence 12 months prior, work done before sale | No more than 2 structures, no more than once in any 3-year period |

**Rebuttable presumption** of sale: selling within 1 year after completion. **Conclusive presumption**: selling 5+ structures within 1 year.

**Practical GC implications:**
- You may be hired by an owner-builder as a sub or as an unlicensed "consultant" — but if you act like a GC (hire subs, control means and methods, contract for the whole project), CSLB can argue you're operating without a B license on a private project
- Many owner-builders end up over their heads and want to convert to a contractor-led project mid-stream — be careful about taking over a half-built project; document existing conditions before signing
- Owner-builders are **personally liable** for workers' comp, payroll taxes, mechanics liens, etc. Some get sued by their own "helpers" claiming employee status

### 3.3 Application Components

For a typical California single-family addition or new construction:

| Component | What it is | Notes |
|-----------|-----------|-------|
| **Permit application form** | Owner info, contractor info, scope, valuation | Sign as licensee in space provided |
| **Plans** (2–4 sets, often digital only now) | Site plan, floor plans, elevations, sections, structural details, schedules | Most jurisdictions are e-plan; PDFs at 11×17 or 24×36 |
| **Site plan / plot plan** | Property boundary, existing/proposed structures, setbacks dimensioned, easements, drainage | Often requires a separate "topo + utility" base for larger projects |
| **Structural calculations** | Lateral analysis, gravity load path, hold-down schedule, footing design | Required when project exceeds CRC prescriptive limits |
| **Truss calculations / shop drawings** | From truss manufacturer | Provided post-permit usually, before framing inspection |
| **Soils report (geotech)** | Required for hillside, expansive soil, sites with imported fill, or larger projects | Often required if any cut/fill over local threshold |
| **Title 24 Energy Compliance (CF1R)** | Generated by CEC-approved software (EnergyPro, CBECC-Res) | Signed by qualified person + owner |
| **CalGreen checklist** | Worksheet showing mandatory measures | Builder signs |
| **Storm Water Pollution Prevention Plan (SWPPP)** | If project disturbs > 1 acre, or local trigger smaller | State WDID required + Risk Level 1/2/3 |
| **Accessibility** | For multifamily (R-2) only — single-family is exempt; ADUs generally exempt | CBC Ch. 11A or 11B |
| **Fire sprinkler plans** | NFPA 13D for SFR, NFPA 13R for low-rise multifamily | Required for new construction since 2011 statewide |
| **Title report or assessor's parcel map** | To verify ownership and APN | Sometimes required |

**Pro tip**: Before you submit, walk through the jurisdiction's "submittal checklist" line by line. Most plan check rejections at intake are missing-document rejections, not technical rejections.

### 3.4 Plan Check Process

| Type | When | Turnaround |
|------|------|------------|
| **Over-the-counter (OTC)** | Small projects: water heater swap, panel upgrade, simple re-roof, sometimes simple bathrooms | Same day, 30 min – 2 hours |
| **Standard plan check** | Most additions, new construction, ADUs | 4–12 weeks first review (longer in busy cities) |
| **Expedited plan check** | Customer pays premium for faster turnaround | 1–3 weeks; cost: 1.5x–2x normal plan check fee |
| **Third-party plan check** | Some jurisdictions allow approved private firms to do the review | Same as expedited; speeds it up if jurisdiction is slow |
| **HCD plan check** | For factory-built housing and mobile home park work | State-level review |

**The corrections cycle:**
1. Submit → Intake review (admin: completeness only)
2. First review (8–12 weeks) → "Plan check corrections" letter issued
3. Designer answers each item, redlines plans, resubmits
4. Second review (typically faster: 2–4 weeks)
5. Repeat as needed (usually 2–3 cycles total)
6. Plans stamped "Approved" → permit ready to issue
7. Pay permit fees → permit issued

**Expedited plan check** is worth it when:
- Carrying costs (construction loan interest) exceed the expedite fee per week
- You have a hard deadline (lease end, occupancy)
- Trade scheduling will collapse if delayed
**Not worth it** when the project has structural complexity that requires significant back-and-forth — you'll burn the expedite fee and still get corrections.

### 3.5 Permit Fees — How They're Calculated

Most California jurisdictions use **valuation-based** fees, tied to the **ICC Building Valuation Data** (BVD) table (e.g., $200/sf for SFR Type V-B construction in 2025). The jurisdiction multiplies your reported valuation (or the BVD minimum, whichever is higher) by a fee schedule percentage.

**Permit fee = building permit fee + plan check fee + electrical + mechanical + plumbing + state surcharges + other**

Other fees layered on:
- **Plan check**: typically 65%–85% of building permit fee
- **State surcharges**: SB-1473 ($1/$25,000 valuation, capped); SMIP (Strong Motion Instrumentation Program); CalGreen disposition fee
- **School impact fees**: $4.79/sf (Level 1, 2024) for new residential — paid before permit issues
- **Park / impact / capacity / connection fees**: Highly variable; can total $20,000–$80,000+ for a new SFR in some jurisdictions
- **Encroachment, grading, fire, planning** as applicable

**Typical residential ranges (very approximate):**

| Project type | Building + plan check fees only |
|--------------|----------------------------------|
| Water heater swap | $50–$200 |
| Re-roof (typical 25 sq) | $200–$600 |
| Panel upgrade (200A) | $200–$500 |
| Kitchen remodel (no structural) | $500–$2,500 |
| Bath remodel | $300–$1,200 |
| 600 sf addition | $3,000–$10,000 |
| New 2,500 sf SFR | $8,000–$30,000 (before impact fees) |
| New 800 sf detached ADU | $3,000–$15,000 (impact fees often waived under SB 13) |

Always pull the jurisdiction's current fee schedule and run an estimate at preconstruction, not at bid time.

### 3.6 Permit Validity, Expiration, Renewal

**State default (CRC R105.5):** Permits expire when:
- No work commenced within **180 days** of issuance, **OR**
- Work suspended/abandoned for **180 days** at any point

**Many jurisdictions extend** to one year with required inspections within each 180-day window. Always check local language.

**Renewal/extension**:
- Most jurisdictions allow one-time extension (1 year) for cause, with a fee
- If expired, you may need to re-permit under the current code cycle (problem if code has updated and your project no longer complies)
- Some jurisdictions allow "permit reactivation" within a grace period with a re-inspection

**Critical habit**: Call an inspection at least every 5 months to keep the permit alive, even if it's just a footing pre-pour. Don't let a permit lapse during a delay.

## 4. Inspection Sequence — California Residential

Inspections are the GC's job to schedule (CRC R109). You call when ready, the inspector comes (typically next business day; same-day if requested early), the inspector either approves, partial-approves, or fails. **Don't cover or close anything until the relevant inspection has passed.**

Standard sequence for new SFR / addition with foundation, framing, MEP, and finish:

| # | Inspection | When called | What inspector checks |
|---|-----------|-------------|------------------------|
| 1 | **Setback / staking** (sometimes) | After forms but before excavation | Setbacks per approved plans |
| 2 | **Footing / foundation** | After forms, rebar, hold-down/anchor bolts placed; before pour | Depth, width, rebar size/spacing/cover/lap, hold-down placement and embedment, anchor bolts (size, spacing, edge distance, washers), setbacks, soil bearing condition |
| 3 | **Slab / under-floor** | In-slab plumbing in place, vapor barrier, capillary break, rebar/mesh; before pour | Plumbing rough (under slab), vapor retarder integrity, rebar/wire, post-tension if applicable |
| 4 | **Under-floor** (raised foundation) | Floor framing in place, rough plumbing/mechanical/electrical under floor done | Joist size/spacing, blocking, sub-floor venting, plumbing and electrical under floor |
| 5 | **Shear wall / nailing** (often separate in seismic zones) | Sheathing on, edge and field nailing complete, before any cover | Sheathing thickness, nail size, edge nail spacing (per shear schedule), staples allowed? hold-downs tied to plates |
| 6 | **Rough framing** | All framing complete, including roof; windows installed; before any insulation or drywall | Connections (Simpson hardware), header sizes, post sizes, blocking, fire blocking at chases and stair stringers, draft stopping (attic > 3,000 sf), ICC-ES report compliance for engineered lumber, hold-down installation, ridge beam connections, deck ledger flashing and lag/bolt schedule |
| 7 | **Rough plumbing** | DWV piped, water lines run, gas lines piped; pressure test active | DWV 10-ft head test (or 5-psi air test); water supply 50-psi minimum test; gas 10-psi or 1.5x working pressure for 15 min; trap arm lengths; cleanout access; vent terminations; expansion provisions; nail plates at studs |
| 8 | **Rough mechanical** | Ductwork installed, equipment located, B-vent/PVC vent run, combustion air openings sized | Duct sealing (CalGreen mandatory), duct support, return air pathways, combustion air, condensate routing, dryer duct length, refrigerant line set protection, equipment clearances |
| 9 | **Rough electrical** | Boxes installed, wires pulled, panel rough complete; before insulation/drywall | Box volume fill, cable staples within 8" of box, AFCI/GFCI placement, panel working clearance, grounding electrode system, bonding, hole drilling within stud zone (1.25" from edge, nail plates if not), conductor type (NM-B legal in CA per CEC amendments), Title 24 lighting placement |
| 10 | **HERS pre-drywall** (some climate zones / projects) | Insulation installed, ducts sealed | QII insulation verification (visual + R-value); duct leakage if test-out cannot be done at end |
| 11 | **Insulation** | All insulation installed (often combined with HERS) | R-values per CF1R, baffle at eaves, no compression, vapor retarder per climate zone |
| 12 | **Drywall nailing** (seismic zones / high-wind / shear walls cont. through to drywall) | Drywall hung, screwed/nailed, before tape | Screw/nail pattern per shear schedule (often 7" o.c. edges, 10" field for 1/2" drywall shear); only if drywall is part of the lateral system |
| 13 | **Lath inspection** (stucco) | Lath, weep screed, two-layer WRB, flashings installed; before scratch coat | Wire lath fastening (every 6" o.c. at studs), weep screed minimum 4" above grade / 2" above paved surface, head flashings over openings, two layers of grade D building paper or approved WRB, kick-out flashings at roof/wall intersections |
| 14 | **HERS final tests** | Equipment in, ducts complete, building closed | Duct leakage (target ≤ 6% of nominal CFM for new); envelope leakage (target ≤ 5 ACH50 for prescriptive new in most climate zones); refrigerant charge or fault detection diagnostics; HVAC airflow; fan watt draw |
| 15 | **Building final** | All work complete, site clean, address visible, drainage final | All previous corrections cleared, smoke/CO detector locations, GFCI/AFCI verified live, water heater strapping and platform/pan, P&T discharge, T-stat installed, equipment startup, attic access, crawlspace access, exterior drainage |
| 16 | **Plumbing final** | (Often combined with building) | All fixtures connected, traps wet, hose bibs with vacuum breakers, irrigation backflow, gas leak test final |
| 17 | **Electrical final** | (Often combined with building) | All circuits live, breakers labeled, all GFCIs/AFCIs functional, panel cover, exterior receptacle WR/TR, EV-ready label and breaker space |
| 18 | **Mechanical final** | (Often combined with building) | Equipment running, condensate routing, refrigerant lines protected, exhaust fans terminating outside, dryer vent termination |
| 19 | **PV final** (if applicable) | After utility approves PTO | Inverter, AC disconnect, labeling, MPU if needed, rapid shutdown labels |
| 20 | **Fire sprinkler final** | NFPA 13D acceptance test | 200-psi hydrostatic test, flow test at most remote head |
| 21 | **Planning final** | Some jurisdictions for conditions of approval | Landscape, fencing, lighting per design review |
| 22 | **Final / Certificate of Occupancy** | Everything passes + all conditions cleared + all fees paid | CO issued (new construction) or Final approval (alterations) |

### 4.1 Foundation Inspection Deep Dive

What the inspector usually has in hand:
- Approved plans showing footing schedule
- Hold-down schedule with location plan
- Soils report (if applicable)

What they're literally looking at:
- **Depth**: At least 12" below undisturbed natural grade and below frost line (CRC R403.1.4 — California's frost depth is typically 12" except in high-elevation areas)
- **Width**: Per Table R403.1 or engineered design
- **Reinforcement**: Per plan. CRC requires minimum 2 #4 bars in stem walls when on expansive soil; engineered designs may require more.
- **Hold-downs**: Embedded the correct depth, correct distance from end of shear panel, tied to rebar, not floating
- **Anchor bolts**: 5/8" min in CRC R403.1.6 (or 1/2" in some configurations), spaced per plan (usually max 6 ft o.c., max 12" from ends, min 7" embedment, with 3" × 3" plate washers in seismic zones)
- **Setbacks**: Measured to face of forms — if you're tight to the line, expect the inspector to actually measure

**Common fails**: Hold-down anchors floating (not tied), anchor bolt spacing exceeded, no plate washers, rebar cover too small, hold-downs at wrong end of shear panel, forms not properly braced against blow-out.

### 4.2 Framing Inspection Deep Dive

The framing inspection is the **highest-stakes inspection** for an SFR. If the inspector fails framing, you've got an open building that you can't close until you fix it. Hardware additions and re-nailing are tedious and expensive.

What the inspector checks (partial — entire CRC Ch. 6 applies):
- **Wall framing**: Stud size/spacing, top plate splices (lapped 48"), corner construction, headers per CRC Table R602.7
- **Floor framing**: Joist size/spacing, hangers, blocking, sub-floor nailing, ledger flashing
- **Roof framing**: Rafter or truss connections, hurricane ties (CRC Table R602.10.6 for high-wind, anywhere in California you'll see them required by engineer), ridge connections, valley/hip support
- **Shear walls**: Sheathing thickness, panel edges blocked, nail size/spacing per shear schedule, hold-downs installed and torqued
- **Hold-downs**: Strap embedment into concrete, threaded rod thread engagement, anchor bolt-to-stud connection, take-up devices on rod systems
- **Fire blocking**: At all interconnections between concealed vertical and horizontal spaces, in soffits, at intersections, at stair stringers, around vent stacks (CRC R302.11)
- **Draft stopping**: In floor/ceiling assemblies for spaces over 1,000 sq ft, in attics over 3,000 sq ft (CRC R302.12)
- **Notching/boring**: Studs (max 25% of width for bearing, 40% for non-bearing), joists (per CRC R502.8)
- **Connectors**: Simpson hardware (or equivalent) per plan, all nail holes filled, correct nail type (SDS, N10, common, etc.)
- **Engineered lumber**: Manufacturer's documentation present, installed per spec, no field-cutting beyond allowance
- **Stair geometry**: Riser height (max 7.75"), tread depth (min 10"), uniformity within 3/8", headroom (min 6'-8"), handrails (CRC R311)
- **Guards**: At any walking surface > 30" above grade, min 36" tall, sphere ≤ 4" (CRC R312)
- **Egress windows**: Sill height max 44", opening min 5.7 sq ft (5.0 sq ft if grade floor), min width 20", min height 24" (CRC R310)

**Common fails**: Missing hurricane ties, missing fire blocking around vents and at top of walls into attic, hold-down nuts not snugged + 1/2 turn, wrong nail size in connectors, beam bearing < 1.5", inadequate header support, mis-cut top plates without metal strap.

### 4.3 MEP Rough Inspection Cheat Sheet

| Trade | Pre-inspection self-check |
|-------|---------------------------|
| **Plumbing** | DWV at 10-ft head, holding 15 min; water lines at 50 psi, no drop; gas at 10 psi for 15 min; cleanouts accessible; vents at correct slope (1/4" per ft for 2" and smaller); trap arms within length limits; AAVs only where allowed locally |
| **Electrical** | All boxes set; cables stapled within 8" of every box; cables in stud bays drilled center; nail plates over any drilling within 1.25" of stud edge; AFCI/GFCI plan posted on panel; panel grounded; conductors not nicked; box fill calculated; equipment grounding conductors continuous |
| **Mechanical** | Ductwork sealed with mastic (not just tape); supports every 4 ft horizontal / 10 ft vertical; combustion air calculated and ducts in place; condensate line slope; B-vent clearances to combustibles; furnace platform/clearances; refrigerant line insulation continuous; bath fans ducted to exterior |

### 4.4 What Triggers a Failed Inspection vs. a Partial Approval

- **Partial approval**: Inspector signs off on parts that are ready, lists items to complete before next visit. Common when the GC calls inspection knowing some areas aren't ready yet but wants a portion blessed.
- **Failed inspection**: Inspector writes a correction notice; you may owe a re-inspection fee on the next call (typically $75–$200). Two-fail rule in some jurisdictions: third visit can require formal supervisor escalation.
- **Stop work**: Issued when the inspector finds work proceeding without permit, dangerous conditions, or systematic violations.

### 4.5 Certificate of Occupancy (CO) vs. Final Approval

- **CO (Certificate of Occupancy)**: Issued only for new buildings or changes of occupancy. The building cannot be legally occupied until CO is issued. CBC §111.
- **Final approval**: For alterations, additions where occupancy classification doesn't change. The permit is closed when final is signed off.
- **Temporary Certificate of Occupancy (TCO)**: Some jurisdictions issue a TCO when the building is safe to occupy but minor items remain (landscaping, punch). Usually has an expiration (30–90 days).

**What delays CO most often:**
- Outstanding planning conditions (landscaping, fencing, lighting)
- Public Works (driveway not finished, sidewalk damage not repaired, sewer lateral not inspected)
- Fire (sprinkler acceptance not done, address numbers not visible, defensible space in WUI)
- School fees unpaid
- HERS reports not registered with the HERS provider data registry
- Solar PTO not received from utility

## 5. Title 24 Part 6 — California Energy Code

Title 24 Part 6 is the single largest source of permit complexity for residential GCs. Don't try to memorize the prescriptive tables — instead, understand the framework and know when to call the energy consultant.

### 5.1 Prescriptive vs. Performance Compliance Paths

| Path | When used | Pros | Cons |
|------|-----------|------|------|
| **Prescriptive** | Simple projects following the table values directly | No software/consultant cost | Locked into specific R-values, U-factors, etc. — no trade-offs |
| **Performance** | Anything non-trivial; allows trade-offs | Can swap a bigger window for better insulation; allows more design flexibility | Requires CEC-approved software (CBECC-Res, EnergyPro); usually $300–$1,500 per project for a consultant |

For new construction or substantial remodels: **performance path**. For a simple window replacement or HVAC changeout: **prescriptive**.

### 5.2 The Three CF Forms

| Form | What it is | Who signs | When generated |
|------|-----------|----------|----------------|
| **CF1R — Compliance Documentation** | The plan-set energy document. Shows all measures the project will use. Generated by CEC-approved software. | Documentation author + responsible party (owner or owner's agent) | Submitted with plans at plan check |
| **CF2R — Installation Certificate** | After installation, the installer/contractor certifies that they installed what the CF1R required. | Installing contractor or builder | At appropriate inspection stage (e.g., HVAC CF2R at mechanical rough, insulation CF2R at insulation inspection) |
| **CF3R — HERS Verification** | The third-party HERS rater certifies that they verified the field measure. | CEC-certified HERS rater | After HERS testing, registered with HERS provider |

All forms must be registered with a HERS provider (CalCERTS or CHEERS) and downloaded by the building department before final.

### 5.3 HERS Rater — Who They Are and What They Test

A **HERS (Home Energy Rating System) rater** is a CEC-certified independent third party — they are not employed by the GC, the HVAC contractor, or the owner. They are paid by the builder/GC but answer to the HERS provider and the state.

HERS-verified measures (common):

| Measure | What HERS rater does | Failure consequence |
|---------|---------------------|---------------------|
| **Duct leakage** | Pressurizes duct system with duct blaster; measures CFM @ 25 Pa | New systems: ≤ 6% of nominal flow; if fail, seal more and retest |
| **Envelope leakage (blower door)** | Pressurizes house to 50 Pa; measures CFM/ACH | Target varies (prescriptive often 5 ACH50); if fail, more air sealing |
| **Refrigerant charge** | Verifies subcooling or superheat in spec window | Out of spec: HVAC sub recharges or service |
| **Airflow** | Measures CFM at registers vs. tonnage | Low airflow: clean filter, oversized duct, replace fan |
| **Fan watt draw** | Measures electrical draw at rated airflow | Excessive: ECM motor required |
| **QII (Quality Insulation Installation)** | Visual + spot-checks insulation install | Fix gaps, voids, compression before drywall |
| **Whole-house ventilation (ASHRAE 62.2)** | Verifies continuous mechanical ventilation rate | Adjust fan setting or upsize |
| **High-performance attic / wall** | Verifies assembly composition | Fix to spec before close-in |
| **Compact hot water distribution** | Verifies pipe layout near tank | Re-route pipes |
| **Demand recirculation** | Verifies pump installation/wiring | Install per spec |

Schedule the HERS rater early. The good ones book 2–4 weeks out. **Coordinate the HERS visit with your inspection schedule** — typically: insulation HERS just before drywall, equipment HERS just before final.

### 5.4 Climate Zone Quick Reference

California has 16 climate zones. Each has different prescriptive requirements.

| Zone | Region (example cities) | Distinguishing requirements |
|------|------------------------|---------------------------|
| **CZ 1** | North coast (Arcata, Crescent City) | No SHGC; heating-dominated |
| **CZ 3** | SF Bay coast (SF, Oakland, Berkeley) | No SHGC; mild |
| **CZ 4** | South Bay / inland (San Jose) | SHGC required; mixed |
| **CZ 5** | Central coast (Santa Maria) | No SHGC |
| **CZ 6** | LA coast (Long Beach) | SHGC |
| **CZ 8** | LA inland (Anaheim, Pasadena) | SHGC; cooling-dominated |
| **CZ 9** | LA-Hollywood / Glendale | SHGC; cooling |
| **CZ 10** | Riverside, San Bernardino | Hot dry; heavy insulation needs |
| **CZ 12** | Sacramento valley | Hot summer, cold winter |
| **CZ 13** | Fresno / Central Valley | Hottest dry |
| **CZ 14** | Palm Springs / high desert | Extreme cooling needs |
| **CZ 15** | Imperial Valley | Hottest; tight envelope critical |
| **CZ 16** | Mountains (Truckee, Mammoth) | No SHGC; extreme cold; frost depth deeper |

### 5.5 Key Prescriptive Targets (2022/2025 Code, Residential)

These are general benchmarks — confirm exact values for your climate zone via the current Residential Compliance Manual.

| Element | Typical prescriptive target |
|---------|------------------------------|
| Ceiling/attic | R-38 to R-60 (climate-dependent) |
| Wall (above grade) | R-21 to R-23 cavity + R-5 continuous (climate-dependent) |
| Slab edge | R-7 to R-10 (CZ 14, 16) |
| Window U-factor | ≤ 0.30 (most zones); ≤ 0.36 in CZ 1, 16 |
| Window SHGC | ≤ 0.23 (CZ 2, 4, 6–15); no req. in CZ 1, 3, 5, 16 |
| Duct leakage | ≤ 6% of nominal CFM (new systems) |
| Envelope leakage | ≤ 5 ACH50 (prescriptive, most zones) |
| HVAC efficiency | Heat pumps preferred under 2022 code; minimum 14 SEER2 / 6.7 HSPF2 for systems |
| Water heating | Heat pump water heater (HPWH) is now the prescriptive baseline in many zones |
| Lighting | 100% high-efficacy (LED) |
| Cool roof | Required on low-slope roofs; varies on steep-slope |

### 5.6 Photovoltaic (PV) Mandate

Since the **2019 code cycle (effective 2020)**, **new single-family homes must include a PV system.** This carried through 2022 and 2025 cycles.

**System size formula** (simplified): kWdc = (CFA × A) / 1000 + (Nunit × B), with A and B values from Section 150.1(c)14 of the Energy Code, climate-zone dependent.

**Exceptions / reductions:**
- Solar access roof area (SARA) < 80 sq ft contiguous → exempt
- For steep-sloped roofs: SARA excludes roof areas between 300° and 90° of true north
- Snow load conflict per ASCE 7-16 Ch. 7
- If installed with a battery storage system (≥7.5 kWh usable, JA12-compliant), PV size may be **reduced by 25%**
- Buildings approved by planning before 1/1/2020 with mandatory conditions of approval
- Minimum PV system size after applying SARA: ≥ 1.8 kWdc (else exempt)

**For ADUs**: Detached new ADUs are subject to the PV mandate; attached ADUs to existing SFR are generally exempt unless the existing SFR is also being newly constructed or scope triggers PV.

**Battery storage**: Not mandatory in residential under 2022, but credited heavily. Watch for the **2025 cycle** which substantially expands incentives and may push toward mandatory in some configurations.

### 5.7 EV-Ready / EV-Capable Requirements

For new one- and two-family dwellings with attached garages:
- **EV-capable**: At minimum, one parking space must have a raceway (conduit), a 40-amp breaker space reserved in the panel (labeled "EV CAPABLE"), and load calculations showing the panel can serve a 40A EV charge.
- **EV-ready**: Some jurisdictions (and the 2022 CalGreen) require fully wired Level 2 (208/240V, 40A) outlet.
- Service capacity must be sized to charge all required EV spaces simultaneously at 40 A min.
- Construction documents must show conduit run from panel to parking space.
- Panel directory must label the EV breaker space.

**Practical note**: Don't take the cheapest panel — for any new SFR, install a 200A minimum service with EV breaker space identified. Future retrofit costs are real.

### 5.8 Other Notable Title 24 Requirements

- **High-efficacy lighting**: 100% LED throughout. No incandescent or halogen. Dimmers must be LED-rated.
- **Domestic hot water**: Pipe insulation for runs over a threshold; demand recirculation required in some cases; HPWH increasingly default.
- **Whole-house mechanical ventilation**: Per ASHRAE 62.2, balanced or exhaust-only, continuously operating or controlled. Bath fans often serve.
- **Cool roof** (low-slope, ≤ 2:12): Solar reflectance index requirements vary by climate zone.
- **Indoor air quality**: MERV 13 filtration in supply or return for new homes.

## 6. CalGreen — Title 24 Part 11

CalGreen is California's mandatory green building code. **Mandatory measures apply to all new construction and certain alterations statewide.** Tier 1 and Tier 2 are voluntary unless a local jurisdiction has adopted them as mandatory (some have).

### 6.1 Mandatory vs. Voluntary

| Category | Mandatory? | Notes |
|----------|-----------|-------|
| Chapter 4 — Residential Mandatory | Yes, statewide | Default level for all new residential |
| Appendix A4 — Residential Voluntary (Tier 1 / Tier 2) | No, unless locally adopted | Some cities require Tier 1 (e.g., Santa Monica, Berkeley) |
| Chapter 5 — Nonresidential Mandatory | Yes | For commercial |
| Appendix A5 — Nonresidential Voluntary | No | |

### 6.2 Key Residential Mandatory Measures (2022, with 2025 updates)

| Measure | Requirement | Documentation |
|---------|-------------|--------------|
| **Water-conserving fixtures** | Showerheads ≤ 1.8 GPM @ 80 psi; lavatory faucets ≤ 1.2 GPM; kitchen faucets ≤ 1.8 GPM; toilets ≤ 1.28 GPF; urinals ≤ 0.125 GPF | Product cut sheets / WaterSense labels |
| **Construction waste management** | ≥ 65% diversion of C&D waste (was 50% in older code; now 65% per 2022 mandatory) | Waste management plan, hauler tickets, end-of-job summary |
| **VOC content limits** | Adhesives, sealants, paints, primers, coatings, carpet, carpet cushion, resilient flooring, composite wood (per CARB ATCM) | Product cut sheets, MSDS |
| **Moisture control** | Capillary break / Class II vapor retarder under slab on grade; flashing details at openings; foundation drainage | Shown on plans; inspector verifies |
| **Termite protection** | Per CRC + treated lumber within 12" of ground; physical barriers in some areas | Plans + product |
| **Operations and maintenance manual** | Provided to occupant at occupancy | Builder prepares |
| **Pollutant control during construction** | Duct openings protected during construction; filter changeout at substantial completion; flush-out option | Logs + photos |
| **Indoor moisture management** | Building envelope assemblies designed to prevent moisture intrusion | Plans show flashing/WRB details |
| **Commissioning (residential basic)** | Minimal — typically just HVAC start-up verification and operating instructions | Builder/installer documents |
| **EV charging** (overlaps with Energy Code) | EV-capable infrastructure | Plans + inspector |

### 6.3 CalGreen Inspection

The CalGreen inspector (often the same building inspector wearing a different hat) verifies:
- Posted documentation on site (waste tracking, product cut sheets)
- Water-conserving fixtures installed (flow tested)
- Low-VOC products used (verified against your submitted product list)
- Construction waste hauler tickets present
- O&M manual prepared

**Best practice**: Build a **CalGreen binder** at project start. Include:
- Approved CalGreen checklist (signed)
- Cut sheets for every CalGreen-regulated product (paint, adhesive, sealant, carpet, composite wood, fixtures)
- C&D waste plan and running hauler tickets
- Photos of installation conditions

Hand the binder to the inspector at CalGreen verification. It will save you a re-inspection.

## 7. Zoning and Land Use

Zoning is regulated by **planning**, not the building department. **Building permits cannot be issued for projects that violate zoning** — planning sign-off (or "zoning clearance") is a prerequisite to building permit.

### 7.1 Zoning vs. Building Code

| | Zoning | Building Code |
|--|--------|---------------|
| Department | Planning / Community Development | Building & Safety |
| Regulates | What you can build (use, density, height, location on lot) | How you build (construction, safety, energy) |
| Document | Zoning ordinance / municipal code | CBC, CRC, CPC, CMC, CEC, Title 24 |
| Discretion | Often discretionary (variances, CUPs, design review) | Mostly ministerial (objective standards) |
| Public process | Often public hearings, neighbor notification | Usually none |

### 7.2 Common Zoning Designations

| Designation | Typical meaning |
|-------------|----------------|
| **R-1** | Single-family residential |
| **R-2** | Duplex / two-family |
| **R-3 / R-4** | Multifamily (low to high density) |
| **RS** | Residential Suburban |
| **RM** | Residential Multifamily |
| **MU** | Mixed-use (residential + commercial) |
| **C-1, C-2** | Commercial |
| **M / I** | Manufacturing / Industrial |
| **PD / SP** | Planned Development / Specific Plan (custom rules per project) |
| **AG / OS** | Agricultural / Open Space |

Every city has its own labels — RS-7, R-1-7000, R1-2L — but the principle is the same: density tier, often expressed as units per acre or minimum lot size.

### 7.3 Reading a Zoning Ordinance — What to Pull

Before designing, look up these in the zoning code:

| Term | What it means | Where to find |
|------|--------------|---------------|
| **Setback (yard)** | Min distance from property line — front, side, rear, side-street (corner) | Zone development standards table |
| **Height limit** | Max building height, usually measured from natural or finished grade | Same table |
| **Lot coverage** | Max % of lot that can be covered by buildings | Same table |
| **Floor area ratio (FAR)** | Total building floor area ÷ lot area; e.g., FAR 0.5 = 5,000 sf max on 10,000 sf lot | Same table |
| **Parking required** | Spaces per unit / per bedroom | Parking ordinance section |
| **Minimum lot size** | For new lots or lot splits | Same table |
| **Density** | Max units per acre, or min lot area per unit | Same table |
| **Building separation** | Required distance between buildings on same lot | Sometimes in code |
| **Encroachments** | Allowable projections into setbacks (eaves, chimneys, bay windows) | Specific code section |
| **Use** | Permitted, conditional, prohibited | Use table for that zone |

### 7.4 ADU State Law — Key Provisions (Practical Reference)

State law (Gov. Code §66310 et seq. as amended through SB 1211, AB 1033, etc.) requires ministerial approval of ADUs and JADUs across nearly all single-family and multifamily zones, regardless of local rules. Key provisions:

| Topic | State minimum |
|-------|----------------|
| **ADU on SFR lot** | One ADU + one JADU allowed (at least 800 sf attached or detached) |
| **ADU on multifamily lot** | Up to 25% of existing units (or at least one) as conversions of non-livable space, **plus** up to 8 detached ADUs (SB 1211, 2024) |
| **Setbacks (detached ADU ≤ 800 sf)** | 4 ft side and rear, no front setback override |
| **Height (detached)** | 16 ft min allowed; 18 ft if within 1/2 mi of major transit or attached to multistory multifamily; 25 ft for attached ADU (up to height of primary dwelling) |
| **Size (detached)** | At least 800 sf must be allowed (regardless of FAR or coverage); often up to 1,200 sf |
| **Size (attached)** | At least 50% of primary dwelling floor area, up to 1,200 sf |
| **Owner occupancy** | Cannot be required by local jurisdiction (through 2024 sunset, then partial reinstatement — check current law) |
| **Parking** | None required within 1/2 mi of transit, in historic district, on same block as car-share, in permit-parking area, when ADU is converted from existing space |
| **Approval time** | Ministerial approval within 60 days of completed application |
| **Impact fees** | Waived for ADUs ≤ 750 sf (SB 13); proportional for larger |
| **Utility connections** | Cannot require separate water/sewer meter for ADU on SFR lot |
| **Fire sprinklers** | Cannot be required for ADU unless primary residence has them |
| **JADU** | Up to 500 sf, must be within existing or proposed SFR walls, can share bath with primary, requires deed restriction |

**The "5-foot rule"** in many local ADU ordinances is dead — state law caps setbacks at 4 ft.

### 7.5 SB 9 — Two Units + Lot Split

SB 9 (effective 2022, Gov. Code §§65852.21 and 66411.7) provides:
- Ministerial approval of up to **2 units** on a parcel zoned for single-family
- Ministerial approval of **lot split** of an SFR lot into two ~equal lots
- Combined: up to 4 units on a former single SFR lot (2 per lot)
- 4-ft side/rear setbacks
- Owner-occupancy required for lot splits (3 years)
- No demolition of housing affordable to LMI households

**Practical caution**: SB 9 is complex; many jurisdictions are still resisting via creative objective standards. Verify with a land-use attorney before bidding a SB 9 project.

### 7.6 Variances and CUPs

- **Variance**: Relief from a development standard (setback, height, lot coverage) based on a hardship unique to the property. Discretionary. Usually requires findings the planning commission must make.
- **Conditional Use Permit (CUP)**: Approval for a use that's allowed only with discretionary approval (e.g., a daycare in a residential zone, large family residence as commercial).
- **Design review**: Aesthetic / context-based review, varies wildly by city.
- **Coastal Development Permit (CDP)**: Coastal zone work.

**Timeline implications**: Variances often add 3–9 months. **Bid with this in mind** — never assume a variance will come through. If owner needs a variance, your contract should be contingent.

### 7.7 Non-Conforming Structures

A **legal non-conforming structure** is one that complied with code when built but no longer complies with current zoning (e.g., built in 1925 at the property line; current setback is 5 ft).

| Work type | Effect on non-conforming status |
|-----------|-------------------------------|
| **Routine maintenance / in-kind repair** | Allowed; doesn't lose status |
| **Interior alterations** | Usually allowed |
| **Additions** | Often allowed, but the addition must meet current zoning (the existing non-conforming part stays) |
| **Vertical addition / second story** | Often triggers a discretionary review |
| **Substantial damage / demolition** | If destroyed > a threshold (often 50% of value), must rebuild to current zoning — non-conforming status lost |
| **Change of use** | Lost |
| **Cessation of use for X months** | Lost (often 6–12 months) |

**Many cities have ADU/SB 9 exceptions** that allow the use of non-conforming setbacks for new ADUs/SB 9 units.

**Pre-construction action**: Pull the **legal description, recorded plat, and assessor data** before any addition project. Confirm that the existing structure's footprint matches what's recorded. Unpermitted prior expansions can sabotage a project.

## 8. Special Inspections and Third-Party Testing

Special inspections are **independent inspections of specific construction elements by a qualified inspector hired by the owner** (not by the GC) and reporting to the building official (CBC Chapter 17 / CRC R109.2).

### 8.1 When Required

Triggers for residential are usually structural and seismic:

| Element | Trigger |
|---------|---------|
| **Structural concrete** | Required when f'c > 2,500 psi, or for designated seismic systems, or for ALL concrete in some jurisdictions; cylinder testing per ACI 318 |
| **Reinforcing steel** | When placement is critical (shear walls, moment frames, retaining walls > 6 ft tall) |
| **Anchors in concrete** | Post-installed anchors (epoxy, mechanical) in seismic systems |
| **Structural masonry** | All structural CMU/brick |
| **Structural steel** | Welding, high-strength bolt tightening (esp. for hold-downs / moment frames) |
| **Soils** | Compaction testing of structural fill, footing subgrade verification, slope stability |
| **Wood structural elements** (newer requirement) | Designated seismic systems with high R values; some jurisdictions require shear wall nailing inspection by special inspector |
| **Designated seismic systems / seismic resisting elements** | Per CBC §1705.12 / §1705.13 |
| **Sprayed fire-resistive materials** | Less common in SFR; common in apartments |
| **Pile foundations** | Driving records, load tests |
| **Spray-applied insulation** | Some jurisdictions for closed-cell foam in fire-rated assemblies |
| **High-strength bolts** | Pretensioned bolts |

### 8.2 Statement of Special Inspections (SSI)

Per CBC §1704.3, the Engineer of Record (EOR) — usually the structural engineer — prepares a **Statement of Special Inspections** as part of the permit submittal. The SSI lists:
- Each element requiring special inspection
- Whether inspection is **continuous** or **periodic**
- The qualifications required of the inspector
- The reporting protocol

The owner hires the special inspection agency. The GC coordinates scheduling.

**Continuous vs. periodic:**
- **Continuous**: Inspector present at all times during the work (e.g., during concrete placement, structural welding)
- **Periodic**: Spot inspection at key points (e.g., rebar placement before pour, bolt installation)

### 8.3 Who Performs

- **Approved agency** (most common): An ICC- or CBO-certified firm with appropriate certifications (ACI for concrete, CWI for welding, NICET for soils, etc.)
- **Approved special inspector** (individual): Some jurisdictions accept individuals with appropriate certification

**Deputy inspector** is a legacy term still used in California (especially LA region) for a specialty inspector certified by the local jurisdiction's deputy inspector program (e.g., LA City has a deputy inspector list for concrete, welding, etc.). These are basically equivalent to ICC-certified special inspectors.

### 8.4 Inspector of Record (IOR) vs. Special Inspector

- **Inspector of Record (IOR)**: Term used by DSA (Division of the State Architect) and OSHPD (hospital construction). Continuous, full-time inspector on the project. Required for public schools and hospitals. **Not used in private SFR work.**
- **Special inspector**: Element-specific inspector under CBC Ch. 17. Used for private residential when triggered.

### 8.5 Testing Laboratory Requirements

| Test | Standard | Typical frequency |
|------|----------|-------------------|
| **Concrete cylinders** | ASTM C31, C39 | 1 set of 4 per 50 cy or per day per mix, whichever more frequent |
| **Slump** | ASTM C143 | Each truck if specified, else per cylinder set |
| **Soil compaction** | ASTM D6938 (nuclear gauge) or D1556 (sand cone) | Per lift, per defined area |
| **Rebar mill certs** | Per ASTM A615 / A706 | Source documentation |
| **Weld inspection** | AWS D1.1 visual + (sometimes) UT, MT, PT | Per joint or per percentage |
| **Bolt verification** | RCSC | Pretensioned bolts |
| **Air leakage / duct leakage** | Title 24 procedures | One-time at HERS visit |

Lab must be accredited (AASHTO or A2LA or equivalent).

## 9. Enforcement and Violations

### 9.1 Stop-Work Orders

A **stop-work order** halts all construction at the site immediately. Causes:

| Cause | Frequency |
|-------|-----------|
| Working without a permit | Very common |
| Working beyond the scope of the approved permit | Common |
| Failure to call required inspections (e.g., covering work without inspection) | Common |
| Unsafe conditions or imminent danger | Less common |
| Repeated failed inspections | Less common |
| Permit expiration | Less common |
| Code enforcement complaint with violation observed | Common |
| Failure to comply with conditions of approval | Less common |

**How to respond:**
1. **Stop. Do not continue work.** Continuing after a stop-work is an independent violation with severe penalties (daily fines, license discipline).
2. Read the stop-work notice carefully — it states the violation and the contact person.
3. Contact the inspector or supervisor the same day. Be calm and professional.
4. Determine what is required to lift it: usually some combination of (a) pulling the missing permit, (b) opening up covered work for inspection, (c) bringing the violation into compliance, (d) paying penalties (often 2x–10x normal permit fee).
5. After cure, request a re-inspection. The inspector signs off and the stop-work is lifted in writing.

**Don't:** Argue with the inspector at the moment of issuance. The decision has been made. Resolve administratively after.

### 9.2 Code Enforcement (Different from Building Department)

Most cities have a **code enforcement division** that is separate from building inspection. They handle:
- Property maintenance complaints (overgrown weeds, junk vehicles, illegal occupancy)
- Unpermitted construction complaints (often called in by neighbors)
- Substandard housing
- Nuisance and zoning violations
- Short-term rental enforcement

Typical residential code enforcement path:
1. **Complaint** (neighbor, drive-by inspector, anonymous tip)
2. **Notice of violation** (NOV) — usually 10–30 days to cure
3. **Notice and order of abatement** (NSC/NOAB) — formal order, often 30 days
4. **Administrative citation** with fines (typically $100–$1,000 per violation per day)
5. **Recordation of lien** against property
6. **Receivership / abatement by city** (rare; usually only for severe substandard housing)
7. **Criminal misdemeanor charges** (rare; for repeat or egregious violators)

### 9.3 Contractor License Law and Discipline

**CSLB license requirements for residential GC:**
- **Class B (General Building)** — primary classification for any structure requiring 2+ unrelated trades (a B is required for almost all residential construction beyond single-trade work)
- **Class A (General Engineering)** — fixed works (heavy civil); not normally used for residential
- **Specialty C-classes** — for single-trade specialists (C-10 electrical, C-36 plumbing, C-20 HVAC, etc.)

**Important nuance**: A **B license cannot solely contract for a single specialty trade** unless that trade is incidental and supplemental to a larger framing/structural project. A B contractor working solely on a roof is on shaky ground — they need a C-39 sub or to do the roof as part of a larger scope.

**License maintenance:**
- $25,000 contractor's bond
- Workers' comp insurance if employees (or B&P §7125 exemption affidavit if no employees)
- Renewal every 2 years
- $9,000 LLC employee/worker bond (LLCs only)
- LLCs: $1M–$5M general liability insurance (scaled to # of personnel)
- Sole prop/corp: No mandatory liability insurance, but **always carry it** — minimum $1M GL is industry standard

**Disciplinary actions** (B&P §7110, §7113, §7114 et seq.):
- Failure to comply with building laws (e.g., performing unpermitted work)
- Department of contracting without a license (a $5,000-and-up fine plus criminal charges)
- Failure to pay subs/workers
- Abandonment
- Diversion of funds
- Workmanship defects
- License discipline can include: probation, suspension, revocation. Revocation is career-ending.

### 9.4 GC Liability for Unpermitted Work — Specific Scenarios

| Scenario | GC's exposure |
|----------|--------------|
| GC builds without permit, owner caught later | License discipline (B&P §7110); civil liability to owner for full remediation; **uncollectible mechanic's lien** (a contractor with material violations has a weakened lien) |
| GC builds with permit but never calls final | Same as above on next sale or refinance |
| GC builds with permit, fails 3+ inspections, walks away | Civil exposure for breach + abandonment |
| GC remodels a home with existing unpermitted work in another area | If the GC knew and didn't disclose → potential negligent misrepresentation; if the city forces remediation as condition of permit and GC didn't price it, may eat the cost depending on contract |
| Owner-builder hires GC as "consultant" but GC operates as de facto contractor | Unlicensed contracting if scope exceeds consulting (B&P §7028 + §7044 interpretation) |

**Insurance trap**: Many general liability policies exclude work performed without permits. If you cause a fire from unpermitted electrical work, your carrier may deny the claim.

## 10. Dealing with Inspectors and the Building Department

### 10.1 Build the Relationship

Inspectors are people. They have routes, they have favorite contractors (and ones they roll their eyes at), and they have discretion within the gray zones of the code.

**Practices that build trust:**
- Show up on time. If you call an inspection, be there or have a competent representative there with the approved plans.
- Approved plans on site, current revisions clearly marked. Inspector should not have to ask for plans.
- Site clean and accessible. Provide a ladder where one is needed.
- Pre-mark items you know they'll check (hold-down ID numbers, shear panel call-outs). Saves time, demonstrates competence.
- Don't argue trivial things. Save fights for the issues that actually matter.
- Be honest. Inspectors talk to each other. Reputation for dishonesty is irreversible.

**Practices that destroy trust:**
- Covering work after the inspector said don't cover it
- Calling for an inspection when you know the work isn't ready
- Confronting an inspector aggressively
- Going around the inspector to their boss for minor issues
- Lying about a sub's licensure or work performed by others

### 10.2 What Inspectors Can and Cannot Require

**They can require**:
- Code compliance per the adopted code and local amendments
- Compliance with manufacturer's installation instructions (often referenced by code)
- Compliance with the approved plans
- Compliance with the conditions of approval (planning, fire, etc.)
- Re-inspection where work is not ready or fails

**They cannot require** (in theory; often varies in practice):
- Things not in the code or approved plans (personal preference)
- Means and methods unless code-required
- Brand-specific products unless code-required
- Work outside the permitted scope of work
- Inspection of work already approved at an earlier stage (without re-cause)

**Code citation discipline**: When an inspector asks for something, the question to ask politely is: **"What's the code section?"** A reasonable inspector will cite. If they can't, the issue is preference, not code, and you have grounds for professional pushback.

### 10.3 How to Push Back Professionally

Steps in escalation order (only escalate if the prior step failed):

1. **On-site dialogue**: "I want to understand the issue. Can you walk me through what you're seeing and which code section governs?"
2. **Bring documentation**: Manufacturer's installation instructions, ICC-ES report, code commentary, structural engineer's letter (the most powerful — a stamped letter from the EOR carries weight).
3. **Engineer's letter / RFI to building department**: Have the EOR or architect write a stamped letter responding to the inspector's concern with a code-justified position.
4. **Supervisor / chief building official**: If unable to resolve, request a meeting with the chief building official (CBO). Be professional, be evidence-based.
5. **Code interpretation request**: Some jurisdictions have a formal process to request a written code interpretation.
6. **Appeal to Board of Appeals**: Most cities have a Building Code Board of Appeals (CBC §113). Formal process. Rarely used in residential.

**The big rule**: Pick your battles. Eating $400 in framing rework to settle an inspector dispute is usually cheaper than 4 weeks of project delay while you appeal.

### 10.4 Handling Failed Inspections

When an inspector fails an inspection:

1. **Read the correction notice carefully**. Sometimes the inspector writes only one or two items but observed more — they may add others on re-inspection.
2. **Ask for clarification** if any item is unclear. Do this in front of the inspector before they leave.
3. **Photograph the issue** if you'll need to coordinate with subs.
4. **Pay the re-inspection fee** (typically $75–$250).
5. **Fix all noted items + walk the entire scope yourself to catch anything else** — getting failed twice for the same inspection is a credibility hit. Inspector will be sharper next visit.
6. **Re-call for inspection** when ALL items are ready. Don't call when 80% done.
7. **Walk the inspector through the corrections** when they arrive: "Here's what you noted, here's how we addressed each one." Inspector sees due diligence and signs off faster.

### 10.5 Pre-Construction Meetings (Pre-Cons)

For complex projects, request a **pre-construction meeting** with the building department before starting. Useful when:
- Hillside, complex foundation, or unusual structural system
- Phased construction (multi-permit, multi-occupancy)
- Tight site (encroachment, shoring)
- Coastal, WUI, or high-fire-severity zone
- Historic property
- Public visibility (project that will draw complaints)
- Owner-builder doing a substantial project

**Pre-con agenda:**
- Introduce team (GC, designer, EOR, key subs)
- Walk plans together; confirm interpretations
- Identify special inspections and HERS schedule
- Confirm inspection sequence and notification preferences
- Identify any conditions of approval the building department will check at final
- Confirm permit expiration date and renewal plan
- Confirm scope: anything outside permit needs separate permit
- Designate the **single point of contact** at the building department for this project

A 1-hour pre-con can save 2 weeks of misunderstanding mid-project.
