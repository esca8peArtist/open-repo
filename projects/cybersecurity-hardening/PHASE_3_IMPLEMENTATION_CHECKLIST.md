## Section 2: Messaging Finalization (Due July 20)

- [ ] **Finalize 3 sector-specific messaging variants** (policy / labor / academic)
  - Use: templates from PHASE_3_PILOTING_ARCHITECTURE.md Section 3
  - Customize each variant with:
    - 1 unique subject line per sector
    - 2-paragraph lead emphasizing sector-specific value prop
    - 1 custom call-to-action
  - Review against Phase 1–2 messaging for consistency; note tone updates
  - Owner: Anya + Communications specialist
  - Time: 6 hours

- [ ] **Create 3 sector-specific briefing decks** (10-minute overview slides per sector)
  - Deck 1 — Policy sector: FOIA documentation focus, regulatory gap analysis, litigation readiness
  - Deck 2 — Labor sector: member protection angle + worker-accountability angle (separate slides)
  - Deck 3 — Academic sector: clinic litigation prep, curriculum integration, research collaboration
  - Format: Google Slides or PDF; include 8–12 slides per deck
  - Owner: Research assistant + Anya
  - Time: 8 hours

- [ ] **Create Wave 1–2–3 contingency messaging** (for low-response scenarios)
  - If Wave 1 read rate <40%: draft alternative "value-prop refresh" email
  - If election incident occurs: draft incident tie-in email for all pending contacts
  - Format: draft emails, decision trigger attached
  - Owner: Anya
  - Time: 2 hours

- [ ] **Finalize sector-specific follow-up materials** (post-briefing send)
  - Policy sector: FOIA index + threat model summary + regulatory framework proposal
  - Labor sector: member training outline + organizing security quick-reference + opt-out toolkit
  - Academic sector: litigation memo template + curriculum module outline + research collaboration framework
  - Format: PDF or markdown; 5–10 pages per sector variant
  - Owner: Research assistant
  - Time: 8 hours

**Subtotal: 24 hours | Deadline: July 20, 2026**

## Section 3: Infrastructure Setup (Due July 25)

- [ ] **Calendly scheduling links** (create 5 time slots per Wave 1 organization)
  - Set up Calendly for Anya; create one calendar block for Phase 3 briefing calls
  - Availability: Mon–Fri, 1–5 PM ET (standard availability for policy/labor/academic orgs)
  - Duration: 30 minutes per call; 30-minute buffer between calls
  - Include: automatic Zoom link generation, meeting prep documents linked in confirmation
  - Owner: Anya or Coordinator
  - Time: 1.5 hours

- [ ] **Outcome tracking spreadsheet** (create live tracking dashboard)
  - Use: PHASE_3_COHORT_TRACKING.csv as template
  - Add automated columns:
    - Days since outreach (formula: TODAY() - Initial_Outreach_Date)
    - Status flags (RED if >10 days since send with no read; YELLOW if briefing not scheduled by Day 5; GREEN if read confirmed)
    - Engagement score (manual 1–5 rating post-briefing)
  - Set up weekly review cadence (Friday 4 PM ET): Anya reviews all Waves, identifies at-risk orgs
  - Owner: Anya or Coordinator
  - Time: 2 hours

- [ ] **Output tracking database** (create publication/litigation/curriculum watch list)
  - For each Wave 1–2 organization, create saved searches for:
    - Organization name + "policy brief" / "white paper" / "report"
    - Organization name + "litigation" / "brief" / "filing"
    - Organization name + "curriculum" / "course" / "training"
  - Tools: Google Alerts (free; email delivery), Muck Rack (if org has media monitoring license)
  - Frequency: weekly scans starting Sept 1; escalate to bi-weekly after Gate 2
  - Owner: Research assistant
  - Time: 3 hours

- [ ] **Email infrastructure** (set up outreach email account or folder)
  - Outreach account: dedicated Gmail/Outlook with consistent branding (if separate account) OR
  - Email folder: "Phase 3 Outreach" in main account with color-coded labels per wave + per status
  - Labels/colors: Wave1 (red), Wave2 (orange), Wave3 (yellow), Read_Confirmed (green), Meeting_Scheduled (blue), Output_Identified (purple)
  - Set up rules: auto-label replies by sender domain (e.g., @brennancenter.org → "Wave1 Read_Confirmed")
  - Owner: Anya or Coordinator
  - Time: 1 hour

- [ ] **Meeting notes template** (standardized briefing call documentation)
  - Template fields: Organization / Contact / Date / Time / Attendees / Use Case Identified / Specific Output (yes/no, if yes: type + timeline) / Resources Needed / Follow-up Actions / Engagement Score (1–5)
  - Tool: Google Doc template, copy per organization, shared in email confirmation
  - Owner: Anya
  - Time: 1 hour

- [ ] **Wave 3 contact verification** (delay-able; due July 25 if Wave 1–2 on track)
  - Same process as Wave 1–2; verify 7 Wave 3 primary + secondary + tertiary contacts
  - Owner: Research assistant
  - Time: 3 hours

**Subtotal: 11.5 hours | Deadline: July 25, 2026**
