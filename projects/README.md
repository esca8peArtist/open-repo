# Projects Directory

This directory stores all your individual projects that use the SuperClaude Framework.

## Structure

Each project should be in its own subdirectory:

```
projects/
├── project-1/
├── project-2/
└── project-3/
```

## Usage

1. Create a new project folder:
   ```bash
   mkdir projects/my-new-project
   cd projects/my-new-project
   ```

2. Initialize your project (e.g., npm init, python venv, etc.)

3. Use SuperClaude commands in your project directory:
   ```bash
   /sc:brainstorm
   /sc:implement
   /sc:test
   ```

## Benefits

- **Shared Framework**: All projects share the same SuperClaude installation
- **Isolated Files**: Each project's files remain separated and organized
- **Easy Management**: Simple folder structure for all your work
- **Version Control**: Each project can have its own git repository
