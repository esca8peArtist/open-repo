```bash
  -H "Authorization: token $GITHUB_PAT" \

  -d "{
    \"description\": \"Domain 56 — Civil Service Politicization and the Destruction of Nonpartisan Governance Architecture | Democratic Renewal Research Framework | H.R. 492 Legislative Window June 2026\",
    \"public\": true,
    \"files\": {
      \"domain-56-civil-service-politicization-nonpartisan-governance-2026.md\": {
        \"content\": $(cat /tmp/domain-56-gist-content.md | python3 -c 'import json,sys; print(json.dumps(sys.stdin.read()))')
      }
    }
  }"
```

The response JSON will contain `html_url` — that is the Gist URL to record.
