## Table 2: Adaptation → Phase 2 Decision Tree

**How to read this table**: When an adaptation pattern in the left column is observed meeting the threshold in the middle column, take the action in the right column. "Phase 2 domain research" means full new domain research (4,500-6,500 words). "Domain update" means a minor version increment (adding evidence, correcting facts, adding cross-references). "Companion brief" means a separate 1-2 page document that supplements without modifying the base domain. "No action" means the pattern is expected and does not require a response.

| Adaptation Pattern Observed | Threshold for Action | Phase 2 Action |
|-----------------------------|---------------------|----------------|
| Multiple sectors adding similar new data from outside the framework's current scope (e.g., healthcare data consistently added to Domain 11 by 3+ organizations) | 3 or more independent organizations from different sectors | Phase 2 domain research on the identified gap (e.g., Domain 11b Healthcare Access Collapse — already in exploration queue) |
| Organizations in one sector applying a domain to a use case the framework didn't address (e.g., Domain 6 applied to state court capture by 3+ organizations) | 3 or more independent organizations in the same sector | Phase 2: new domain candidate identified. Add to exploration queue with priority elevation. |
| Load-bearing domain element consistently removed (e.g., Domain 29 contested qualifier removed in 3+ adaptations) | 3 or more independent adaptations removing the same element | Clarification brief for that element. Distribute to all Phase 1 contacts who received the domain. Do not modify the base domain. |
| Domain consistently modified to add international comparisons the framework omitted | 2 or more organizations making the same addition | Domain update (minor version): incorporate the comparative data the organizations are adding. Credit the organizations. |
| A whole domain consistently underused (<10% of sector contacts producing any output referencing it within 90 days) | One full sector below 10% after 90 days | Re-targeting analysis: wrong audience or wrong distribution channel? Fix targeting before considering domain modification. |
| Sector-specific framing reorientation (e.g., law schools add environmental law lens to Domain 5 in 3+ syllabi) | 3 or more independent sector adaptations in the same direction | Phase 2 domain candidate: the reorientation identifies a research gap. Add to exploration queue with sector-identified priority. |
| Domain vocabulary appearing in partisan political advertising or campaign materials without analytical framework | Single instance if prominent | Monitoring escalation: document the misuse. Issue clarification note to Phase 1 contacts if the misuse is generating media coverage of the misrepresentation. |
| Framework sections cited in support of voter suppression or anti-democratic actions | Single confirmed instance | Immediate response: public clarification statement; direct correction outreach to the using institution. This is fundamental misuse, not adaptation. |
| AG coalition uses Domain 33/37 in a major filing that succeeds at preliminary injunction stage | One confirmed winning filing | Domain deepening brief: expand the specific legal theory used in the successful filing. This is a validated Phase 2 investment — proven to work in court. |
| 2+ law school clinics assign a domain as primary case material (not supplementary reading) | Two confirmed clinical assignments | Academic companion brief: reformat the domain with full doctrinal apparatus, cleaned-up academic register, and expanded case law citations. |
| Adaptation → Phase 2 domain candidate already in queue is advanced by observed adaptation | Sector adapting current framework in the direction of queued candidate | Priority elevation: move the queued candidate up. The adaptation is empirical evidence of demand. |
| Hard-deadline domain (regulatory, legislative, litigation deadline approaching within 90 days) | Calendar-driven regardless of Phase 1 adoption | Initiate Phase 2 research immediately. Do not wait for Phase 1 feedback signals. Deadline-driven, not demand-driven. |

**Version decision rules**:
- Minor version increment (v1.x): factual correction, new case law, new evidentiary data added. Does not change analytical conclusions.
- Major version increment (v2.0): analytical structure changes, policy conclusions updated, load-bearing elements modified. Requires change notification to all Phase 1 contacts.
- Companion brief: supplements without modifying. Does not change version number. Adopters can use companion independently.
- New Phase 2 domain: entirely new domain added. Does not modify existing domains. Increment version only when integrating into the full framework document.

**Cross-sector escalation protocol (use when 2+ sectors simultaneously below velocity benchmark at 6 weeks)**:

1. Audit distribution confirmations — verify the framework reached the correct departmental contacts (policy counsel, research directors, clinic directors — not communications, development, or administrative staff).
2. Issue 2-page crisis-specific executive brief targeted to each sector's current most urgent crisis. Use this as the entry point for re-distribution.
3. Activate peer endorsement: identify the 1-3 highest-engagement early adopters across any sector. Request direct peer introductions to their networks in the underperforming sectors.
4. If cross-sector stall persists at 12 weeks: commission a formal Phase 2 distribution strategy review. The problem is structural, not content-based.
