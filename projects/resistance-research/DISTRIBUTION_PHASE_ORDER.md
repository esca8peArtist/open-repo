# Distribution Phase Order
## Recommended Domain Sequencing for Each Path

*Created April 27, 2026 (Session 519).*
*This document shows the recommended rollout sequence for the 35-domain framework depending on which distribution path is chosen. The sequencing reflects: (a) which domains provide foundational context, (b) which domains are most time-sensitive, (c) which audiences should receive which domains, and (d) internal cross-reference logic.*

---

## Path A — Immediate Broad Distribution

Path A distributes all domains to all audiences simultaneously. The sequencing below is for **Substack serialization** (one post per 3-5 days) and **institutional outreach prioritization** (which organizations to contact first). Reddit posts can be published in this order or in audience-community order.

## Path A+Domain37 Hybrid — Two-Phase Sequencing

The Hybrid path runs Path A's general-audience sequencing simultaneously with a separate Domain 37-targeted track to election protection organizations.

### Phase A (General Audiences) — Same as Path A above

Follow the Phase A1-A4 sequence above for Substack, Reddit, and general institutional outreach.

**One adjustment for Hybrid**: In Phase A2, Domain 37 should still be included in the general serialization (Post 16 in the sequence above). Election interference is relevant to all audiences, not only election protection specialists. The targeted Phase B track is an *additional* distribution, not a substitution.

### Phase B (Domain 37 Election Protection Track) — Parallel Execution

Targeted to organizations with specific capacity to act on the election interference findings.

| Step | Action | Timing | Target Organizations |
|------|--------|--------|---------------------|
| B1 | Domain 37 direct outreach — initial wave | Week 1-2 post-launch | Brennan Center, Democracy Docket, LWV National |
| B2 | Domain 37 state-level outreach | Week 3-4 | State ACLU affiliates, state LWV chapters, state election law clinics |
| B3 | Domain 37 advocacy window outreach | By May 30 | Organizations tracking DOJ consent decree deadlines (Advocacy Window 1) |
| B4 | Domain 37 pre-election law outreach | By September 2026 | Law school election law clinics, election lawyers networks (Advocacy Window 4 — Section 3 disqualification prep, preliminary injunction preparation) |
| B5 | Domain 37 operational outreach | By October 2026 | Poll observer deployment organizations, election protection legal hotlines (Advocacy Window 5) |

**Key Domain 37 advocacy windows** (from domain document):
- May 30, 2026: DOJ consent decrees finalized or blocked — first advocacy window
- June 30, 2026: Emergency EO routing window — OMB/legal review challenge opportunity
- August 7, 2026: Primary election stress-testing — SAVE error rates, EAC grant compliance data
- September 2026: Pre-election litigation positioning — Section 3 disqualifications, preliminary injunctions
- October 2026: Poll observer deployment, legal hotline readiness, real-time monitoring infrastructure
