# Phase 2 Resource Allocation Contingency Matrix

**Version 1.0 — June 14, 2026**

**Lead finding**: The Phase 2 research sprints (10-14h per domain, 22-30h total) are the highest-leverage work in the advocacy pipeline but they are NOT required for any of the immovable external deadlines. Domain 59's Senate Finance express path (45 min) and Domain 51's Tier 1 expansion (30 min) together consume under 2 hours and cover both legislative windows. The research sprints deepen the advocacy impact; they do not enable the initial sends. This matrix lets the user match available capacity to the right scope of work for each engagement scenario — without over-committing in a constrained week or under-using capacity in a strong-signal scenario.

---

## Matrix Structure

**Rows (Engagement Scenarios)**: 5 scenarios drawn from the STRONG / MODERATE / WEAK branch logic in `DAY_7_CHECKPOINT_ROUTING_DECISION_TREE.md`.

**Columns (Phase 2 Option Sets)**:
- **Option Set A — Full scope** (10-14h per domain): Both full research sprints, all Tier contact waves, full cross-domain synthesis
- **Option Set B — Compressed scope** (6-8h per domain): Zones 1-2 only per domain, Tier 1-2 contact waves, no Tier 3
- **Option Set C — Minimal scope** (2-3h per domain): Express paths only, no research sprints, highest-priority contacts only

**Which option to use**: Each scenario row identifies the recommended option set AND specifies what changes if capacity is unavailable for that option. No scenario is binary — every scenario has a fallback.

**Condition**: 3+ Score 3+ replies; but researcher-hours are constrained (fewer than 10h available over 5 days for sprint work — user has other demands in the June 17-22 window).

**Domain context**: Strong signal confirms both domains are landing. The constraint is researcher time, not advocacy opportunity. The protocol prioritizes Domain 59 because of the Senate Finance window over Domain 51 when capacity forces a choice.

**What this means in practice**:
- D59 Zone 1-2 research sprint (4-5h) runs first, Days 1-2. Senate briefing annex + OBBBA table produced.
- D51 Zones 1-2 research sprint defers to Days 3-4 if capacity opens; otherwise defers to a later session entirely.

**Domain 59 priority rationale**: Senate Finance markup window closes June 25-30. A Senate staff briefing supplement produced by Zone 1-2 research (4-5h) can reach CBPP before the markup closes. Domain 51's Zone 1 FEC enforcement addendum (flagged to CLC) is valuable but not time-pressured in the June 25 window.
