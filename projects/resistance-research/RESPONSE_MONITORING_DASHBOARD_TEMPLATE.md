## Google Sheets Formula Reference

When copying this data into Google Sheets, use the following formula pattern for automated calculations:

### Weighted Score Formula
Assign numeric weight values in a lookup table (a separate sheet named "Codes"):
- Column A: Code (SE, BC, CIT, FWD, SMM, AR, BNC, NO)
- Column B: Weight (1.0, 1.5, 2.0, 0.75, 0.5, 0.0, 0.0, 0.0)

In the main tracker sheet, weighted score column:
```
=IFERROR(VLOOKUP([ResponseCode_cell], Codes!$A:$B, 2, FALSE), 0)
```

### Running Total (section-level)
```
=SUM([WeightColumn_start]:[WeightColumn_end])
```

### Response Percentage
```
=COUNTIF([ResponseCode_column], "<>NO") / COUNTA([ContactName_column])
```

### Trigger Threshold Indicator (conditional formatting)
For T+14 Phase 2 path determination:
- If weighted score >= 3.0: format green (STRONG)
- If weighted score >= 2.0 and < 3.0: format yellow (MODERATE)
- If weighted score < 2.0: format red (WEAK)

Apply in conditional formatting: `=$B$[total_row] >= 3` for green, `=$B$[total_row] >= 2` for yellow, `=$B$[total_row] < 2` for red.

### Days Since Send
```
=TODAY() - DATE([send_year], [send_month], [send_day])
```

### Response Due Countdown
```
=[Target_date_cell] - TODAY()
```
Format as: 0 = due today; negative = overdue; positive = days remaining.
