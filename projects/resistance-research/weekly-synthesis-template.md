## Quick Copy-Paste Checklist for Monday Morning

Use this checklist to walk through your weekly synthesis:

- [ ] **9:00 AM**: Open Gmail, count new replies in `phase1-outreach/replies/` label
- [ ] Score each reply using reply-triage-framework.md
- [ ] Update Contacts sheet in Google Sheets with reply dates and scores
- [ ] Log into Bitly dashboard, record weekly click counts
- [ ] Update Gist Views sheet with Bitly data
- [ ] Check Google Alerts for new mentions
- [ ] Check Regulations.gov for Domain 39/42 comments (if applicable)
- [ ] Open this template, fill in Sections 1–7
- [ ] Copy filled template to `monitoring/phase-1-week-X-synthesis-[date].md`
- [ ] If checkpoint date: Fill in Section 10 and follow Phase 2 decision protocol
- [ ] Commit synthesis file to git
- [ ] Done! (~15–20 minutes total)
