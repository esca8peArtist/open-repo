# Phase 3 Execution Timeline — Nov 2026 through March 2027

*Authored: June 6, 2026. Phase 3 operational scope: November 2026 - August 2027. This document covers the first five months (Nov 2026-March 2027).*

---

## Phase 3 Operating Assumptions

**Launch gate**: Phase 3 launches November 2026 regardless of Phase 2 adoption metrics, per the strategic roadmap's go/no-go criteria (minimum threshold: 3 contacts at Level 2+ adoption by December 2026 — expected to be met based on current Phase 2 trajectory).

**Capacity baseline**: Phase 2 actuals: ~200 hours / 7 domains over 7 months = 28.5 hrs/domain, approximately 4-5 sessions per domain. Phase 3 is planned at the same pace with a slightly higher proportion of update-pass work vs. full new research.

**Political context dependency**: The November 2026 midterm outcome determines which domains move from Tier 2 to Tier 1 urgency. If Democrats gain either chamber of Congress, Domain K (judicial restructuring) and Domain H (constitutional architecture) become legislative-window domains that must be in distribution by January 2027. The timeline accounts for both scenarios.

**Phase 2 carryover assumption**: Domains G (press freedom), 37a (post-election certification), and 31x (healthcare tariff collision) are Phase 2 obligations that should close by October 2026. If any remain incomplete at Phase 3 launch, they become the first deliverables (Week 1-2, November 2026) before new Phase 3 research begins.

## Parallel-Track Opportunities

These domains can run concurrently without resource conflicts:

**Track A (Legislative window track)**: Domain K (Dec) → Domain H (Dec-Jan) → Domain I (Jan-Feb) — these three follow sequentially but each feeds the same congressional and constitutional reform audience. They must be sequential because they share primary researcher time.

**Track B (Update-pass track)**: Candidate 6 (Nov, update pass) and Candidate 8 (Jan, update pass) are independent of Track A. They can be assigned to separate sessions without competing for the same source research.

**Track C (Technical track)**: Domain J (Feb-March) is a standalone domain requiring technical depth. Schedule dedicated sessions without overlap with Domain I final production.

**Track D (Meta-track)**: Domain L (March) draws on the outputs of all Phase 3 domains and the movement networks activated in Phase 2. It can only begin after Phase 3 outreach has generated enough new organizational contacts to survey (minimum: 50+ new Tier 3 contacts reached through Phase 3 channels).

## Contingency: Phase 2 Slippage (Aug 31 → Sept 30 close)

If Phase 2 distribution does not complete until September 30 rather than August 31, the following adjustments apply:

**Domains that defer to Q2 2027** (can slip without losing their primary window):
- Domain J (cryptographic democracy) — state pilot legislative sessions run Jan-April 2027; a Feb-March 2027 production start still reaches the window
- Domain L (movement infrastructure) — spring philanthropic cycles run March-May 2027; a March-April 2027 production start still reaches the window
- Domain I (international standards) — the V-Dem/Freedom House submission deadline is hard (Feb 2027), but the State Dept appropriations hearing window extends through April 2027; missing February submission is consequential but not fatal

**Domains that are load-bearing for Phase 4 and must not defer**:
- Domain 37a — the post-election certification window is Nov 4-Jan 3, 2027; no slippage possible
- Domain K — the Congressional seating window is Jan 3, 2027; no slippage possible for Tier 1 distribution; any slip past December 31 means distributing into an already-active legislative session rather than ahead of it
- Domain H — tied to Domain K's constitutional reform network; must accompany Domain K distribution or trail by no more than 2 weeks
- Domain G (press freedom, Phase 2 obligation) — must close before Phase 3 can formally launch; carrying a Phase 2 obligation into Phase 3 delays the Phase 3 announcement and the knowledge commons site completion

**Phase 4 load-bearing domains**: Based on the Phase 3 strategic roadmap's 5-year vision, Phase 4 will require:
- Domain H (constitutional architecture) as the structural repair blueprint — if incomplete at Phase 3 close, Phase 4 cannot build on it
- Domain I (international standards) as the comparative evidence base — Phase 4's international integration work (Freedom House documentation, State Dept appropriations advocacy) depends on this being a citable, production-quality document
- Domain J (cryptographic democracy) as the technical foundation for any state-level pilot program advocacy in 2028+
- Domain L (movement infrastructure survey) as the distribution network map — without this, Phase 4's Tier 3 outreach strategy has no empirical foundation

Any domain that is Phase 4 load-bearing must be treated as non-deferrable even under Phase 2 slippage.
