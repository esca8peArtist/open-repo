## Appendix — Engagement Scoring Formula Reference

```
engagement_score = (reply_score × 0.6) + (gist_click_proxy × 0.2) + (opened_24h_proxy × 0.2)

Reply scores:
5 = Integration signal (using in active work product)
4 = Implementation question (how to use operationally)
4 = Referral (connecting to colleague or network)
3 = Clarification question (asking about specific claim or domain)
3 = Substantive critique (methodological or factual challenge)
1 = Acknowledgment only ("thanks, will read")
0 = Decline (opted out or not relevant)
0 = No reply

Gist click proxy:
1 = confirmed click (Bitly) or Gist view spike attributable to this contact
0 = no click detected

Opened 24h proxy:
1 = OOO reply received (confirms open), OR Gist view delta within 2h of send
0 = no open signal

Example: Bassin replies with an implementation question (score 4), we detected a Gist click (1), and he replied within 24h but no OOO received (0):
  engagement_score = (4 × 0.6) + (1 × 0.2) + (0 × 0.2) = 2.4 + 0.2 + 0 = 2.6 (NEAR-PASS)
```
