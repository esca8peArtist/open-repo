## CONTINGENCY: Outcome Ambiguity and Boundary Cases

If your May 21 synthesis produces results that fall on a boundary between classifications (e.g., exactly 2 Score 3+ replies but unclear if substantive vs. acknowledgment-only), use this decision tree:

**Boundary: STRONG ↔ MODERATE** (one reply short of STRONG)
- **Decision rule**: Is there a Score 5 signal? If yes: classify STRONG. If no: check organizational type. Think tank reply + any other Score 3+ = STRONG. Otherwise = MODERATE.
- **Path selected**: STRONG path (parallel Domain 57+59)
- **Rationale**: One think tank reply is sufficient evidence of institutional traction to warrant parallel research launch

**Boundary: MODERATE ↔ WEAK** (one reply total)
- **Decision rule**: Is the one reply a Score 3+ (substantive)? If yes: MODERATE. If acknowledgment-only: WEAK.
- **Path selected**: If Score 3+: MODERATE. If acknowledgment: WEAK.
- **Rationale**: Substantive single reply indicates minimal but real traction; proceed with single-domain June 10 launch. Acknowledgment-only indicates delivery issue or non-engagement; trigger analysis phase.

**Boundary: WEAK ↔ TOO_EARLY** (zero replies but >5 Gist views)
- **Decision rule**: If zero replies but Gist delta > 5: indicates internal link sharing without public response. Classify TOO_EARLY (extend monitoring). If Gist delta ≤ 5: classify WEAK (trigger analysis).
- **Path selected**: TOO_EARLY (continue monitoring through June 7)
- **Rationale**: Internal sharing indicates interest that hasn't converted to external response yet; wait for external response signal
