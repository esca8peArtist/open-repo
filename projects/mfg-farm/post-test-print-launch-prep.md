### Naming Convention Details

**File naming syntax**: `modrun_[component]_[variant]_v[version].stl`

- **Component**: clip, rail, bracket, hardware
- **Variant**: basic, premium, designer, compact, extended
- **Version**: v1.0 (release), v1.1 (revision), v2.0 (major redesign)

**Examples**:
- `modrun_clip_basic_v1.0.stl` — Initial launch basic clip (PLA, all colors)
- `modrun_clip_premium_v1.0.stl` — Premium clip (PETG-friendly geometry, larger engagement surface)
- `modrun_rail_single_v1.0.stl` — Single rail foundation (compatible with all clip variants)

**Do not name**:
- `clip.stl`, `rail.stl` (too generic, confusing at scale)
- `modrun_final_FINAL_v3_rev2.stl` (avoid multiple revisions in one filename)

### Print Settings Documentation

Create `PrintSettings.md` in root with recommended settings for Bambu X1C/P1S:

```markdown
# ModRun Print Settings (Bambu X1C / P1S)

## Material: PLA (eSUN PLA+ recommended)

- **Nozzle Temp**: 210°C
- **Bed Temp**: 60°C
- **Infill**: 15% (clips are non-structural)
- **Layer Height**: 0.2mm (balance speed/quality)
- **Support**: None required (geometry designed for no-support)
- **Print Speed**: Default (200mm/s Bambu profile)
- **Estimated Time**: 25-45 minutes per clip (design dependent)

## Material: PETG (Overture or eSUN recommended)

- **Nozzle Temp**: 235°C
- **Bed Temp**: 80°C
- **Infill**: 15%
- **Layer Height**: 0.2mm
- **Support**: None
- **Print Speed**: Reduced to 150mm/s (PETG slower than PLA)
- **Estimated Time**: 35-60 minutes per clip

## AMS (Automatic Material System) Settings

- **Palette**: PLA + PLA (or PLA + PETG for color variants)
- **AMS slot assignment**: Check firmware version for optimal path.
- **Load filament**: Press AMS → Load → color selection → confirm.

## Post-Print

- Let model cool 5 minutes on bed (removes without damage at this point).
- Remove support material (none required).
- Visual inspection: Check for layer adhesion, dimensional fit.
- Optional: Light sanding (150-grit) if rough edges present.
```

### USPS Ground Advantage Shipping Rates (2026)

ModRun cable clips are light packages (single clip order: 3-4oz, bundle order: 8-12oz). USPS Ground Advantage is cost-optimal for these weights and distances.

**Shipping rate structure** (per Pirate Ship 2026 rates):

| Weight | Zone 1-2 | Zone 3 | Zone 4 | Zone 5 | Zone 6 | Zone 7 | Zone 8 |
|--------|----------|--------|--------|--------|--------|--------|--------|
| 1-4 oz | $3.50 | $3.85 | $4.20 | $4.60 | $4.95 | $5.25 | $5.50 |
| 5-8 oz | $4.15 | $4.75 | $5.35 | $6.15 | $6.95 | $7.70 | $8.40 |
| 9-12 oz | $4.75 | $5.60 | $6.60 | $7.75 | $8.95 | $10.05 | $11.10 |

**Zone determination**: Zip code of customer determines zone (Pirate Ship calculates automatically)

**Representative weights**:
- Single clip (modrun_clip): 3.5 oz
- Single rail (modrun_rail): 5 oz
- 3-pack bundle (3 clips): 10 oz
- Kit (rail + 4 clips): 12 oz
- Deluxe Kit (2 rails + 14 clips): 28 oz (requires 2-zone shipping split)

### Etsy Shipping Configuration

In Etsy Shop Settings → Shipping Profiles:

**Profile 1: "Standard Domestic (USPS Ground Advantage)"**

Set up using **Calculated Shipping** (recommended for accuracy):

1. Click "Create Shipping Profile"
2. Name: "Standard Domestic"
3. Processing time: 2 business days
4. Origin location: [Your city, state]
5. Select shipping method: **USPS Ground Advantage**
6. Shipping type: **Calculated by weight** (Etsy calculates zones automatically)
7. Base cost: $0.00 (Etsy calculates per-zone rates)
8. Weight: Enter representative weight (e.g., 0.25 lbs for single clip)
9. Additional weight: Add tier for additional oz (if multi-unit orders)

**Setup steps in detail**:
- Etsy auto-calculates zone rates based on customer zip code and package weight
- You enter packaging weight/dimensions (e.g., "clip order: 0.22 lbs in 6×4×2 box")
- Etsy pulls live USPS rates and displays to customer at checkout
- Customer pays actual zone rate (you don't guess)

**Recommendation**: Use Etsy's default USPS Ground Advantage rates. They are accurate and transparent.

### Processing Time & Return Policy (Shipping Component)

**Processing time**: Set on **each listing** to 2 business days
- This is the time from order purchase to shipment date
- 2 business days allows: Order received (same day) → Print queue next morning → Finishing/packaging (day 2) → Ship via Pirate Ship (same day)
- Delivers realistic expectation; beating it improves reviews

**Return shipping**: "Seller pays for return shipping"
- 30-day policy standard
- Etsy generates return label; you pay return postage
- Encourages returns and builds trust (buyers less hesitant)
- Expected return rate: 2-5% (manageable at launch volume)
