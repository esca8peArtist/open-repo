## Section 5: Etsy Algorithm Best Practices (2026)

### Title Structure

- **Character target:** 60–80 characters (70-char soft cap from February 2026 NLP update; longer titles face mobile demotion)
- **First 40 characters:** Primary keyword phrase, intact — do not split it ("Cable Management Clips" not "Clips Cable Management")
- **Characters 41–70:** Key differentiator + use case modifier
- **Formula:** `[Primary keyword phrase] — [Key differentiator] | [Use-case modifier]`
- **Never:** Brand name in title (wastes high-weight real estate), filler adjectives ("amazing," "beautiful"), keyword chains without natural phrase flow

### Tag Strategy (13 Tags Required)

- Use all 13 tags — empty slots are wasted query-match opportunities
- Each tag = a distinct multi-word phrase (single-word tags are near-useless in 2026)
- Do not repeat title phrases verbatim — use synonymous and adjacent terms to widen query surface
- Etsy auto-handles pluralization — do not waste a slot on both "cable clip" and "cable clips"
- Occasion/gift tags ("gift for him," "back to school") belong in tags, not titles
- Multi-word tags outperform single words ~3:1 in query-match contribution

### Ranking Factor Priority Matrix (2026)

| Signal | Weight | ModRun Action |
|---|---|---|
| Listing Quality Score (CTR + conversion + velocity) | Highest | Launch best product first; optimize photos before listing goes live |
| Relevance (title/tag/attribute/description alignment) | High | Use intact keyword phrases; complete all attributes |
| Shop-level score (reviews, response rate, shipping) | Medium-High | Set 3–5 day processing time; ship in 24–48 hours |
| Shipping price | Medium | Free shipping only (bake $4.50–$5.50 USPS First Class into product price) |
| Star Seller status | Gate (30% of buyer searches) | Achieve within 90 days of first sale |
| Recency boost | Medium (7–14 day window) | One chance per listing; launch with full optimization |
| Title length quality | Medium (new 2026) | Stay under 70 chars; avoid keyword stuffing |
| Video (5–15 sec) | Low but differentiating | Under 10% of 3D printed listings have video — easy win |

### Description Best Practices

- First 160 characters: primary keyword + compelling benefit statement (highest Etsy NLP weight)
- First 250 characters (combined): also serves as ChatGPT semantic matching zone (Etsy-ChatGPT integration live May 2026)
- Use the PCOS structure: Problem → Core function → Options → Social proof hook
- Include actual numbers (clip jaw width, cable diameter range, pack count, magnet spec)
- Natural language for AI discovery: "For the home-office worker who loses a cable every time they move the monitor"

### Should You Update Titles Monthly?

No. Title stability is a ranking signal — frequent changes confuse Etsy's LQS accumulation and can reset ranking position. Update titles only when: (a) a keyword is demonstrably underperforming after 30 days, or (b) the seasonal window requires a genuinely different primary keyword. If a listing fails its 14-day recency window, do not edit the same listing — create a new one. The LQS is tied to the listing URL.

### Psychological Pricing Rules

1. Avoid round numbers at non-boundary prices ($14.99 > $15.00; $16.99 > $17.00)

3. Never discount below $14.99 once Offsite Ads threshold is crossed ($10k annual sales) — at the mandatory 12% commission, $12.99 ad-attributed sales drop to ~42% gross margin (the floor)

### Pre-Launch Verification (All Listings)

Before any listing goes live, verify all items pass:

- [ ] Title under 70 characters, primary keyword in first 40
- [ ] All 13 tag slots used with multi-word phrases
- [ ] First 160 description characters contain primary keyword + benefit
- [ ] Minimum 5 photos uploaded (lifestyle hero shot is image 1)
- [ ] Video uploaded (7–15 seconds, MP4/MOV, square or vertical)
- [ ] All attributes complete (material, color, style, size if applicable)
- [ ] Free shipping set ($0 shipping, cost absorbed into price)
- [ ] Processing time set to 3–5 business days
- [ ] Shop policies complete (returns, shipping, privacy)
- [ ] Auto-reply message template active for customer messages
- [ ] USPS tracking enabled on all orders
