## Section 2: Appropriate Trademark Classes

### International Classification for ModRun Products

ModRun's current product line is cable management clips, rails, and desk organizational accessories — physical, non-electric plastic products made by FDM 3D printing.

**Primary class: International Class 20**

Class 20 covers "furniture, mirrors, picture frames; goods (not included in other classes) of wood, cork, reed, cane, wicker, horn, bone, ivory, whalebone, shell, amber, mother-of-pearl, meerschaum and substitutes for all these materials, or of plastics."

The critical phrase: "goods not included in other classes... of plastics." Cable management clips, desk organizers, and magnetic bin labels made of PLA (a plastic material) fall squarely within Class 20 when they are non-electric and non-mechanical. The goods identification should read: "Cable management clips made of plastic; desk organizer accessories made of plastic; non-electric cable organizing accessories made of plastic."

**Why Class 20, not Class 9:**
- Class 9 covers electrical and electronic equipment, including electric cables and conductors
- ModRun cable clips are not electronic — they are passive, non-electric plastic organizers that hold cables without any electrical function
- Class 9 would be appropriate only if ModRun sold the cables themselves or cable-related electronic accessories
- Filing in the wrong class risks refusal or limited protection scope

**Secondary class consideration: International Class 21 (if applicable)**

Class 21 covers "household or kitchen utensils and containers; combs and sponges; brushes; brush-making materials; articles for cleaning purposes; steel wool; unworked or semi-worked glass; glassware, porcelain and earthenware." This class includes some desk organizational accessories. For ModRun's current SKUs, Class 20 is the stronger primary filing; Class 21 adds cost without significant incremental protection at this stage.

**Digital designs class: International Class 9 (for future consideration)**

If ModRun later sells digital STL/3MF files for 3D printing, those digital products fall under Class 9 (downloadable computer files; downloadable design files for three-dimensional printing). A second trademark filing in Class 9 would be a Phase 3+ consideration once digital design sales become a significant revenue stream.

**Recommendation for June 15 filing:** Single class, Class 20 only. Cost: $350. Sufficient for Amazon Brand Registry and protection of physical products. Add Class 9 (digital designs) in 12–18 months if digital file sales materialize.

## Section 3: Filing Fees

### Current USPTO Fee Schedule (Effective January 18, 2025, Updated May 1, 2026)

The old TEAS Plus / TEAS Standard two-tier system was consolidated effective January 18, 2025 into a single base fee with targeted surcharges:

| Fee Type | Amount |
|---|---|
| Base application filing fee (per class) | **$350/class** |
| Surcharge: incomplete application | $100/class |
| Surcharge: custom goods description (free-form, not from ID Manual) | $200/class |
| Surcharge: extended custom description (over 1,000 characters) | $200/1,000 chars |
| Statement of Use (Section 1(b) only, after Notice of Allowance) | $150/class |
| Extension of time to file Statement of Use | $125/class |
| Response to Office Action (no fee if filed within 3 months) | $0 |
| Response to Office Action extension (3 more months) | $125/class |
| Renewal (Section 8/9, every 10 years) | $650/class |

**Total cost for a clean single-class filing:**
- Base fee: $350
- No surcharges if you use the Trademark ID Manual to identify goods (do not use free-form text)
- **Total out-of-pocket: $350**

**If using a trademark attorney ($800–$2,500 typical):** An attorney will handle the ID Manual search, identify the correct class, draft the goods description, and respond to any Office Actions. For a straightforward word mark like "MODRUN" in Class 20, DIY filing is viable if you follow the USPTO's identification guidance precisely. Attorney adds certainty against refusal; DIY saves $500–$2,000.

**DIY recommended for ModRun if:** The word "MODRUN" is not already registered in Class 20 (verify via USPTO TESS search), and the goods description is taken verbatim from the Trademark ID Manual.

## Section 4: Trademark Search Requirements

Before filing, conduct a clearance search to ensure "MODRUN" is not already registered or pending in Class 20 by another entity.

### Free USPTO TESS Search

1. Navigate to: tess.uspto.gov
2. Search for: MODRUN (exact), MOD RUN (with space), MODRUN (Class 20 filter)
3. Also search phonetically similar marks: MODRUN, MOD RUN, MODERN RUN
4. Check "Live" registrations only (dead/abandoned marks are not barriers)
5. Look for similar goods descriptions in Class 20 (desk accessories, cable management, plastic organizers)

**If MODRUN is clear in Class 20:** File as planned. Confidence in clearance is reasonable.

**If a conflict is found:** Options include (a) modifying the mark (MODRUN 3D, MODRUN LABS, etc.), (b) narrowing the goods description to avoid overlap, or (c) consulting a trademark attorney for a freedom-to-operate opinion. Do not file without addressing the conflict — a rejected application forfeits the $350 filing fee.

### Common Law Search

Also conduct a basic Google, Etsy, Amazon, and domain name search for "ModRun" to identify any unregistered users of the same mark. Common law rights can complicate a trademark application even without USPTO registration.

## Section 5: Application Type — Section 1(a) vs. 1(b)

| Application Type | Description | Filing Timeline | Recommendation |
|---|---|---|---|
| Section 1(a) — Use in Commerce | Mark is already in use on actual goods sold | Can file now if products are being sold | **Recommended** if Etsy sales have occurred |
| Section 1(b) — Intent to Use | Mark is not yet in use; plan to use in the future | Requires later Statement of Use filing + $150/class extra | File if launching before first sale |

**For ModRun:** If any Etsy sale has occurred under the "ModRun" brand name (shop name, product branding, or packaging label), a Section 1(a) use-in-commerce application is appropriate and avoids the additional Statement of Use step. The specimen (evidence of use) can be a screenshot of the Etsy listing showing the mark in connection with the goods, or a photograph of the product with the mark on packaging.

**If no sales have occurred before June 15:** File Section 1(b) Intent to Use. This locks the filing date (which establishes priority against later filers) and converts to full registration after submitting the Statement of Use once products are on sale. Timeline is 2–4 months longer than 1(a) due to the additional Statement of Use step.

## Section 6: Examiner Response Timelines

### Section 1(a) Application — Complete Timeline

| Stage | Duration | Cumulative |
|---|---|---|
| File application → Serial number assigned | Same day | Day 0 |
| Application reviewed by Examining Attorney | ~8–10 months | Month 8–10 |
| If approved: TMOG publication (30-day opposition window) | ~3 months after approval | Month 11–13 |
| If no opposition: Registration certificate issued | ~3 months after publication | **Month 14–16** |
| **June 15, 2026 filing → Estimated registration** | | **August–October 2027** |

**Important clarification:** The USPTO's stated 8–10 month examination timeline is for initial review — not registration. Many applications with clean goods descriptions and no conflicts are approved at first review without Office Actions, compressing the total timeline to 12–14 months. Contested applications (with Office Actions requiring responses) add 3–6 months per response cycle.

**For Amazon purposes:** Brand Registry with a pending application is available from Day 0. The pending serial number is sufficient.

### If an Office Action Is Issued

| Scenario | Probability | Timeline Impact | Action |
|---|---|---|---|
| Non-substantive Office Action (clarify goods description) | ~30% | +2–4 months (if responded promptly) | Respond within 3 months (free); use ID Manual language |
| Substantive Office Action (likelihood of confusion with existing mark) | ~15% | +4–8 months | May need attorney; argue against likelihood of confusion |
| Final refusal | ~5% (if prior OA not resolved) | Application abandoned | Refile with modified mark or appeal to TTAB |
| No Office Action (clean filing) | ~55% | No delay | Proceed to publication |

**If Office Action is received:** Respond within 3 months of the Office Action date (at no additional fee). Extensions of 3 months are available for $125/class. A prompt, well-reasoned response to a non-substantive OA typically resolves within 1–2 months of submission.

## Section 7: Contingency Planning

### If Application Is Challenged (Opposition Period)

After publication in the USPTO Trademark Official Gazette (TMOG), third parties have 30 days to file an opposition. Grounds for opposition include likelihood of confusion with an existing mark or ownership disputes.

**Probability of opposition for MODRUN in Class 20:** Low. "ModRun" is a coined term without close phonetic or conceptual similarity to established cable management or desk accessory brands. The primary risk would be a brand using "Mod" + a term suggesting movement in the desk accessories category. A TESS search before filing mitigates this risk.

**If opposition is filed:** The case moves to the Trademark Trial and Appeal Board (TTAB). This is a legal proceeding. An unchallenged opposition (where ModRun does not respond) results in automatic denial. If challenged, estimated attorney cost for TTAB proceedings: $3,000–$10,000 depending on complexity. Not anticipated for this mark.

### If Registration Is Delayed Beyond 14 Months

Amazon Brand Registry does not require a registered trademark — only a pending application. The 2026 requirement update (spring 2026 mandate for FBA sellers to use Brand Registry or Amazon barcodes) makes Brand Registry enrollment more urgent, but the pending application path remains valid.

**If registration is delayed past March 2027** (when Phase 3 Shopify D2C is targeted): This does not block Shopify D2C launch. Trademark registration enhances anti-counterfeiting tools but is not a prerequisite for Shopify store operation.

## Section 8: Step-by-Step Filing Checklist

Execute in this order before June 15, 2026:

- [ ] **Step 1 — TESS search:** Search tess.uspto.gov for "MODRUN" in Class 20. Confirm no conflicting live marks.
- [ ] **Step 2 — Google/Etsy/Amazon search:** Confirm no prominent common-law users of "ModRun" in cable management or desk accessories.
- [ ] **Step 3 — Gather specimen (Section 1(a)):** Screenshot of Etsy listing showing "ModRun" brand name on the listing page, or photograph of product with "ModRun" on packaging/labeling. Save as JPEG or PNG.
- [ ] **Step 4 — Access USPTO Trademark Center:** Navigate to trademarks.uspto.gov. Create or log into your USPTO account.
- [ ] **Step 5 — Start new application:** Select "File a trademark application" → Section 1(a) [use in commerce] if products have been sold.
- [ ] **Step 6 — Mark information:** Select "Standard characters" (word mark). Enter: MODRUN (all caps, no stylization — a standard character word mark covers all fonts/styles).
- [ ] **Step 7 — Goods identification:** Under Class 20, use ID Manual. Search for: "cable organizers, not of metal" or "cable management accessories, not of metal, not electrically conductive." Select the closest matching approved identification phrase from the ID Manual. Do not free-form text (avoids $200 surcharge).
- [ ] **Step 8 — Specimen upload:** Upload the Etsy listing screenshot or product photo as the specimen of use.
- [ ] **Step 9 — Dates of first use:** Enter the date of first Etsy sale as "date of first use anywhere" and "date of first use in commerce."
- [ ] **Step 10 — Owner information:** Enter owner name (individual name or business entity), address, email.
- [ ] **Step 11 — Review and pay:** Review application for errors. Pay $350 filing fee by credit card. Submit.
- [ ] **Step 12 — Save serial number:** USPTO assigns serial number within minutes of filing. Save this number — it is required for Amazon Brand Registry application.
- [ ] **Step 13 — Amazon Brand Registry:** Navigate to brandregistry.amazon.com. Begin enrollment using the trademark serial number. Approval typically within 1–14 days.
