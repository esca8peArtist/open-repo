# Etsy Shop Setup Validation — May 2026 Updates

**Date**: 2026-05-17  
**Reference**: `etsy-listing-launch-checklist.md` (648 lines)  
**Status**: Validated with three material updates required

## Summary of Changes

The checklist is substantially sound. Three areas require updates or attention before execution:

### 1. Return Policies: Updated UI, Still Valid Strategy

**What changed**: Return policies are now configured **at the listing level**, not shop-wide. The navigation path (Shop Manager > Settings > Policy Settings) still works, but you'll apply policies per listing or bulk-apply via Listings Manager > Change return & exchange policies.

**Strategic guidance remains valid**: 30-day returns, buyer pays return shipping—this is still the correct choice for a cable clip product.

**Confidence**: High

### 2. Etsy Fees: Correction Required in Pricing Formula

**Current assumption in checklist**: 9.5% (transaction + payment processing)  
**Correct rate**: 10–11% effective

Breaking this down for ModRun clips at typical volume:
- Transaction fee: 6.5% of total (including shipping)
- Payment processing: 3% + $0.25 per transaction
- At $16.49 average order (1 clip + shipping): 6.5% + 3% + fixed = **11.0% total**

**Action**: Update the pricing formula section to use 10.5% as the planning baseline. Adjust revenue projections accordingly.

### 3. Materials Tag Field: Removed—Attributes Only

**What changed**: Etsy removed the legacy "Materials tag" text field from listing forms. Sellers must now use **Attributes only**.

**Action**: When entering product details, skip any text-based materials field and fill Material attributes (e.g., Material: "Plastic", Specification: "PLA").

## Minor Updates (Low Priority)

- **AI Disclosure**: If using AI tools for product design, disclosure may be required. For an original ModRun design, this is N/A.
- **Subcategory Required**: Select a specific subcategory (e.g., "Home Organization > Storage & Organization") before saving listings—the form now enforces this.
- **Share & Save Program**: New optional program—earn 4% fee discount on traffic you drive via unique links. Worth activating at launch (no downside).
