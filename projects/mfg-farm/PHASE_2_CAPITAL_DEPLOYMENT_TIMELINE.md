# Phase 2 Capital Deployment Timeline — ModRun

**Lead finding:** The total capital requirement for Phase 2 pre-launch (June 1–July 15) is $750–$1,100 depending on how aggressively you deploy. The single longest-lead item — trademark filing ($350/class, 8–10 month examination timeline) — should be filed by June 15 regardless of Phase 1 outcome because it does not depend on test print results. A registered trademark is required for Amazon Brand Registry, which unlocks Vine reviews (the critical cold-start solution for FBA). Filing June 15 targets Brand Registry access by April 2027, which aligns with the Phase 3 Shopify D2C launch if Phase 2 executes on schedule.

## Section 2: What Can Be Pre-Ordered June 1–15 (Before Phase 1 Confirmation)

The following items do not require Phase 1 test print validation and can be initiated immediately:

### Can Initiate June 1–15 (No Gate):

**1. Trademark Filing ($350)**
- Does not depend on product validation
- Initiating early is risk-free: filing fee is non-refundable but the mark is not product-specific
- Apply for "MODRUN" (word mark) in the appropriate class
- Complete the Trademark ID Manual search for goods identification before filing
- Action: File by June 15 at latest (see TRADEMARK_FILING_STRATEGY.md for step-by-step)

**2. Filament Restocking ($110–$150)**
- eSUN PLA+ 10kg bundles in black, white, grey via Amazon Prime
- No dependency on Phase 1 outcome: filament will be consumed in production regardless
- Order by June 1 for delivery by June 3–5
- Required before printing FBA batch

**3. Packaging Supplies ($55–$75)**
- Poly mailers (1,000-unit order from Shop4Mailers)
- FBA-specific: 100 units of clear poly bags (FNSKU compatible, approximately $8–$12)
- Thermal barcode labels for FNSKU stickers (Rollo-compatible, 4"×6", 250 pack)
- USPS tracking labels (already covered by Pirate Ship workflow)

**4. Amazon Seller Central Account Setup ($0)**
- Open Professional Seller account before Phase 1 confirmation
- Complete Handmade application (requires design process documentation but not test print validation)
- Apply for Brand Registry with pending trademark number (file trademark first, then use application serial number)
- This setup takes 2–4 weeks and can run in parallel with Phase 1

**5. Bambu Lab printer quote requests ($0)**
- Contact MatterHackers (sales@matterhackers.com) and Bambu Lab B2B portal for quotes on 3 P1S units
- Quote requests are free and lock in visibility on pricing and availability
- No commitment required; confirms procurement path before June 15 capital deployment

### Gate on Phase 1 Completion (May 30–31):

**6. Amazon FBA inventory batch (25–50 units, $93–$188 COGS)**
- Start printing only after Phase 1 test print confirms production-quality output
- If Phase 1 validates, begin FBA batch immediately (targeting Amazon listing live by June 20–25)
- If Phase 1 reveals a design flaw requiring revision, pause FBA batch 1–2 weeks for correction

**7. P1S printer purchase ($399–$650 per unit)**
- Equipment is a large-dollar decision; wait for Phase 1 to confirm production viability before committing
- Pre-order quotes already in hand; execution happens June 3–15 post-validation
